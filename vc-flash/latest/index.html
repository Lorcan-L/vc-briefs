赛道热度观察

1. AI仍是压倒性主线，但资金结构在分层

Crunchbase Q1 2026 数据显示，全球VC投资达到约3000亿美元，其中AI公司获得约2420亿美元，占80%；OpenAI、Anthropic、xAI、Waymo四笔巨额融资合计约1880亿美元，占全球季度融资约65%。这不是普涨，而是“少数基础设施级公司 + 物理AI/应用基础设施”的集中下注。

2. 美国3月回到“去巨额轮后的常态”，但AI仍拿走多数资金

AlleyWatch 统计，美国3月VC融资约190.6亿美元、630笔交易；AI企业获得114.6亿美元，占60.1%。这说明即便剔除OpenAI/Anthropic/Waymo式超大轮，AI仍是默认主赛道。新增资金更多流向国防自主系统、海事机器人、企业AI基础设施、开发者工具，而不是单纯基础模型。

3. 中国市场更强调“政策确定性 + 产业落地”

新浪财经/深圳市投资基金同业公会口径显示，2026年一季度中国股权投资市场新募基金数量和金额分别同比增长114.0%和79.8%；资金集中在人工智能、具身智能、半导体、商业航天/低空经济、生物医药与新能源链。4月仅8家具身智能企业即获得超200亿元融资，显示人民币基金更看重国家战略、量产订单和退出通道。

4. 具身智能进入“订单验证期”

中国证券报对红杉、高瓴一季度布局的梳理显示，投资逻辑已经从“技术概念”转向“量产与订单”：自变量机器人10亿元A++轮、千寻智能近20亿元A轮、星海图B轮、灵猴机器人B轮等案例均强调产线、交付、客户覆盖和实际生产力创造。

5. 深科技非共识领域获得头部机构继续加仓

低空经济、商业航天、可控核聚变等仍然高门槛，但因政策、产业周期和退出预期改善，红杉、高瓴、IDG等继续出手。该类项目融资频率不如AI高，但单笔金额和机构质量较强。

分析师点评

1. 市场不是回暖到“人人有钱”，而是强烈两极化。 AI、机器人、硬科技、政策支柱赛道获得高估值和大额融资；传统消费/纯互联网/无技术壁垒项目仍处冰河期。

2. AI投资从“模型能力竞赛”外溢到“部署基础设施”。 数据、仿真、评测、云/网络、开发者工具、安全和垂直场景，正在接棒成为更可投的中早期机会。

3. 物理AI是本期最明确主线。 光轮智能、Shield AI、Saronic、Mind Robotics、自变量/千寻/星海图等共同指向同一趋势：资本愿意为“AI控制现实世界”的数据、硬件、执行和评测系统支付溢价。

4. 退出预期正在重新成为中国市场关键变量。 并购基金、创业板新标准、硬科技IPO预期改善，使人民币VC更愿意投“政策点名”的未来产业。

5. 后续重点追踪方向：物理AI数据基础设施、机器人量产供应链、国防/海事自主系统、AI云与网络基础设施、可控核聚变商业化节点、AI消费硬件的真实留存与复购。
