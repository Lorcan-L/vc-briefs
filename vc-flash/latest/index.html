    • 3. 赛道热度观察


  • AI Coding / 开发者工具：仍是美元VC最愿意给高估值的方向，但下一阶段会从增长叙事转向企业席位、留存、安全合规和工作流锁定能力。

  • 企业 Agent / Agentic workflow：资金从“聊天式助手”转向“可接系统、可衡量ROI、可替代流程”的执行层系统，种子轮金额被明显抬高。

  • 医疗AI：从药物发现扩散到临床运营、试验匹配、病历结构化和收入周期管理，投资人更偏好离现金流近的场景。

  • 能源/半导体/国防航天：AI算力扩张后的二阶基础设施需求继续释放，硬科技项目获得更稳定关注。

  • 中国硬科技：人民币资金集中在具身智能、商业航天、半导体设备/材料与国产替代，评估重点是产业订单和量产能力，不是demo热度。

    • 4. 值得关注的早期项目


  • Unframe：大额种子轮说明企业 Agent 正从演示工具进入流程级系统，若能绑定ERP/CRM/工单/财务等核心系统，商业化天花板高于普通Copilot。

  • 医疗运营AI项目：相比药物发现，临床运营与病历/试验工作流更容易形成短周期付费和明确ROI，值得跟踪A轮前数据。

  • 实时视频/世界模型早期公司：若能服务广告、电商素材、影视预演或机器人仿真数据，可能从内容工具升级为生产基础设施。

  • 中国具身智能供应链：优先看有量产客户、数据闭环和供应链议价能力的团队，弱化单次机器人展示。

    • 5. 分析师点评


    本期VC市场的关键词不是简单的“AI继续热”，而是“AI资本支出向可交付系统和供给侧基础设施迁移”。巨额frontier AI与AI Coding融资会继续制造估值锚，但更可投资的早期机会正在中间层：企业流程Agent、医疗运营、身份/风控、实时生成视频、具身数据、能源与半导体供应链。短期建议重点追踪三类验证指标：企业客户净留存、是否能接入核心业务系统、以及硬科技项目的订单/量产节奏。
