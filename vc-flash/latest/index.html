1. 本期核心判断

资本正在押注 AI 从模型层进入“可交付系统 + 物理世界 + 关键基础设施”：头部巨轮次继续抬高总量，但最新可操作的增量资金更集中在实时生成式视频、医疗AI、身份风控、深科技硬件、agentic security、AI数据中心互连、biotech临床资产与中国具身智能/半导体/低空经济等政策确定性赛道。

3. 赛道热度观察

  • 全球VC总量创纪录，但繁荣高度集中。 Crunchbase披露Q1全球VC约3000亿美元、6000家公司，AI约2420亿美元、占80%；OpenAI、Anthropic、xAI、Waymo四笔合计约1880亿美元。结论是：总量看似火热，但普通创业公司的融资环境并没有同步宽松。
  • AI投资从模型层外溢到交付层和基础设施层。 Reactor、Unframe、Triomics、Didit、Mintlify、nEye.ai、Artemis分别对应实时内容生成、企业交付、医疗流程、身份风控、AI文档、数据中心互连和安全运营，说明资本正在买“AI进入生产系统后的管道”。
  • Physical AI / 具身智能是中美共振主线。 美国侧关注Project Prometheus、Glydways、Waymo、国防自治系统；中国侧强调量产、订单、核心零部件、仿真数据与家庭/工厂场景。
  • 医疗AI与biotech更看重临床/支付路径。 Triomics、Signos代表医疗AI从诊断demo进入临床运营和消费健康；biotech tracker显示资金继续流向AI药设、免疫、肿瘤、呼吸疾病和临床读出明确的资产。
  • 硬科技成为AI capex外溢后的瓶颈资产。 算力、数据中心互连、能源、半导体、关键材料、机器人供应链和国防航天从“配套”变成独立投资主线。
  • 中国市场关键词是政策确定性 + 退出路径 + 量产交付。 新浪财经/深圳市投资基金同业公会称中国Q1新募基金数量和金额同比增长114.0%和79.8%，资金高度集中于AI、具身智能、半导体、商业航天/低空经济、生物医药、新能源链和高端装备。

  • Reactor：5900万美元出stealth，实时生成式视频开发者平台。若能降低视频生成延迟和工程门槛，可能成为AI视频应用层“卖铲子”公司。
  • Itera：1200万美元Seed，fluid circuit board。高技术风险，但具备底层硬件差异化，适合跟踪原型到可制造性的进展。
  • Didit：种子轮累计750万美元，AI-native身份与欺诈基础设施。Agent经济、自动交易、跨境金融和加密场景越活跃，身份/权限/风控越容易成为预算刚需。
  • Mintlify：4500万美元Series B，AI-readable documentation。AI coding agent要可靠调用API和SDK，文档机器可读化是“agentic dev infra”的隐性入口。
  • Artemis：7000万美元Growth，agentic cybersecurity。安全运营从告警看板走向自动检测、调查、修复，预算确定性强。
  • 中国具身智能早期链条：除头部整机外，更值得跟踪执行器、传感器、控制系统、仿真数据和场景交付商，因为整机热度会向供应链瓶颈扩散。

5. 分析师点评

本期VC市场不是简单“AI继续火”，而是 AI资本进入二次分层：

1. 巨轮次抬高总量，但不代表普通创业公司融资变容易。 Q1全球数据被少数frontier/physical AI交易强烈扭曲；真正的早期融资仍要证明技术壁垒、订单、合规路径或可量化ROI。

2. 资本从“模型能力”转向“交付与控制权”。 Unframe、Triomics、Didit、Reactor、Mintlify、Artemis都在解决AI进入生产系统后的交付、身份、临床信息、实时内容生成、文档可读、安全闭环问题，而不是只做模型包装。

3. 硬科技重新获得成长溢价。 半导体、数据中心、能源、国防航天、具身智能和关键材料不再是AI之外的支线，而是AI capex外溢后的瓶颈资产。

4. 中国市场更看重政策确定性和量产交付。 先进制造、AI、半导体、具身智能和商业航天吸金明显；判断项目质量时应优先看订单、研发强度、供应链能力和退出预期。

5. 后续30-60天建议追踪：AI coding与企业Agent的ARR兑现、AI数据中心互连/能耗优化、agentic security与身份风控、具身智能真实交付与退货率、AI医疗工作流续约率、核能/半导体/稀土磁材等战略硬科技。
