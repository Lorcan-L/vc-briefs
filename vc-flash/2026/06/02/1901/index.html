赛道热度观察

AI 继续主导资金总量，但集中度极高。Q1 2026 全球 VC $300B，AI $242B / 80%；四个超大轮次合计 $188B。

Agentic AI 正从横向概念转向垂直工作流替代：过去12个月 pure-play 样本 40笔、$1.413B，垂直 Agent 占 74.5% 金额。

中国市场资金集中在 AI、具身智能、半导体、商业航天/低空经济、创新药、新能源链和先进制造。量产订单正在成为估值锚。

值得关注的早期项目

  • Floatboat

    工作上下文/流程/方法由 Agent 接管，是企业 Agent 从聊天助手转向“工作空间操作层”的中国样本。

  • Didit

    AI-native 身份与欺诈基础设施，seed 总额 $7.5M；身份/KYC/风控是 Agent 商业化扩张的安全底座。

  • Trace

    Seed $3M，企业 Agent adoption/control layer；价值点在审批、监督、治理，不是单个 Agent 能力。

  • Itera

    $12M seed，fluid circuit board；非共识 deep tech，若进入先进计算/热管理链条，弹性高。

  • Waypoint Bio

    $20M Series A，AI-native biotech + CAR T；关键看实验验证闭环和临床推进速度。

分析师点评

VC 偏好正在从“Demo 能力”切到“交付能力”。能进入真实系统、形成数据闭环、用订单或 ROI 证明价值的项目，更容易拿到下一轮。
