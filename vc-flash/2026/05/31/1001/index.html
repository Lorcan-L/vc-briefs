5. 分析师点评

本期VC市场的关键变化不是“AI还热不热”，而是AI投资的风险偏好正在变硬：

1. 大额资金押注资本密集型AI。 Project Prometheus $10B、VAST Data $500M、True Anomaly $600M说明头部资本愿意为算力、数据、国防、物理世界AI提前买单。

2. 中早期机会更看重业务结果。 Agentic AI融资数据表明，垂直工作流替代拿走多数资金；创业公司需要证明它能替代某个岗位/流程，而不是只展示自动化能力。

3. 硬科技的估值逻辑回归。 国防、核能、半导体、稀土磁材、具身数据的共同特征是政策确定性和供给稀缺。VC在寻找“技术壁垒+产业需求+退出通道”的组合。

4. 中国市场的关键词是量产和订单。 具身智能不再只讲demo，融资材料中订单、收入、标准参与和产业客户会越来越重要。

后续建议重点追踪：

  • Project Prometheus这类physical AI巨额融资是否带动供应链标的；
  • AI基础设施公司是否能把算力/数据/仿真能力转化为持续毛利；
  • 垂直Agent在医疗、财务、采购、ITSM、安全等场景的留存和替代人力成本；
  • 中国具身智能企业从融资热转向交付、毛利和海外市场的速度。
