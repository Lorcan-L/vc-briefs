VC投资动态调研｜2026-06-05 17:01

1. 本期核心判断

资本继续押注 AI 落地真实经济所需的“稀缺控制层”：数据中心网络、电力/能源、世界模型、机器人训练数据、垂直 Agent 工作流、国防航天、半导体与 AI-native healthcare/biotech。

2. 本期重点交易

  • Maxwell Power｜Strategic Capital / Project Finance｜$750M｜Fairtide Partners｜能源基础设施、储能与太阳能部署

可信度：高。多源融资 roundup 均提及；这不是传统股权轮次，而是 AI 用电压力下能源开发平台拿到大额资本承诺的信号。

  • DriveNets｜Series D｜$410M｜Bessemer Venture Partners、Atreides Management；AMD 等参投｜AI 数据中心网络 / Ethernet fabric

可信度：高。公开报道称累计融资约 $1B、secured business 超 $1B 且现金流为正；AI 集群网络层正在从后台 IT 变成投资主赛道。

  • Tripo AI｜Series A+ / A++｜近 $200M｜Ince Capital、长三角科技基金等｜AI 3D / world model

可信度：中高。中英文报道交叉出现；资金用途指向 3D 生成、持久化世界模型、数据与基础设施。

  • Mecka AI｜Series A + follow-on｜$60M｜Framework Ventures；Menlo Ventures、SV Angel、Kindred Ventures 等｜机器人训练数据

可信度：中高。披露为 $25M Series A + $35M follow-on；核心看点不是机器人本体，而是人类行为/运动数据的采集和商业化。

  • Valar Atomics｜Series B｜$340M｜未列明｜核能 / 工业电力 / Power Grid

可信度：高。美国 2026年4月大型融资统计中位列 Series B 前列；AI 电力约束持续把核能、储能和电网能力推入 VC 主流视野。

  • 中国具身智能与机器人融资集群｜多轮次｜4月约8家公司合计超200亿元人民币｜今日资本、高瓴创投、BV百度风投等参与｜具身智能、机器人、先进制造

可信度：中高。中文来源偏宏观统计，但与国内政策硬科技资金流一致，说明人民币基金仍围绕“政策确定性 + 量产可见性”集中出手。

3. 赛道热度观察

AI 基础设施：资金流向瓶颈层，而非泛应用层

本轮检索里最强的共同信号，是资本不再只追“模型能力”或“AI wrapper”，而是追 AI 规模化落地绕不开的输入：网络、电力、算力效率、数据中心、机器人数据、世界模型和企业系统控制层。这类资产更接近“收费闸口”，议价权强于普通应用。

Agentic AI：垂直工作流继续胜过泛用助手

Agentic AI 融资线索显示，VC 更偏好能接管具体岗位/流程的公司：SOC、会计、采购、医疗后台、金融犯罪合规、研发数据处理等。Seed 与 Series A 仍有机会，但前提是接系统、拿权限、交付可量化 ROI，而不只是聊天界面。

能源与电力：AI capex 外溢成独立投资主题

Maxwell Power、Valar Atomics 等交易说明，AI 的约束正在从 GPU 扩展到电力和能源项目融资。未来 12-24 个月，储能、电网调度、核能、小型模块化反应堆、数据中心电力开发平台会继续被科技投资人重新定价。

中国硬科技：资金集中，不是普惠复苏

中文线索显示，先进制造、半导体、具身智能、商业航天/低空经济、生物医药与新能源链仍是人民币基金重点。市场并非全面回暖，而是向“政策背书 + 技术壁垒 + 退出路径”的项目集中。

4. 值得关注的早期项目

  • 机器人训练数据平台：Mecka AI 代表一个重要方向——如果具身智能数据无法像网页文本一样低成本获取，上游数据平台会成为稀缺收费节点。
  • Agent control / execution layer：权限、工具调用、人类审批、审计、可观测性与安全控制，是企业部署 Agent 的必要中间层。
  • AI-native 专业服务 Agent：采购、会计、SOC、医疗后台、合规等高人力成本场景，更容易从 Seed/Series A 走到收入验证。
  • 能源项目融资平台：懂电力资产开发、并能与数据中心需求绑定的团队，可能成为 AI 基础设施周期里的“金融工程型科技公司”。
  • AI-native biotech / healthcare workflow：不是泛 AI 医疗叙事，而是能进入临床流程、治疗设计、实验自动化或医院运营闭环的公司。

5. 分析师点评

1. VC 市场不冷，冷的是没有稀缺性的 AI 应用。 资金仍在出手，但集中给能控制关键资源、关键流程、关键数据的公司。

2. AI 投资进入“资源约束叙事”。 电力、网络、数据、机器人训练环境、世界模型、临床实验系统，都是下游必须购买或绕不开的环节。

3. 早期项目窗口仍在，但要证明工作流接管。 只做更好看的 UI 或通用助手很难拿钱；能嵌入系统、承担结果、积累专有数据，才有 Series A/B 逻辑。

4. 人民币基金更重视硬科技确定性。 半导体、具身智能、商业航天/低空经济、生物医药、新能源链会继续吸金，但商业化证据会被更严格审视。

5. 后续追踪方向：AI 数据中心 networking、核能/储能开发平台、机器人数据采集、企业 Agent 控制层、AI-native healthcare/biotech、中国具身智能量产订单。
