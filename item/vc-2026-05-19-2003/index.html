摘要

本期全球VC市场仍由AI超大额融资主导，但值得注意的是，资金并非只投向纯软件或大模型公司，而是明显转向“AI + 物理世界”：AI基础设施、自动驾驶、具身智能、国防航天、核能/聚变、半导体、药物发现等资本密集型方向。Q1 2026全球VC投资出现历史级放量，KPMG统计为3309亿美元，Crunchbase统计约3000亿美元；二者口径不同但方向一致：资本高度集中于少数AI与硬科技巨头交易。

2. 赛道热度观察

2.1 AI仍是绝对主线，但资金向“硬AI”迁移

  • KPMG：Q1 2026全球VC投资3309亿美元，环比从1286亿美元大幅上升；10笔20亿美元以上超大额融资贡献超过2060亿美元。
  • Crunchbase：Q1全球创业投资约3000亿美元，约6000家公司获投；AI公司获得2420亿美元，占全球VC总额约80%。
  • AlleyWatch：2026年4月美国VC投资208亿美元，442笔交易；AI吸收151.8亿美元，占73%。其中Project Prometheus一笔100亿美元占美国当月总额48.1%。

判断：AI已经从“应用软件投资主题”变为“资本开支型基础设施周期”。算力、数据中心、机器人、自动驾驶、国防、半导体等重资产环节正在吃掉更大票面。

2.2 资本高度集中，平均数失真

  • Crunchbase指出，OpenAI、Anthropic、xAI、Waymo四家公司合计融资1880亿美元，占Q1全球VC投资约65%。
  • AlleyWatch显示，4月美国市场若剔除Project Prometheus，剩余441笔交易合计108亿美元，仍属健康，但与 headline 208亿美元差距极大。
  • Late-stage交易被巨额outlier扭曲；Series A相对更能反映底层活跃度。AlleyWatch 4月美国Series A共98笔，平均2000万美元，分布更均衡。

判断：当前市场不是全面牛市，而是“少数巨头交易 + 中早期结构性活跃”的组合。看融资总额容易误判热度，必须看剔除outlier后的中位数与交易数量。

2.3 中国一级市场：具身智能进入“订单与量产”筛选期

中国证券报梳理显示，红杉中国、高瓴创投等头部机构Q1重点布局人工智能与实体产业融合：具身智能核心零部件、AI硬件、垂直行业应用。

代表交易：

  • 自变量机器人：10亿元A++轮，红杉中国、深创投等参与。
  • 千寻智能：近20亿元A轮，红杉中国、云锋基金等参与；生产线已在宁德时代基地投运。
  • 灵猴机器人：数亿元B轮，红杉中国等参与；已具备规模化交付能力。
  • 星海图：B轮，高瓴创投等老股东追投；公司表示正从开发者迈向生产力价值拐点。

判断：中国具身智能的融资逻辑正在从“demo与概念”转向“量产能力、订单、交付场景”。

2.4 生物医药：AI制药与临床资产并行升温

  • Isomorphic Labs完成21亿美元Series B，投资方包括Thrive Capital、Alphabet、GV、MGX、Temasek、CapitalG等，用于扩展AI药物设计引擎与管线。
  • CellCentric完成2.2亿美元Series D，用于推进inobrodib多发性骨髓瘤关键临床。
  • Windward Bio完成1.65亿美元crossover融资，推进长效免疫疾病抗体药物。
  • CREATE Medicines完成1.22亿美元Series B，发展in vivo CAR项目。
  • Fathom Therapeutics完成4700万美元Series A，主打物理+AI小分子设计。

判断：生物医药融资恢复并不均匀。最受欢迎的是两类：一是AI/物理模拟驱动的平台，二是临床读出较近、可被IPO/M&A验证的资产型公司。

2.5 Climate/能源：聚变、核能、地热等“稳定清洁电力”获大票

New Market Pitch统计，Q1 2025至Q1 2026期间，气候科技公开融资42笔、总额超过64亿美元；其中聚变、地热、核能等firm clean power融资超过28亿美元。

中国侧，东昇聚变完成数亿元天使轮，红杉中国、IDG资本、高瓴、中科创星等入局；其路线为氘-氦3聚变，计划分阶段推进净能量增益目标。

判断：AI带来的算力用电需求，使能源科技与AI基础设施形成强耦合。资本开始把核能、聚变、电网、储能视作AI时代底层约束的解决方案。

4. 分析师点评

4.1 VC市场进入“超级分化周期”

表面看，Q1 2026全球VC投资刷新历史纪录；实际看，资金极度集中在少数基础AI公司和硬科技基础设施公司。中小项目并未平均受益，且早期交易数量存在下降或滞后披露问题。投资人真正追逐的是：

1. 有机会成为AI时代基础设施的公司；

2. 能把AI落到物理世界、工业世界、国防世界的公司；

3. 能解决算力、电力、数据、自动化瓶颈的公司；

4. 有明确临床读出或产业订单的硬科技资产。

4.2 “软件高毛利叙事”正在让位于“资本开支护城河叙事”

上一轮SaaS周期强调轻资产、高毛利、快速扩张；本轮AI/硬科技周期则越来越像能源、半导体、云计算早期的混合体：需要巨额资本、长期建设、供应链能力、政策协同和战略客户。护城河更深，但资本消耗也更大。

4.3 中国市场的关键词是“技术落地”

中国头部人民币/美元基金对具身智能、AI硬件、低空经济、商业航天、聚变等方向的布局，核心判断不是单纯追逐概念，而是寻找已经出现工程化拐点、但市场尚未形成统一定价共识的标的。订单、量产、交付、场景验证会成为下一轮融资的关键门槛。

4.4 值得持续追踪的方向

1. AI基础设施：数据中心网络、存储、芯片、冷却、电力调度。

2. Physical AI：机器人、自动驾驶、无人系统、工业自动化。

3. 国防与航天科技：双用途技术、空间感知、无人平台、稀土磁材。

4. AI制药：不仅看平台能力，更看管线、合作方和临床节点。

5. 能源基础设施：核能、聚变、地热、储能、电网软件，尤其是与AI算力增长绑定的方向。

6. 中国具身智能供应链：核心零部件、量产制造、行业场景集成商。

5. 结论

本期VC市场的核心矛盾是：资金很充沛，但只对少数方向极度充沛。AI仍是主轴，但已经从模型公司外溢到算力、电力、机器人、国防、生物医药和先进制造。对投资跟踪而言，下一阶段不应只看融资金额，而要重点看三类信号：

  • 剔除巨额outlier后的真实市场活跃度；
  • 被投企业是否有订单、临床节点、量产能力或战略客户；
  • 资本进入的是“概念热度”还是“基础设施瓶颈”。

我的判断：未来6-12个月VC市场将继续呈现“AI巨头吸金 + 硬科技结构性升温 + 普通软件融资承压”的状态。最值得追踪的是AI基础设施、能源基础设施、具身智能产业链和AI制药临床资产。
