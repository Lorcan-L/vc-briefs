VC投资动态调研｜2026-06-01 20:30

1. 本期核心判断

资本继续押注“AI可交付系统 + 物理世界 + 关键基础设施”：AI仍是资金最大吸盘，但更强信号是资金正在从模型叙事迁移到企业工作流、国防航天、能源电力、半导体、具身智能和可退出的硬科技资产。

2. 本期重点交易

  • Project Prometheus｜Late-stage｜$10.0B｜领投方未披露｜Foundational AI / physical-world AI

来源：AlleyWatch。该交易占美国4月VC资金48.1%，显著扭曲总量口径；更适合作为“资本愿意为物理世界AI下注”的极端样本，而非普遍估值锚。

  • Slate Auto｜Late-stage｜$650M｜未披露｜EV / advanced manufacturing

来源：AlleyWatch。非纯软件，说明硬件制造与交通能源仍可获得大额成长资本。

  • True Anomaly｜Late-stage｜$600M｜未披露｜Aerospace / AI / national security

来源：AlleyWatch。国防航天与AI叠加，是本期非消费AI中最强方向之一。

  • VAST Data｜Late-stage｜$500M｜未披露｜AI infrastructure / data OS

来源：AlleyWatch。AI infra仍是大额融资核心入口，尤其是数据层、存储和算力编排。

  • Vulcan Elements｜Late-stage｜$431M｜未披露｜Rare-earth magnets / defense manufacturing

来源：AlleyWatch。稀土磁材、先进制造和国防供应链成为硬科技资金高地。

  • SiFive｜Late-stage｜$400M｜未披露｜RISC-V semiconductor IP

来源：AlleyWatch。半导体IP和国产/自主架构叙事继续吸引长期资本。

  • Valar Atomics｜Series B｜$340M｜未披露｜Nuclear energy / power grid

来源：AlleyWatch。能源基础设施成为AI算力需求外溢下的关键投资主题。

3. 赛道热度观察

美国市场：AI headline 很热，但硬科技广度更值得看

AlleyWatch 统计显示，美国4月VC融资 $20.80B / 442 deals，同比资本增长63.9%、交易数增长21.1%；AI公司吸收 73.0% 资金，达到 $15.18B / 189 deals。但剔除 Project Prometheus 单笔 $10B 后，剩余市场仍有 $10.80B / 441 deals，说明底层并非只有一个超级交易，而是多赛道同时活跃。

Agentic AI：从“通用Agent”转为“垂直工作流替代”

New Market Pitch 对2025年6月至2026年5月公开股权融资的统计显示，agentic AI 市场包含 40笔交易、$1.413B 融资；其中垂直AI Agent占 26/40 笔、$1.052B，占披露资本74.5%。这说明资本不再满足于“能调用工具”的横向Agent，而是更看重能拥有任务结果、接入系统、替代具体岗位/流程的垂直产品。

中国市场：政策确定性 + 硬科技退出预期共同驱动

投资界/清科数据称，2026年一季度中国股权投资新募集基金 1,913只 / 5,590.69亿元，同比增长 114.0% / 79.8%；投资案例 2,568起 / 2,344.25亿元，同比增长 4.9% / 15.4%。新浪财经转载深圳市投资基金同业公会文章称，4月仅8家具身智能企业融资超 200亿元。资金明显集中于AI、具身智能、半导体、商业航天/低空经济、生物医药与新能源链。

4. 值得关注的早期项目

  • Agentic AI早期基础设施：如云端执行环境、权限/审批控制、Agent安全、工具集成、计费和观测层。理由：New Market Pitch 数据显示，Agent执行基础设施和开发平台虽交易数少于垂直应用，但会成为垂直Agent规模化的底座。
  • 垂直工作流Agent：财税审计、医疗行政、采购、供应链、SOC安全、身份欺诈与合规调查。理由：资本偏好能嵌入系统并交付ROI的“工作流替代”，而非泛聊天入口。
  • 电力/核能/储能与AI算力配套：Valar Atomics 的Series B大额融资说明，AI算力热正在向电力供给侧外溢。
  • 中国具身智能供应链：核心零部件、运动控制、数据采集、工厂/家庭场景落地项目。理由：估值溢价明显，但更应看量产、订单和场景验证。
  • 半导体IP与先进封装/制造材料：SiFive、Vulcan Elements 等案例说明，底层硬件和关键材料具备跨AI、国防、制造的复合需求。

5. 分析师点评

本期市场最容易被“Project Prometheus $10B”误导。正确读法是：超级AI交易拉高了融资总量，但真正的可投资趋势是资本在寻找三类资产：

1. 能把AI变成可交付业务结果的垂直系统：企业Agent、医疗运营、财税审计、安全/风控。

2. 支撑AI扩张的关键基础设施：数据层、算力/芯片、电力、存储、工作流编排、Agent安全。

3. 物理世界和政策确定性强的硬科技：国防航天、核能、电动车制造、具身智能、半导体与材料。

短期看，VC市场仍是“强者恒强 + 赛道集中”：顶级项目能融巨额，普通SaaS和缺少技术壁垒的项目仍难。中期最值得持续追踪的是：vertical agent ROI、physical AI量产节点、AI电力供给、半导体供应链国产替代、医疗/金融合规AI闭环。

可信度与来源

  • AlleyWatch：高可信，给出美国4月融资结构和Top deals，但其数据为自有口径，需要与PitchBook/Crunchbase交叉验证。
  • New Market Pitch：中高可信，适合作为agentic AI细分市场样本库，但非全市场口径。

  • 新浪财经转载深圳市投资基金同业公会：中高可信，适合赛道观察；部分具身智能具体公司需后续逐笔核验。
