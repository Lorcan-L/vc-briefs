一、摘要判断

本期VC市场继续呈现“极端集中 + 硬科技化 + AI外溢到物理世界”的结构性特征。

  • 全球层面：Crunchbase数据显示，2026年Q1全球创业投资约 3000亿美元、约 6000家 公司获投；AI相关公司获得约 2420亿美元，占全球VC融资约 80%。资金不是均匀复苏，而是被少数AI基础设施、前沿模型、自动驾驶、机器人、数据中心、半导体与国防/航天项目强力吸走。
  • 美国4月：AlleyWatch统计的美国4月大额融资中，Project Prometheus 单轮 100亿美元 位列第一；其余大额交易集中在AI基础设施、国防航天、核能、EV、半导体、数据平台和Agentic AI。
  • 中国市场：新浪财经转载深圳市投资基金同业公会文章显示，2026年一季度中国股权投资市场新募基金数量和金额分别同比增长 114.0% 和 79.8%；资金主要集中于先进制造、人工智能、半导体、具身智能、商业航天、低空经济、生物医药、新能源链等政策确定性强的方向。上海4月中旬单周披露融资金额约 47.05亿元，它石智航 4.55亿美元Pre-A、鹏熙半导体 超10亿元战略融资 等具代表性。
  • 早期项目亮点：早期轮次里，AI基础设施、AI Agent工具链、机器人核心零部件、AI制药、创新药平台、国防机器人出现明显大额早期融资，说明早期资金也在追逐“能快速形成技术壁垒和产业订单”的方向。

三、赛道热度观察

1. AI仍是绝对主线，但资金正在从“软件叙事”扩散到“物理经济”

AI本身继续吸收最大规模资本。Crunchbase数据显示，2026年Q1全球AI相关融资约 2420亿美元，占全球VC融资 80%。但从4月美国大额融资结构看，资金已不只是投模型与SaaS，而是进入：

  • AI基础设施：VAST Data、Aria Networks、SiFive、数据中心自动化、网络硬件、存储/数据库/计算一体化。
  • Agentic AI：Factory、Hightouch、Parallel、Avoca、Rogo等，集中在开发者工具、营销、金融研究、客服/销售自动化。
  • Physical AI：Project Prometheus、Scout AI、Skydio、Reliable Robotics、True Anomaly、Hermeus等，覆盖国防机器人、无人机、自动飞行、航天安全、高超音速航空。

判断：AI投资的关键词正在从“模型能力”转为“算力/数据/能源/制造/国防/真实场景闭环”。纯应用若没有明确ROI、分发渠道和行业数据壁垒，融资难度会明显上升。

2. 国防科技、航天与关键供应链成为美国大额融资集中区

AlleyWatch 4月榜单中，True Anomaly（航天安全，6亿美元）、Vulcan Elements（稀土磁体，4.307亿美元）、Hermeus（高超音速航空，2亿美元）、Scout AI（国防机器人，1亿美元）均进入大额交易名单。资本逻辑与传统互联网投资不同，核心驱动来自：

  • 地缘政治与供应链安全；
  • 政府/军方预算可见性；
  • 技术门槛高、竞争格局相对集中；
  • AI能力向无人系统和空间资产管理外溢。

3. 能源、核能与电网侧投资升温，本质是AI算力的“瓶颈外包”

Valar Atomics完成 3.4亿美元Series B，Glydways、Slate Auto等交通/能源相关项目也获得大额融资。AI数据中心扩张带来的电力、冷却、材料、网络和边缘基础设施压力，正在把VC投资带到过去更偏PE/产业资本的重资产领域。

4. 生物医药出现结构性修复：AI制药与临床确定性资产更受青睐

Xtalks tracker显示，2026年生物医药融资覆盖AI制药、免疫、肿瘤、神经、心代谢、ADC、in vivo CAR等方向。代表交易包括：

  • Isomorphic Labs：21亿美元Series B，AI驱动药物设计；
  • Beeline Medicines：3亿美元Series A；
  • CellCentric：2.2亿美元Series D，用于多发性骨髓瘤关键临床；
  • Windward Bio：1.65亿美元 crossover，长效免疫抗体；
  • Coultreon Biopharma：1.25亿美元Series A，自免疾病；
  • CREATE Medicines：1.22亿美元Series B，in vivo CAR；
  • Oricell Therapeutics：超1.1亿美元pre-IPO，实体瘤CAR-T。

判断：生物医药并非全面回暖，而是向两端集中：一端是AI/平台型发现能力，另一端是临床进展明确、读出节点清晰的资产。

5. 中国资金更偏“政策确定性 + 产业落地”

中国市场的热度集中在：

  • 具身智能/机器人：它石智航4.55亿美元Pre-A、维他动力近5亿元Pre-A、颗粒进化、苇熠科技、创屹科技等；
  • 半导体与先进封装：鹏熙半导体超10亿元战略融资、道宜半导体A轮；
  • 商业航天/低空经济：百灵航天、沃飞长空等；
  • AI应用：法律AI、医疗管理系统、智慧停车等。

新浪财经转载文章指出，中国2026年一季度新募基金数量与金额同比明显增长，但资金分配极度集中：先进制造、AI、半导体几乎垄断关注度。融资容易的企业通常具备政策红利、技术壁垒、可验证产品化进展和明确营收/订单。

五、分析师点评

1. VC市场不是“全面复苏”，而是“强主题牛市”

表面上融资规模大幅回升，但资金高度集中于少数确定性主题。Q1全球VC融资规模创纪录，并不代表普通创业公司融资环境显著改善。相反，市场对“可被AI或政策明确放大的资产”给出超高溢价，而对缺乏技术壁垒、订单验证或退出路径的项目依旧谨慎。

2. AI投资进入资本开支时代，轻资产软件不再天然占优

这一轮AI投资正在从模型/应用，进入算力、数据中心、网络、能源、半导体、机器人、自动驾驶、国防航天。VC正在接受更高CapEx、更长交付周期和更复杂合规，只要终局市场足够大、订单方足够确定。

3. 中国市场的核心变量是“政策确定性 + 退出通道”

中国一季度新募基金回暖，与创业板新标准、并购基金、政策性金融工具、六大支柱/未来产业等制度环境有关。资金不是追逐流量互联网，而是追逐能解释给LP、政府和产业方的硬科技资产。

4. 早期项目估值正在两极化

早期轮次中出现1亿美元级别A轮甚至种子轮，表明资本对少数明星团队/硬科技方向愿意提前定价。但这也会带来后续融资压力：如果18-24个月内不能证明产品、订单或临床/技术里程碑，下一轮可能面临估值倒挂。

5. 值得持续追踪的方向

优先级建议：

1. AI基础设施：数据中心网络、存储/数据库/计算一体化、GPU利用率优化、AI安全运维、模型部署成本优化。

2. Agentic AI的高ROI垂直场景：金融研究、法务、营销、客服、软件工程，但必须看真实工作流嵌入和付费留存。

3. Physical AI/具身智能：机器人本体、核心零部件、运动控制、仿真训练、工业/仓储/家庭场景落地。

4. 国防航天与低空经济：无人系统、空间态势感知、卫星运营、eVTOL配套；政策和订单确定性强，但合规门槛高。

5. AI制药与创新药平台：关注临床读出明确、平台能力可复用、合作药企强的项目。

6. 能源与关键材料：核能、小型电网、稀土磁体、电力设备、冷却系统，AI算力扩张会持续外溢需求。

六、风险提示

  • 大额融资存在数据滞后、轮次定义差异、媒体二次转载误差；未深读或抓取失败页面需后续交叉验证。
  • AI相关估值扩张过快，可能出现“融资额领先商业化”的泡沫风险。
  • 国防、航天、核能、医疗等方向监管和交付周期长，对普通VC基金期限结构提出挑战。
  • 中国硬科技项目受政策、产业周期、IPO节奏影响显著，退出通道改善尚需持续观察。
