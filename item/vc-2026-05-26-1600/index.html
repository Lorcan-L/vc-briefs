VC投资动态调研｜2026-05-26 16:00

1. 本期核心判断

资本正在继续押注“AI基础设施与物理世界落地”的双主线：一端是AI/Agentic AI/算力基础设施的超级集中融资，另一端是国防航天、核能、先进制造、具身智能等硬科技资产获得更高确定性溢价。

2. 本期重点交易

  • Project Prometheus / Late-stage / 100亿美元 / 未披露 / 基础AI、Physical AI

- 来源：AlleyWatch；可信度：高。该交易占美国4月VC融资总额48.1%，显著扭曲当月late-stage均值。

  • Slate Auto / Late-stage / 6.5亿美元 / 未披露 / EV、汽车制造

- 来源：AlleyWatch；可信度：高。非AI硬件与制造仍能获得巨额后期资本。

  • True Anomaly / Late-stage / 6亿美元 / 未披露 / 航天、AI、国家安全

- 来源：AlleyWatch；可信度：高。国防科技成为AI之外最强非软件主题之一。

  • VAST Data / Late-stage / 5亿美元 / 未披露 / AI基础设施、数据平台

- 来源：AlleyWatch；可信度：高。AI基础设施继续吸收大额资金。

  • Vulcan Elements / Late-stage / 4.31亿美元 / 未披露 / 稀土磁体、国防制造

- 来源：AlleyWatch；可信度：高。关键材料与供应链安全被机构资金重估。

  • Valar Atomics / Series B / 3.4亿美元 / 未披露 / 核能、工业电力

- 来源：新浪财经转载深圳市投资基金同业公会；可信度：中。国内具身智能融资热度突出。

3. 赛道热度观察

AI与Agentic AI：资本极端集中，但应用层开始分化

  • Crunchbase数据显示，2026年Q1全球VC融资约3000亿美元，其中AI获得2420亿美元，约占80%。OpenAI、Anthropic、xAI、Waymo四笔巨额交易合计约1880亿美元，占全球季度融资约65%。
  • AlleyWatch数据显示，美国4月AI公司获得151.8亿美元，占当月美国VC资金73.0%。
  • Agentic AI已形成独立融资主题。New Market Pitch统计的100家Agentic AI公司累计融资约98亿美元，Top 10吸走约60%，最活跃投资方包括Andreessen Horowitz。代表方向包括法律AI、客服AI、AI软件工程、AI招聘、AI金融分析、AI客服与医疗工作流。

硬科技与物理世界：AI之外的第二条确定性曲线

  • 美国4月Top 10交易里，国防航天、稀土材料、半导体IP、核能、电动交通占据多个席位。
  • 非纯软件项目的共同特征是：政策/国家安全/能源约束强，资本开支大，但一旦验证商业化，后期资金愿意给高额票据。
  • 国内市场同样呈现“政策确定性+硬科技”的资金流向：新浪财经/深圳市投资基金同业公会文章指出，2026年一季度中国股权投资新募基金数量同比+114.0%、金额同比+79.8%，资金集中在AI、具身智能、半导体、商业航天、低空经济、创新药、新能源链等方向。

早期轮次：数量仍活跃，但钱更挑剔

  • AlleyWatch美国4月数据：Early-stage 254笔、11.3亿美元；Series A 98笔、19.6亿美元；Series B 40笔、18.3亿美元。
  • Series A平均约2000万美元，分布相对均衡，是观察底层投资人信心的更干净指标。
  • Seed/天使机会仍存在，但更偏“有明确企业场景/技术壁垒/产业客户”的项目；纯概念AI应用的融资难度上升。

4. 值得关注的早期项目

  • Tribal：1000万美元Seed，企业原生AI平台。看点是创始团队来自Salesforce、Wix、Spot.io等复杂企业系统背景，切入点不是通用agent，而是企业系统原生AI。
  • H / Agentic AI for enterprises：New Market Pitch统计其Seed即融2.2亿美元，投资方包括Accel、UiPath、Bpifrance。看点是企业级action agent的高资本门槛与欧洲AI主权叙事。
  • /dev/agents：5600万美元Seed，定位AI agents操作系统，投资方包括Index Ventures、CapitalG。看点是Agent基础设施入口竞争。
  • 东昇聚变：数亿元天使轮，红杉中国、IDG资本、高瓴、中科创星等入局。看点是核聚变作为未来能源方向，天使轮即获得一线机构抱团，说明深科技长周期资产正在被提前抢筹。
  • 维他动力：近5亿元Pre-A，消费级机器狗/具身智能。看点是国内机器人从工业场景向消费/家庭端延展，资本关注产品化与量产能力。

5. 分析师点评

1. VC市场并非全面回暖，而是结构性繁荣。 总量被AI超级融资显著推高，但如果剔除Project Prometheus、OpenAI、Anthropic、xAI等巨额交易，底层市场更像是“健康但高度挑剔”的修复周期。

2. AI投资从“模型崇拜”转向“基础设施+行业闭环”。 最有吸引力的资产不只是大模型公司，还包括AI数据基础设施、交付平台、AI安全、垂直Agent、企业工作流自动化。

3. 物理世界资产正在重新定价。 国防、航天、核能、稀土、半导体、机器人等赛道的融资热度显示，VC正在把AI能力与产业控制力绑定起来评估。

4. 中国资金更重视政策确定性与退出路径。 具身智能、半导体、低空经济、商业航天、创新药等方向受益于政策与产业协同；传统增长故事若缺乏技术迭代，融资仍会困难。

5. 后续值得追踪的方向： Agentic AI垂直应用、AI基础设施、机器人量产链、核能/电力供给、国防科技、半导体与关键材料、AI+药物研发。
