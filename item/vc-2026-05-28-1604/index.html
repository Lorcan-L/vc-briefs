3. 赛道热度观察

AI：主线从“模型”变成“物理世界 + 基础设施”

  • AlleyWatch数据显示，美国4月VC融资达208亿美元、442笔交易，同比资本增长63.9%、交易数增长21.1%；AI公司吸走151.8亿美元，占全部资本73.0%。
  • Project Prometheus 单笔100亿美元融资极大抬高总量，但剔除后美国市场仍有108亿美元、441笔交易，说明底层活跃度并不只靠一个超级项目支撑。
  • VAST Data、光轮智能、Didit、Unframe等交易共同指向：资本更偏好“把AI落进企业/物理世界/数据基础设施”的公司。

防务、航天、能源和先进制造：非AI资金也在硬科技化

  • True Anomaly、Vulcan Elements、Hermeus等防务/航天制造交易合计超过12亿美元，说明国家安全、供应链自主和先进制造正在成为AI之外的大额资金入口。
  • Valar Atomics 3.4亿美元Series B代表能源基础设施被重新定价：投资人把核能、聚变、工业电力看作未来AI算力与电气化的底座，而不是单纯气候科技叙事。

Biotech/AI制药：资金集中在“临床节点 + 平台转化”

  • Xtalks tracker显示，Windward Bio、CellCentric、Coultreon、CREATE、Fathom、Isomorphic等融资集中在免疫、肿瘤、呼吸、in vivo CAR与AI/物理药物设计。
  • 投资人更愿意投临床节点明确、未来12-24个月有读出窗口，或能证明AI/物理模型进入lab-in-the-loop的项目。

中国市场：政策确定性 + 订单验证共同决定融资效率

  • 新浪财经/深圳市投资基金同业公会提到，2026年一季度中国股权投资市场新募基金数量和金额分别同比增长114.0%和79.8%，但资金高度集中在先进制造、AI、半导体、具身智能、商业航天、创新药、新能源链。
  • 光轮智能案例特别典型：不是机器人本体，而是“物理AI时代卖铲人”。其披露2026Q1新增订单5.5亿元、单季收入超过2025全年，说明VC正在用真实订单和复售率给AI基础设施重估。

4. 值得关注的早期项目

  • Didit：AI-native identity and fraud infrastructure，累计种子轮750万美元。身份、KYC、风控会成为Agent和AI交易系统商业化的基础层，金额小但方向关键。
  • Tribal：1000万美元Seed，企业原生AI平台，由Salesforce/Wix/Spot.io背景团队创立。值得看其是否能从“企业Agent平台”跑出明确垂直工作流和付费闭环。
  • Fathom Therapeutics：4700万美元Series A，AI+物理模拟小分子设计。若能把蛋白动态建模推进到真实候选物优化，是AI制药早期平台的高质量样本。
  • Serif Biomedicines：Flagship Pioneering 5000万美元启动，Modified DNA平台。属于高技术风险、平台潜力大的遗传药物早期项目。
  • 光轮智能：严格说已进入独角兽阶段，但仍是具身/物理AI基础设施里的关键观察样本；后续重点不是估值，而是订单复售率、海外交付和行业标准话语权。

5. 分析师点评

1. 这不是“VC全面回暖”，而是资本向少数高确定性基础设施集中：AI基础设施、防务航天、能源电力、半导体、物理AI数据、临床节点明确的biotech。

2. AI赛道正在被重新分层：模型公司负责吸走超级大额资金，应用公司必须证明企业预算、交付效率和续费；卖铲公司则围绕数据、评测、算力、身份、安全和采购形成新入口。

3. 硬科技融资的共同语言变成“国家战略 + 供应链安全 + 可量化订单/临床/工程里程碑”。讲愿景仍有用，但没有交付证据会很难拿到好价格。

4. 未来30天建议持续追踪三类信号：Project Prometheus类物理AI巨额融资是否带动上下游；中国具身数据/仿真公司订单复售是否持续；能源/核能项目是否继续获得产业资本和政府基金参与。
