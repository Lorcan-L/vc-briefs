VC投资动态调研｜2026-05-26 14:30

1. 本期核心判断

资本正在继续押注“AI + 真实世界基础设施”：AI基础模型/Agentic AI仍是最大吸金主题，但新增资金明显向具身智能、国防航天、能源、半导体和AI制药等硬科技场景扩散。

2. 本期重点交易

  • Project Prometheus｜Late-stage｜100亿美元｜未披露｜基础AI / physical-world AI

- AlleyWatch 披露其单笔交易占美国4月VC总额48.1%，是本期最强烈的资金集中信号。

  • Slate Auto｜Late-stage｜6.5亿美元｜未披露｜电动车 / 制造

- 美国4月第二大交易，说明硬件制造类后期项目仍可获得大额资本支持。

  • True Anomaly｜Late-stage｜6亿美元｜未披露｜航天 / AI / 国家安全

- AI与国防航天交叉赛道继续升温。

  • VAST Data｜Late-stage｜5亿美元｜未披露｜AI基础设施 / 数据系统

- AI算力与数据基础设施仍是机构资金重点方向。

  • Vulcan Elements｜Late-stage｜4.31亿美元｜未披露｜稀土磁体 / 国防制造

- 反映先进制造与供应链安全主题升温。

  • Valar Atomics｜Series B｜3.4亿美元｜未披露｜核能 / 电网

- 单笔交易拉高Series B均值，能源基础设施成为非AI大额融资代表。

  • Windward Bio｜Crossover｜1.65亿美元｜OrbiMed、Novo Holdings、RA Capital、Sanofi Ventures等｜免疫/呼吸疾病抗体药物

- 生物医药后期临床读出前融资仍有吸引力。

  • CREATE Medicines｜Series B｜1.22亿美元｜Newpath Partners、ARCH Venture Partners等｜in vivo CAR / 自免与肿瘤

- 创新疗法平台型公司继续获得中后期资本支持。

  • Coultreon Biopharma｜Series A｜1.25亿美元｜Sofinnova、Forbion、Novo Holdings、Regeneron Ventures等｜自免创新药

- A轮即获得大额融资，说明临床路径清晰的免疫赛道热度较高。

  • Unframe｜追加融资｜5000万美元｜Highland Europe｜企业AI交付平台

- 企业级AI落地平台仍有持续融资能力。

  • Tribal｜Seed｜1000万美元｜未披露｜企业原生AI平台

- 早期Agentic/Enterprise AI仍有种子轮机会。

  • 维他动力｜Pre-A｜近5亿元人民币｜今日资本、高瓴创投、BV百度风投等｜消费级具身智能/机器狗

- 中国市场具身智能继续获得顶级机构下注；新浪财经援引4月8家具身智能企业融资超200亿元。

3. 赛道热度观察

AI仍是最大主线，但资金更偏“基础设施化”

AlleyWatch数据显示，美国2026年4月VC总额208亿美元 / 442笔，同比资本额增长63.9%；AI公司获得151.8亿美元 / 189笔，占全部资本73.0%。但剔除Project Prometheus的100亿美元极端值后，美国市场仍有108亿美元 / 441笔，说明不是单点繁荣，而是广泛活跃。

非AI大额资金流向硬科技

4月美国Top 10交易中，国防航天、核能、半导体、先进制造、电动车/清洁交通显著靠前。资金正在从“软件AI”扩展到“AI驱动或AI受益的实体基础设施”。

中国市场：政策确定性 + 退出预期驱动硬科技集中

新浪财经/深圳市投资基金同业公会材料显示，中国一季度股权投资新募基金数量同比增长114.0%、金额同比增长79.8%；资金高度集中于人工智能、具身智能、半导体、商业航天、低空经济、创新药、新能源链。

Biotech出现结构性修复

Xtalks/Labiotech跟踪显示，2026年生物科技融资集中在神经精神、免疫、呼吸、AI制药、细胞治疗/基因药物。典型交易包括Isomorphic Labs 21亿美元B轮、CellCentric 2.2亿美元D轮、Windward Bio 1.65亿美元Crossover、CREATE 1.22亿美元B轮。市场对“多资产管线 + 临床读出明确 + AI提效”的公司更友好。

AI制药正在从“模型故事”进入lab-in-the-loop和候选药物生成阶段。

  • 维他动力｜近5亿元人民币Pre-A｜消费级具身智能

具身智能从工业场景向消费端外溢，估值和融资规模值得跟踪是否过热。

  • Serif Biomedicines｜5000万美元启动资金｜Modified DNA遗传药物平台

Flagship Pioneering孵化，平台技术存在高不确定性但具备高上限。

5. 分析师点评

1. VC市场不是简单复苏，而是“集中式再定价”：资金总量回暖，但集中押注少数能解释超大市场、政策确定性或基础设施瓶颈的赛道。普通软件和传统消费仍难获得同等估值溢价。

2. AI投资进入二阶阶段：第一阶段投模型与工具，当前阶段投“AI进入物理世界和企业流程的基础设施”，包括AI数据系统、企业AI交付、机器人、国防航天、AI制药。

3. 硬科技成为AI外溢资金的主要承接者：核能、电网、半导体、稀土磁体、航天、低空经济等交易说明资本正在寻找AI算力需求、地缘安全与产业政策交汇处。

4. 早期项目仍有窗口，但门槛上移：Seed/A轮项目需要证明技术壁垒、真实场景、商业闭环或临床/监管路径，不再只是讲“AI概念”。

5. 后续值得追踪方向：Agentic AI垂直落地、具身智能量产订单、AI基础设施利润率、AI制药临床读出、国防科技采购转化、新能源/核能与数据中心电力需求联动。
