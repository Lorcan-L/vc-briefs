3. 赛道热度观察

AI仍是主轴，但资金正在硬化

  • AlleyWatch统计：2026年4月美国VC融资为$20.80B、442笔，同比增长63.9%；AI公司获得$15.18B，占全部资本73.0%。
  • Project Prometheus单笔$10B占全美当月资本48.1%，这说明headline数据仍被巨轮次强烈扭曲；剔除该交易，美国市场仍有$10.80B、441笔交易，底层活跃度并不弱。
  • AI资金不只进入模型公司，也流向AI基础设施、航天安全、公共安全、实时生成视频和企业生产交付。

国防航天、能源、半导体成为AI之外最强硬科技支线

  • True Anomaly、Vulcan Elements、Hermeus等显示国防/航天/先进制造获得机构资金持续加仓。
  • Valar Atomics的$340M Series B把核能/工业级电力推到中期融资焦点；电力正在变成AI时代基础设施约束。
  • SiFive $400M说明半导体IP和RISC-V仍能承接大额成长轮资金。

企业AI从“试点”转向“交付与治理”

  • Unframe新增$50M且披露12个月TCV超过$100M、净收入留存400%，是本期最值得注意的商业化信号：企业AI正在从demo转向可交付、可集成、可治理。
  • Agentic AI数据库显示，资金高度集中在法律、客服、软件工程、招聘、医疗、金融分析等明确工作流，前10家公司约吸收该类样本60%资金。

生物医药回暖，但钱偏向“可结构化数据 + 临床转化”

  • Labiotech等tracker显示2026生物科技融资持续被追踪，资金关注点从泛平台转向有清晰R&D路径、临床数据或专科流程价值的公司。
  • Triomics的肿瘤AI Series B代表医疗AI更受欢迎的方向：不是替代医生，而是把纵向病历、试验匹配、chart prep和数据抽象变成可运营基础设施。

中国市场：具身智能、半导体、商业航天和政策确定性继续吸金

  • 新浪财经转引深圳市投资基金同业公会：2026年一季度中国新募基金数量和金额同比增长114.0%和79.8%，但资金高度集中在先进制造、AI、半导体。
  • 2026年4月仅8家具身智能企业融资超200亿元，说明中国一级市场正在把“机器人量产与订单验证”作为AI应用落地的核心抓手。

4. 值得关注的早期项目

  • Reactor：$59M，Lightspeed领投，实时生成视频开发平台。亮点是把world model能力包装为开发者平台，若低延迟和可控性成立，可能成为视频生成从内容工具走向交互系统的基础层。
  • Itera：$12M Seed，fluid circuit board。亮点是缩短硬件工程师真实元器件测试和迭代周期，属于AI/硬件开发链条里的效率基础设施。
  • Didit：总seed $7.5M，AI-native身份与反欺诈基础设施。KYC/KYB、交易监控、wallet screening合一，适合金融科技、Web3和跨境产品场景。
  • Triomics：$22M Series B，肿瘤AI。虽已到B轮，但仍属于医疗AI工作流中值得跟踪的早中期样本，价值在于结构化复杂癌症病历并服务试验匹配。
  • 维他动力：近5亿元Pre-A，机器狗/消费级机器人。中国消费级机器人若能走出玩具化，可能成为具身智能商业化的前哨。

5. 分析师点评

本期VC市场的关键不是“AI还有没有泡沫”，而是资金在用更高的门槛筛选AI：

1. 巨额轮次继续制造总量幻觉。 Project Prometheus一笔交易占美国4月融资近一半，说明看VC大盘必须剔除极端值，否则容易误判市场普遍繁荣。

2. 机构偏好从纯软件转向“软件+硬件+基础设施”。 国防航天、核能电力、半导体、先进制造都在同一张投资逻辑里：AI越强，现实世界的能源、算力、制造和安全约束越值钱。

3. 企业AI进入交付考核期。 Unframe这类公司比“通用Agent故事”更值得追踪，因为它们披露的是TCV、NRR、部署方式、治理与集成能力，这些指标更接近真实购买意愿。

4. 早期机会仍在，但要贴近瓶颈。 实时视频、身份风控、医疗数据结构化、硬件迭代工具、机器人量产链条，都是“AI落地过程中的瓶颈层”。这类公司未必最性感，但更可能从预算中分到真实现金流。

后续建议重点跟踪：

  • Project Prometheus及类似physical AI巨轮次的真实产品定义和客户；
  • 企业AI交付平台的续约率、部署周期和毛利；
  • 核能/电力/数据中心互连是否持续进入Series B-C主流轮次；
  • 中国具身智能融资后的量产交付、订单质量和核心零部件国产化进度。
