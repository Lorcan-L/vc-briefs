一、核心结论

1. 全球VC金额被AI超级交易显著抬高

Crunchbase 口径显示，2026年Q1全球VC约3000亿美元，AI相关融资约2420亿美元、占比约80%；KPMG口径显示Q1全球VC约3309亿美元、8464笔交易。差异来自统计范围和交易口径，但两者共同指向：AI mega-deals 是本轮数据高位的核心驱动。

2. 资金集中度极高，少数头部公司决定市场表观热度

OpenAI、Anthropic、xAI、Waymo 等少数项目合计融资规模已达到千亿美元级。若剔除超级轮次，常规VC市场并不能简单判断为“全面复苏”。

3. AI正在从模型层向实体经济和硬科技扩散

4月美国大额融资中，Project Prometheus、True Anomaly、VAST Data、SiFive、Valar Atomics、Vulcan Elements 等项目显示，资本正在加码 AI基础设施、国防科技、太空安全、核能、关键材料、半导体与自主系统。

4. 中国市场热点更偏“政策牵引 + 工程落地”

具身智能/机器人、半导体、商业航天/低空经济、核聚变/未来能源、新能源链、高端装备与新材料成为资金关注方向。4月“8家具身智能企业超200亿元融资”的行业媒体统计显示局部赛道热度明显，但需注意该类中文来源多为行业媒体/二次统计口径。

三、赛道热度观察

1. AI基础模型与AI平台：最热，但集中度最高

AI相关融资占全球VC融资比例达到极高水平，但主要由OpenAI、Anthropic、xAI等超级轮次拉动。对普通AI应用创业公司而言，融资环境并不等同于“全面宽松”。资本更偏好具备平台地位、算力资源、企业客户、模型能力或生态入口的公司。

趋势判断：AI仍是主线，但“泛AI应用”估值红利正在让位于“基础设施 + 明确商业化 + 产业场景”。

2. AI Infra / 数据中心 / 半导体：确定性最高

Databricks、VAST Data、SiFive 等融资显示，AI带来的算力、数据、存储、网络、芯片、部署工具链需求仍在持续扩张。AI不只是软件周期，也在推动一轮新的资本开支周期。

资金规模趋势：大额成长轮与后期轮持续出现，且投资方质量高；这类公司通常拥有更清晰的企业预算来源。

3. Physical AI / 具身智能 / 机器人：中美共同热点

Project Prometheus 的100亿美元融资，以及中国4月具身智能企业集中融资，说明资本正在押注AI进入实体经济、制造、家庭、物流、自动化和工业场景。

中国市场特点：更强调量产、供应链、场景验证与政策支持。人形机器人之外，核心零部件、运动控制、传感器、工业/仓储/巡检机器人更值得跟踪。

4. Defense Tech / Spacetech / Autonomous Systems：美国大额轮次突出

Shield AI、True Anomaly、Hermeus、Scout AI、Reliable Robotics 等案例显示，国防科技、太空安全、无人系统和自主飞行正在成为美国VC主线之一。地缘政治、政府采购与dual-use technology共同推动了该赛道升温。

趋势判断：这类项目具备高壁垒和订单确定性，但也面临监管、交付周期和政府预算波动风险。

5. 能源与关键材料：AI时代的底层约束被重新定价

Valar Atomics、东昇聚变、Vulcan Elements 等融资说明，能源、核能、核聚变、稀土磁材和关键材料正在从传统硬科技叙事变成AI基础设施的关键变量。AI数据中心耗电、算力扩张和供应链安全会继续推高该方向关注度。

6. 中国硬科技：具身智能、半导体、低空经济、核聚变、创新药

中文来源显示，2026年中国股权融资资金继续向先进制造、AI、半导体、具身智能、商业航天/低空经济、生物医药、新能源链、高端装备与新材料集中。政策确定性和退出渠道改善预期，是国内资金敢于进入这些领域的重要原因。

五、分析师点评

共识

  • 2026年VC市场不是均匀复苏，而是极端结构化复苏：钱主要流向AI、AI基础设施、硬科技、国家战略相关赛道。
  • AI仍是全球资本最强主线，但真正获得大额资金的不是所有AI公司，而是少数平台型公司、基础设施公司，以及能够把AI转化为物理世界生产率的公司。
  • 中美市场逻辑分化明显：美国偏“AI + 国防 + 太空 + 能源安全”，中国偏“AI + 制造 + 国产替代 + 新质生产力”。

分歧与风险

  • 数据口径风险：Crunchbase与KPMG统计口径不同，不能直接横向比较金额和笔数。
  • mega-deal扭曲风险：OpenAI、Anthropic、xAI、Waymo等超级交易显著抬高总体金额，不能据此判断普通创业公司融资普遍改善。
  • 传统VC定义风险：部分超级融资可能包含战略投资、结构化融资或非典型股权安排，未必完全等同传统VC。
  • 中文来源可靠性：部分中国赛道数据来自行业媒体/协会或二次转载，适合作为趋势信号，不宜作为精确统计结论。

Leader综合判断

当前VC市场最重要的变化是：AI投资正在从“模型公司融资竞赛”升级为“AI时代基础设施与实体产业重估”。

未来值得持续追踪的方向：

1. AI基础设施全栈：算力、数据中心、电力、散热、网络、存储、推理部署、企业AI工程化工具。

2. 具身智能与机器人：机器人本体、运动控制、传感器、视觉-语言-动作模型、工业/仓储/巡检/家庭服务场景。

3. 自主系统与无人平台：自动驾驶、无人机、无人系统、低空经济、边缘AI。

4. Defense Tech 与 Spacetech：太空态势感知、卫星、无人系统、反无人机、关键基础设施安全。

5. 未来能源与电力系统：核能、核聚变、储能、电网调度、数据中心能源配套。

6. 半导体与关键材料：AI芯片、先进封装、光互连、稀土磁材、高性能材料、国产替代关键环节。

投资启示：下一阶段不应只问“是不是AI公司”，而应问：这家公司是否站在AI资本开支链条上？是否解决实体产业中的高价值瓶颈？是否具备技术里程碑、订单验证、产业链位置和后续资本承接能力？
