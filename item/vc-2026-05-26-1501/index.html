VC投资动态调研｜2026-05-26 15:01

1. 本期核心判断

资本正在继续押注“AI + 物理世界基础设施”：一边是AI/Agentic AI从模型、算力、企业交付向应用闭环渗透，另一边是防务航天、核能、先进制造、关键金属、生物医药等高资本开支硬科技赛道获得大额资金。

2. 本期重点交易

  • Project Prometheus｜Late-stage｜100亿美元｜领投方未披露｜基础AI/Physical AI

AlleyWatch 报道其单笔融资占美国4月VC总额48.1%，是本期最强信号：资本愿意为“AI进入物理经济”支付超大额期权费。

  • Slate Auto｜Late-stage｜6.5亿美元｜未披露｜汽车/EV/制造

体现美国市场对硬件制造和交通电动化的持续配置。

  • True Anomaly｜Late-stage｜6亿美元｜未披露｜航天/AI/国家安全

AI与国防航天交叉成为非纯软件AI之外最受关注方向之一。

  • VAST Data｜Late-stage｜5亿美元｜未披露｜AI基础设施/数据平台

AI基础设施仍是资本主线，算力之后数据操作系统、存储和企业级AI平台成为承接资金的关键层。

  • Vulcan Elements｜Late-stage｜4.31亿美元｜未披露｜稀土磁体/防务/工业制造

关键材料与供应链自主化正在变成VC可投的大额硬科技主题。

  • Valar Atomics｜Series B｜3.4亿美元｜未披露｜核能/电网/工业能源

本期Series B最大亮点，反映AI与先进制造带来的电力缺口正在反向推动新能源/核能融资。

  • Unframe｜追加融资｜5000万美元｜Highland Europe领投，Bessemer、Craft等参投｜企业AI交付平台

公司披露12个月TCV超过1亿美元、净收入留存400%，显示“AI从试点到生产”的企业交付平台开始被VC重新定价。

  • Boston Metal｜未披露轮次｜7500万美元｜Tata Steel加入，既有投资方参投｜关键金属/低碳金属生产

先进制造+关键材料与产业资本协同，属于气候科技里更接近产业落地的一类。

  • Windward Bio｜Crossover｜1.65亿美元｜OrbiMed、Novo Holdings、Qiming、RA Capital等｜免疫/呼吸疾病抗体药物

生物医药资金集中支持具备临床读出和后续IPO/并购可能的多资产管线。

  • Tribal｜Seed｜1000万美元｜Team8领投，DYDX Capital等参投｜企业原生AI平台

值得关注的早期Agentic/Enterprise AI项目，团队来自Salesforce、Wix、Spot.io生态。

3. 赛道热度观察

AI仍是绝对主线，但资金集中度极高

Crunchbase显示，2026年Q1全球VC融资约3000亿美元，其中AI拿走约2420亿美元，占比约80%；OpenAI、Anthropic、xAI、Waymo四家公司合计融资1880亿美元，占全球季度VC约65%。AlleyWatch的4月美国数据也显示AI获得151.8亿美元，占美国当月VC资本73%。这说明市场不是“全面回暖”，而是“AI超级头部 + 若干硬科技主题”的结构性繁荣。

Late-stage和超大轮次抬高市场总量，早期仍要看真实商业化

美国4月442笔交易融资208亿美元，但Project Prometheus一笔就贡献100亿美元。剔除该项目后，美国4月仍有108亿美元/441笔，说明底层交易活跃度不差，但估值和资本分配明显向头部倾斜。Series A更能反映真实底盘：98笔、平均2000万美元，没有单一交易支配，说明投资人仍愿意投早中期，但前提是技术壁垒与商业路径明确。

中国市场：具身智能、半导体、商业航天、低空经济最强

新浪财经/深圳市投资基金同业公会文章指出，2026年一季度中国股权投资市场新募基金数量和金额分别同比增长114.0%和79.8%；4月仅8家具身智能企业融资超200亿元。国内资金偏好高度政策化与产业化：AI全链条、机器人、半导体、商业航天/低空经济、生物医药、新能源链条是主线。

生物医药资金回到“平台+临床读出”组合

Xtalks梳理的2026年生物医药融资中，Windward Bio 1.65亿美元、Coultreon 1.25亿美元Series A、Fathom 4700万美元Series A、Serif 5000万美元启动融资等案例显示，神经、免疫、呼吸、AI药物发现和基因药物平台仍有资金；但资本更偏向能进入或推进Phase I/II、拥有明确临床节点的资产。

4. 值得关注的早期项目

  • Tribal：1000万美元种子轮，Team8领投。企业原生AI平台，切入复杂企业系统，团队背景贴近Enterprise SaaS生态；适合跟踪其是否能从“企业AI平台”泛叙事中跑出高留存用例。
  • Fathom Therapeutics：4700万美元Series A，Sutter Hill Ventures等参投。物理+AI小分子设计平台，属于AI Drug Discovery中更强调底层模拟与lab-in-the-loop的路线。
  • Serif Biomedicines：Flagship Pioneering 5000万美元启动。Modified DNA平台，偏前沿基因药物，技术风险高但平台想象空间大。
  • Signadori Bio：约1290万美元种子扩展轮。现货型实体瘤免疫疗法，金额小但方向具备差异化。

5. 分析师点评

1. 本轮VC不是普涨，而是“AI超级资本开支周期”带来的结构性高景气。 资金集中在少数超级公司和能够支撑AI落地的算力、数据、电力、制造、国防、材料环节。

2. Agentic AI从概念进入企业交付验证期。 Unframe披露的1亿美元TCV和400% NRR，是比“融资额”更重要的信号：企业愿意为能落地、能集成治理和工作流的AI平台付费。

3. 硬科技的投资逻辑正在从主题叙事转向供应链安全与能源约束。 核能、关键金属、稀土磁体、航天防务等融资，背后是AI与地缘政治共同驱动的基础设施再定价。

4. 中国市场最强线索是具身智能和国产替代。 但估值快速上行后，应重点区分“有量产和订单”的公司与“只靠Demo和政策叙事”的公司。

5. 后续建议持续追踪三条线： 企业AI平台的ARR/NRR数据、AI基础设施和能源约束下的硬科技融资、具身智能从融资热度到真实交付的转化率。
