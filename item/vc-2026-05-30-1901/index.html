VC投资动态调研｜2026-05-30 19:01

1. 本期核心判断

资本正在从“泛AI叙事”转向“AI可交付系统 + 物理世界基础设施”：巨额资金仍由 frontier/physical AI 拉动，但更有含金量的增量信号集中在企业AI交付、实时生成视频、医疗工作流、身份风控、AI基础设施、国防航天、核能/半导体与中国具身智能。

2. 本期重点交易

  • Project Prometheus / Late-stage / $10.0B / 未披露 / Foundational AI、Physical-world AI

AlleyWatch 披露其单笔交易占美国4月VC总额48.1%，是本期总量被“巨轮次”扭曲的核心原因。

  • Slate Auto / Late-stage / $650M / 未披露 / EV、制造

非纯软件资金继续回流硬件与制造，说明资本在押注AI之外的实体供给链重构。

  • True Anomaly / Late-stage / $600M / 未披露 / Aerospace、AI、National Security

国防航天与AI结合继续吸收大额资本。

  • VAST Data / Late-stage / $500M / 未披露 / AI Infrastructure

AI算力与数据基础设施仍是机构票面最重的方向之一。

  • Vulcan Elements / Late-stage / $431M / 未披露 / 稀土磁体、国防工业制造

关键材料和先进制造成为非AI硬科技主线。

  • SiFive / Late-stage / $400M / 未披露 / RISC-V、半导体IP

半导体底层IP继续获得战略性资本支持。

  • Valar Atomics / Series B / $340M / 未披露 / Nuclear、Energy Infrastructure

单笔交易显著拉高Series B均值，能源基础设施进入更高资本强度阶段。

  • Reactor / 未披露 / $59M / 未披露 / 实时生成视频开发平台

从隐身状态推出，代表生成式视频进入开发者平台与生产系统阶段。

  • Unframe / Additional funding / $50M / Highland Europe / Enterprise AI Delivery

“帮大企业把AI真正交付出来”的中间层平台继续受追捧。

  • Triomics / Series B / $22M / 未披露 / Oncology AI、Clinical workflow

医疗AI从诊断概念转向复杂临床信息运营化。

  • Itera / Seed / $12M / 未披露 / Fluid circuit board、Deeptech hardware

早期深科技硬件仍有窗口，但要求原型和工程突破明确。

  • Didit / Seed追加 / $7.5M total seed / 未披露 / AI-native identity & fraud

身份、风控、反欺诈成为AI应用落地的底层刚需。

3. 赛道热度观察

AI：总量强，但更需要拆分看

AlleyWatch数据显示，美国4月VC为 $20.80B / 442笔，其中AI公司获得 $15.18B / 189笔，占资本总额 73.0%。但其中Project Prometheus一笔占总额48.1%，所以“AI很热”背后其实是：头部模型/physical AI吸走超大票面，中腰部机会转向可落地工作流和基础设施。

Early / Series A：更能代表真实市场温度

美国4月Series A为 $1.96B / 98笔，平均约 $20M，没有单笔交易过度主导；这比Late-stage更能说明底层投资人仍在持续下注。Series B则被Valar Atomics的$340M明显拉高。

企业AI：从Copilot走向“交付层”

Unframe、Triomics、Didit、Reactor共同指向一个变化：资本不只买“更聪明模型”，更买“能被企业采购、集成、审计和运营的系统”。agentic AI榜单中，Harvey、Sierra、Parloa、Decagon等法律/客服/企业流程公司累计融资领先，说明垂直场景仍是资金密集区。

硬科技：国防、能源、半导体、先进制造重新抬头

True Anomaly、Vulcan Elements、Hermeus、SiFive、Valar Atomics显示，VC在AI之外继续追逐“战略基础设施”。这类交易资本强度高、周期长，但有政策/地缘/供应链确定性支撑。

中国市场：具身智能、半导体、商业航天为最强叙事

新浪财经/深圳市投资基金同业公会口径显示，2026年一季度中国股权融资复苏，新募基金数量和金额分别同比增长 114.0% / 79.8%；4月8家具身智能企业融资超 200亿元，先进制造、AI、半导体、商业航天、新能源、生物医药为核心方向。

4. 值得关注的早期项目

  • Itera：$12M seed，流体电路板原型。亮点是硬件形态创新，适合追踪其工程验证、量产可行性和应用场景。
  • Didit：$7.5M total seed，AI-native identity/fraud。AI应用爆发后，身份校验、反欺诈和合规会成为横向基础设施。
  • Reactor：$59M，从stealth推出实时生成视频开发平台。若其能成为“视频生成的开发者基础设施”，价值会高于单点视频工具。
  • Signos：$20M，FDA-cleared OTC glucose monitoring for weight management。消费医疗/代谢健康仍有持续需求，但需关注获客成本和保险/监管路径。
  • 中国具身智能链条早期供应商：传感器、执行器、端侧计算、工业/家庭场景集成商，可能比整机公司更早出现可验证营收。

5. 分析师点评

1. 不要被总额误导：本期最大信号不是“VC全面复苏”，而是“少数巨轮次 + 中腰部稳定活跃”。去掉Project Prometheus，美国4月仍有$10.80B/441笔，说明市场不是空心化，但估值和资金仍高度向头部集中。

2. AI投资进入第二阶段：第一阶段买模型和算力，第二阶段买交付、工作流、行业数据、身份安全、开发者基础设施。更值得追踪的是“能形成预算科目”的AI公司，而不是demo型应用。

3. 硬科技的资本逻辑增强：国防、能源、半导体、先进制造受到政策、地缘和AI算力需求共同驱动。它们不是AI的替代方向，而是AI基础设施外延。

4. 中国市场的主线更政策化：具身智能、半导体、商业航天、低空经济、创新药、新能源链条都与退出渠道和政策确定性绑定。投资判断要同时看技术壁垒、产品化证据、订单和IPO/并购可退出性。

5. 下一步追踪方向：企业AI交付平台、医疗临床工作流AI、身份风控、实时生成视频基础设施、RISC-V/先进封装、核能/电网、具身智能零部件与场景运营商。
