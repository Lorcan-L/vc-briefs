摘要

本期检索覆盖英文与中文 VC/风投融资动态，重点参考 KPMG Venture Pulse、Crunchbase、AlleyWatch、中国证券报等来源。不同数据源口径存在差异，但结论高度一致：2026年Q1至4月的VC市场并非全面复苏，而是由AI megadeals驱动的结构性集中行情。

  • KPMG口径：Q1'26全球VC投资 3309亿美元 / 8464笔交易，较Q4'25的1286亿美元显著增长。
  • Crunchbase口径：Q1'26全球VC投资约 3000亿美元 / 6000家创业公司，环比和同比均增长约150%。
  • AI为绝对主线：Crunchbase口径下，AI公司获投 2420亿美元，占全球VC总额约80%。
  • 资金高度集中：OpenAI、Anthropic、xAI、Waymo四笔合计约 1880亿美元，约占全球Q1融资的65%。
  • 美国继续主导：美国获投约 2500亿-2672亿美元，占全球约80%以上。
  • 中国一级市场热点集中在 具身智能、机器人、AI硬件、低空经济、商业航天、聚变、半导体、生物科技 等硬科技方向。

> 分析师判断：当前VC市场更像“AI超级资产驱动的结构性集中”，而不是全市场周期性复苏。headline funding很热，但剔除少数megadeal后，中位数创业公司的融资环境可能仍然偏冷。

2. 赛道热度观察

2.1 AI基础模型与AI平台：资金最集中，但集中度风险最高

AI仍是全球VC最强主线。Crunchbase数据显示，Q1'26 AI公司融资约2420亿美元，占全球VC总额约80%。但这并不代表AI创业公司普遍融资顺利，而是资金极度集中于少数基础模型、算力、自动驾驶和平台型公司。

关键观察：

  • OpenAI单笔1220亿美元融资本身就足以显著改变全球季度总额。
  • Anthropic、xAI、Waymo、Databricks等继续证明，资本愿意为“AI底层平台级资产”支付极高溢价。
  • 这一阶段的AI融资更像基础设施建设，而非传统软件创业融资。

风险提示：

  • 把“AI占比80%”解读为“AI创业生态全面繁荣”是误读。
  • 更准确的判断是：资本正在向少数AI超级资产集中。

2.2 AI Infrastructure：从模型竞争延伸到算力、数据中心和网络

AI模型扩张带动基础设施需求，资金正在流向数据平台、数据中心网络、推理成本优化、能效管理等“卖铲子”方向。

代表交易：

  • Databricks：70亿美元
  • Aria Networks：1.25亿美元A轮，AI data center networking

值得关注的细分方向：

  • 推理成本优化
  • 数据中心能耗与冷却管理
  • AI workload orchestration
  • GPU利用率提升
  • 数据中心网络和互联
  • 边缘推理基础设施

分析判断：

AI Infra是本轮AI周期中确定性较强的方向，但估值已经被明显抬高。未来需要重点看客户粘性、成本改善幅度和真实收入质量。

2.3 Agentic AI：应用层融资加速，进入机构重仓区

4月美国大额融资中，多个1亿美元以上项目集中在AI agent方向，说明应用层AI正在从“demo阶段”进入“workflow嵌入阶段”。

代表交易：

  • Parallel：1亿美元B轮，open web agents / APIs
  • Avoca：1.25亿美元B轮，AI agents for calls/leads
  • Factory：1.5亿美元C轮，software development agents
  • Hightouch：1.5亿美元D轮，agentic marketing
  • Floatboat：中国天使轮，探索Agent接管工作上下文与流程

值得追踪的Agent项目类型：

  • 面向具体岗位的AI worker
  • 法务、财务、销售、客服、供应链agent
  • AI coding / DevOps agent
  • 工业运维agent
  • 医疗行政、保险理赔agent

判断标准：

  • 是否接入真实业务系统
  • 是否替代明确人力成本
  • 是否形成数据闭环
  • 是否具备高频持续使用
  • 是否形成切换成本

2.4 Physical AI / Robotics / 具身智能：中美同时升温

AI正在从纯软件走向物理世界。美国侧体现为physical economy AI、defense robotics、drones；中国侧则集中在具身智能、机器人整机、核心零部件、量产能力和订单验证。

代表交易：

  • Project Prometheus：100亿美元，physical economy AI
  • Scout AI：1亿美元A轮，defense robotics
  • Skydio：1.1亿美元F轮，无人机和自动化
  • 自变量机器人：10亿元A++轮
  • 千寻智能：近20亿元A轮
  • 地瓜机器人：1.5亿美元B2轮
  • 灵猴机器人：数亿元B轮

中国市场特征：

  • 红杉中国、高瓴创投等头部机构不只看技术概念，而是更重视“量产、订单、交付能力”。
  • 具身智能的投资逻辑正在从“模型能力”转向“供应链能力 + 场景ROI + 客户订单”。

值得关注的细分方向：

  • 关节模组、灵巧手、执行器、力/触觉传感器
  • 工业巡检、仓储拣选、柔性制造机器人
  • 低成本整机平台
  • 封闭场景下ROI明确的机器人系统方案

2.5 Defense AI / Autonomous Systems：地缘政治推动资金进入

国防机器人、自主飞行器、边缘AI和无人系统成为美国VC热点。Shield AI、Scout AI、Skydio等交易显示，Defense Tech正在从小众赛道变成AI周期中的重要配置方向。

代表交易：

  • Shield AI：23亿美元
  • Scout AI：1亿美元A轮
  • Skydio：1.1亿美元F轮

分析判断：

该方向壁垒高、订单价值大，但政策敏感、采购周期长。适合深度研究，不适合泛化投资。

2.6 硬科技：商业航天、聚变、半导体、低空经济继续吸引头部机构

中国头部机构在Q1继续布局长周期硬科技资产。红杉、高瓴、IDG等参与东昇聚变天使轮，高瓴联合领投宇石空间，红杉和高瓴追投汇天飞行汽车。

代表方向：

  • 商业航天：宇石空间、Starfish Space
  • 聚变能源：东昇聚变
  • 低空经济：汇天飞行汽车
  • 半导体和先进制造：AI算力、机器人、国防、数据中心共同拉动

分析判断：

硬科技正在成为AI周期的外溢方向，但其投资回报周期通常长于传统VC。对普通财务投资人而言，更值得关注的是核心部件、材料、控制系统、仿真平台等“周边卖铲子”机会。

2.7 FinTech / Healthcare：独立热度下降，AI结合仍有机会

单纯FinTech和Healthcare并非本期主线，但与AI、老龄化、供应链、保险等场景结合后，仍能获得大额融资。

代表交易：

  • Slash：1亿美元C轮，FinTech
  • Chapter：1亿美元E轮，Medicare / Health Insurance
  • Havra：1.13亿美元Seed，Supply Chain AI / FinTech / Logistics
  • OdyssLife：近2亿元人民币，AI健康硬件

4. 分析师点评

4.1 VC市场趋势判断

本期最重要的结论是：VC市场总额大幅增长，但不是普遍复苏，而是集中度进一步提升。

从数据看，Q1'26的融资额达到历史级别，但交易数量并未同步扩张。KPMG数据显示，全球VC金额从Q4'25的1286亿美元升至Q1'26的3309亿美元，但交易数从10097笔下降至8464笔。这意味着更多钱流向更少项目。

Crunchbase阶段数据也印证这一点：

  • Late-stage：2466亿美元 / 584笔
  • Early-stage：413亿美元 / 1800笔
  • Seed：120亿美元，金额增长31%，但交易数下降30%

这说明早期市场也在“大额化、集中化、头部化”。Seed金额增加并不代表早期创业环境全面变好，而是少数高信号项目拿到更大的支票。

4.2 AI周期进入第二阶段：从模型走向物理世界和工作流

2023-2025年的AI投资主线是基础模型和AI应用验证。2026年的明显变化是，资金开始向三个方向外溢：

1. AI基础设施：算力、数据中心、网络、能效、推理成本优化。

2. Agentic workflow：开发、销售、客服、营销、企业后台等高频流程。

3. Physical AI：机器人、无人机、自动驾驶、制造、国防、低空经济。

这意味着投资人不再只押注“谁有更强模型”，而是押注“谁能把AI转化为可部署的生产力”。

4.3 中国一级市场更强调“技术落地”

中国市场的关键词不是单纯的大模型，而是：

  • 具身智能
  • AI硬件
  • 机器人量产
  • 低空经济
  • 商业航天
  • 聚变
  • 半导体和先进制造

头部机构的筛选标准明显转向：

  • 是否已有订单
  • 是否具备量产能力
  • 是否能进入真实产业链
  • 是否有明确降本增效价值
  • 是否具备政策和供应链红利

4.4 当前市场的主要风险

1. Megadeal扭曲风险

- 少数巨额交易抬高整体数据，掩盖了普通创业公司融资仍难的现实。

2. AI估值透支风险

- AI agent、data center、defense AI多个方向已出现A轮/Seed轮过亿美元融资，估值明显前置。

3. 硬科技商业化周期风险

- 机器人、低空经济、商业航天、聚变等方向周期长、资本开支大、退出路径不确定。

4. 数据口径风险

- KPMG与Crunchbase统计口径不同，不能直接混合计算，只能作为趋势交叉验证。

5. 早期项目分化风险

- Seed金额上升但交易数下降，说明资本更挑剔，非AI/非硬科技项目融资环境可能更冷。

5. 值得持续追踪的方向

优先级S：结构性主线

1. AI Compute / Data Center / Power Infrastructure

- 推理成本优化

- 数据中心能效管理

- 液冷和新型散热

- AI workload调度

- GPU利用率提升

2. Agentic AI + Enterprise Workflow

- 垂直行业agent

- AI coding / software factory

- 企业后台自动化

- 销售、客服、财务、法务agent

3. Physical AI / Robotics / Manufacturing Automation

- 核心零部件

- 工业场景机器人

- 仓储、巡检、柔性制造

- 封闭场景下ROI明确的机器人应用

4. Defense / Autonomous Systems

- 无人机和反无人机

- 自主感知与控制

- 边缘AI

- 军民两用通信、导航、仿真系统

优先级A：高潜力但需精选

1. AI Hardware / Edge AI Devices

- 重点看垂直场景硬件，而非泛AI gadget。

2. Biotech + AI

- 适合专业基金，验证周期长，不宜只看AI概念。

3. Semiconductor / Advanced Manufacturing

- AI推理芯片、先进封装、国产替代关键设备和材料。

4. Commercial Space

- 更看好部件、数据服务和供应链，而非纯宏大叙事。

优先级B：热度高但拥挤

1. 低空经济 / eVTOL / Drone Logistics

- 需要监管试点、明确场景和订单支撑。

2. Fusion / New Energy Frontier

- 更适合战略资本或超长期基金；普通VC可关注材料、部件、仿真和控制系统等周边机会。

---

6. 结论

本期VC市场呈现三个核心特征：

1. 总量热，结构冷

- 总额创纪录，但主要由少数AI megadeals驱动。

2. AI从模型层外溢到基础设施、工作流和物理世界

- AI Infra、Agentic AI、Physical AI成为下一阶段融资主线。

3. 中国市场进入硬科技落地验证期

- 红杉、高瓴、IDG等头部机构押注具身智能、AI硬件、低空经济、商业航天、聚变等方向，但更强调量产、订单和产业链位置。

一句话判断：

> 2026年的VC市场不是“钱又回来了”，而是“钱更集中地流向少数能承接AI超级周期的资产”。下一阶段最值得追踪的，不是泛AI概念，而是能把AI转化为基础设施能力、企业流程效率和实体产业生产力的公司。
