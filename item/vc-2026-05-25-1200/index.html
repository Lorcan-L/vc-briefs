2. 赛道热度观察

2.1 AI仍是资金黑洞，但资金正在从“纯软件”外溢到物理世界

Crunchbase 数据显示，2026年Q1全球VC融资约 $300B，约 6,000 家公司获投；其中AI相关公司获得约 $242B，占全球VC融资约80%。这不是传统SaaS周期的线性升级，而是“模型 + 算力 + 数据中心 + 机器人 + 自动驾驶 + 国防/制造”的综合资本开支周期。

最典型信号是 Project Prometheus 单轮 $10B、VAST Data $500M、SiFive $400M、Aria Networks $125M 等交易共同指向：AI投资主线已从模型层扩散到算力、存储、网络、半导体、数据中心自动化，以及面向真实世界任务的physical AI。

2.2 Defense Tech / Aerospace / National Security 成为美国大额支票集中区

4月美国大额融资榜中，True Anomaly、Hermeus、Scout AI、Skydio、Starfish Space、Vulcan Elements等项目密集出现。资金流向不再局限于软件效率，而是转向“国家安全 + 供应链韧性 + 自主系统”。这类公司往往具备：政府/军方需求确定性、硬件壁垒、长期合同潜力，但资本消耗也显著更高。

2.3 能源成为AI周期的二阶主题：核能、聚变、工业脱碳抬头

Valar Atomics $340M Series B、东昇聚变数亿元天使轮、Boston Metal $75M 均说明能源和工业底层技术正在重新获得VC关注。背后逻辑很直接：AI算力需求推高电力与材料约束，资本开始提前布局能源供给、关键金属和低碳工业生产。

中国侧尤其明显：东昇聚变天使轮集合红杉、IDG、高瓴、中科创星等一线机构，说明“未来能源 + 国家战略 + 顶尖科研团队”的组合已经能在极早期拿到超规格融资。

2.4 中国一级市场：先进制造、AI、半导体、具身智能高度集中

Sina/深圳市投资基金同业公会文章指出，2026年Q1中国股权投资市场新募基金数量和金额分别同比增长114.0%和79.8%，但资金高度集中在先进制造、AI、半导体。4月仅8家具身智能企业融资超200亿元人民币，市场呈现强烈“头部项目虹吸”。

中国市场的关键词不是“全面复苏”，而是“结构性复苏”：符合政策导向、技术壁垒清晰、商业化路径可见的项目融资顺畅；传统行业或纯概念项目仍处在融资冰河。

2.5 Biotech融资有修复迹象，但更偏向差异化平台与临床可推进资产

EpiBiologics $107M Series B 由GV和JJDC共同领投，投向组织选择性细胞外蛋白降解平台，核心项目计划推进NSCLC和头颈部鳞癌首次人体临床。Biotech资金仍谨慎，但愿意为“平台技术 + 明确适应症 + 近临床节点”买单。

4. 分析师点评

4.1 当前VC市场不是普涨，是“超级集中化”

本期最重要的判断：VC市场表面上数据极强，但质量结构高度分化。Crunchbase口径下Q1全球VC $300B、AI占80%，但大量资金被OpenAI、Anthropic、xAI、Waymo等少数超级公司吸走。早期项目虽然金额同比增长，但deal count下降，说明“少数更贵的种子/早期轮”替代了“广撒网”。

4.2 2026年的核心投资框架：AI主线 + 物理约束

最值得追踪的方向不是泛AI应用，而是解决AI扩张中真实瓶颈的公司：

1. 算力与数据中心：存储、网络、芯片、冷却、电力调度；

2. Physical AI：机器人、自动驾驶、无人系统、制造执行；

3. 能源与材料：核能/聚变、关键金属、工业脱碳、稀土磁材；

4. 国防与安全：航天安全、自主平台、供应链韧性；

5. 垂直AI应用：金融、客服、营销、开发者工具等已有预算和明确ROI的场景。

4.3 对创业公司的启示：只讲AI故事不够，必须拿出“交付场景”

投资人正在从“模型能力崇拜”转向“商业闭环验证”。能融资的AI公司通常具备至少一项：

  • 明确企业预算入口；
  • 可量化ROI；
  • 数据/工作流/渠道壁垒；
  • 与硬件、能源、制造或国防等高门槛场景结合；
  • 有头部投资方背书并能支撑大额资本开支。

4.4 风险提示：估值透支与退出压力会在下半年显性化

Q1/Q2的融资热度带来两个隐患：

  • 估值集中抬升后，IPO/M&A退出必须跟上，否则中后期基金DPI压力会加剧；
  • 硬科技、国防、能源类项目资本开支大、周期长，若政策或订单兑现慢，下一轮融资压力会非常明显。

5. 后续追踪清单

  • Project Prometheus：是否披露具体产品路线与商业客户。
  • VAST Data / Aria Networks / SiFive：AI基础设施是否出现可持续收入加速。
  • Parallel / Factory / Rogo：agentic AI在开发者、金融、企业流程中的真实留存和付费。
  • 东昇聚变 / Valar Atomics：能源硬科技里程碑是否按计划推进。
  • 中国具身智能头部公司：4月超200亿元融资后，量产交付与订单兑现情况。
