VC投资动态调研｜2026-06-01 18:30

1. 本期核心判断

资本继续押注“AI可交付系统 + 物理世界 + 关键基础设施”：Q1/4月资金端仍由 frontier/physical AI 巨额轮次主导，但可持续的增量机会正在外溢到 AI infra、企业Agent交付、国防航天、能源电力、半导体、医疗运营与中国硬科技供应链。

2. 本期重点交易

  • Project Prometheus｜Late-stage｜$10.0B｜未披露｜Foundational AI / physical-world applications

- AlleyWatch 统计显示，该单笔交易占美国4月VC总额48.1%，是本期最大扰动项。

  • Slate Auto｜Late-stage｜$650M｜未披露｜EV / manufacturing
  • True Anomaly｜Late-stage｜$600M｜未披露｜AI + aerospace / national security
  • VAST Data｜Late-stage｜$500M｜未披露｜AI infrastructure / data platform
  • Vulcan Elements｜Late-stage｜$431M｜未披露｜Defense / rare-earth magnets / advanced manufacturing
  • SiFive｜Late-stage｜$400M｜未披露｜Semiconductor / RISC-V IP
  • Valar Atomics｜Series B｜$340M｜未披露｜Nuclear energy / industrial power
  • 东昇聚变｜天使轮｜数亿元人民币｜红杉中国、IDG资本、中科创星、高瓴、鼎晖百孚、龙芯创投等｜核聚变 / 未来能源
  • 地瓜机器人｜B2轮｜$150M，本轮后B轮累计$270M｜Prosperity7、远景科技、云锋基金等；老股东高瓴创投等｜机器人/边缘AI芯片与具身智能供应链（量子位页面抓取403，使用搜索摘要，可信度中）

3. 赛道热度观察

AI：资本集中度继续抬升，但投资逻辑更偏“可交付”

  • Crunchbase 口径显示，2026年Q1全球VC投资约 $300B / 6000家，环比和同比均增长150%以上；其中 AI 公司获得 $242B，占全球VC资金 80%。
  • 四笔超级轮次 OpenAI（$122B）、Anthropic（$30B）、xAI（$20B）、Waymo（$16B）合计 $188B，约占Q1全球VC资金65%。这说明 headline 被少数公司严重放大。
  • AlleyWatch 口径下，2026年4月美国AI公司获得 $15.18B / 189笔，占全部VC资金 73.0%；剔除 Project Prometheus 后，美国4月仍有 $10.8B / 441笔，底层交易活跃度健康。
  • 资金外溢方向：AI基础设施、企业Agent/工作流交付、实时生成视频、医疗运营、身份风控、自动驾驶/机器人与physical AI。

硬科技：国防、航天、能源、电力和半导体进入核心配置

  • 4月美国Top deals里，True Anomaly、Vulcan Elements、Hermeus等体现国防航天与先进制造热度；Valar Atomics $340M Series B 显示能源/核能正在成为AI算力时代的基础设施资产。
  • SiFive $400M 说明半导体IP与RISC-V生态继续吸引成长资本。
  • 中国侧，东昇聚变天使轮集齐红杉、IDG、高瓴等头部机构，表明“AI的尽头是能源”的逻辑正在从口号转化为长周期风险资本配置。

中国一级市场：募资回暖，硬科技最强

  • 投资界/清科研究数据显示，2026年一季度中国股权投资市场持续回暖：新募集基金 1913只、5590.69亿元，同比 +114.0% / +79.8%；投资案例 2568起、披露金额2344.25亿元，同比 +4.9% / +15.4%。
  • 行业上，IT、半导体、生物技术/医药健康、机械制造活跃度居前，AI、GPU、创新药、智能制造为高热细分方向。
  • 退出端明显改善：Q1共 1182笔退出，其中IPO 762笔、同比 +220.2%，并购退出 213笔、同比 +115.2%，提升硬科技投资人的退出预期。

Biotech / Climate / Energy：医疗修复，能源上行

  • 生物医药融资仍在恢复窗口，但本轮抓取到的biotech tracker更多是入口页，未获得足够可核验的具体单笔交易。
  • 能源侧确定性更强：核能、聚变、电力供应、先进制造材料与AI数据中心需求形成同一条资金主线。

4. 值得关注的早期项目

  • 东昇聚变｜数亿元天使轮｜核聚变：天使轮即获得红杉、IDG、高瓴等头部机构，技术路线为氘-氦3聚变，周期极长但上限极高。
  • 地瓜机器人｜B2轮｜边缘AI/机器人底层系统：B轮累计$270M，重点观察芯片出货、客户绑定与具身智能生态位置。
  • Valar Atomics｜Series B｜核能电力：虽金额已达成长轮规模，但在能源基础设施周期中仍属早期商业化验证阶段。
  • Agentic AI细分公司｜Seed/A轮｜企业工作流Agent：公开榜单显示头部高度集中，早期机会应聚焦明确ROI、强数据闭环、可落地场景。
  • 医疗运营AI｜Seed-B轮｜临床信息流：相较泛医疗问答，更接近付费、合规和医院流程改造。

5. 分析师点评

1. VC市场不是全面狂热，而是“超级AI轮次 + 底层结构性修复”并存：Q1全球$300B和美国4月$20.8B都被巨额AI交易显著放大，但剔除异常值后，Series A、硬科技与中国一级市场仍呈现真实恢复。

2. 资本从模型能力转向系统交付：企业Agent、AI infra、医疗运营、身份风控、实时视频、physical AI会比“又一个通用模型”更容易拿到后续资金。

3. AI约束正从GPU扩展到电力、材料、国防安全与制造能力：能源/核能/聚变、半导体IP、稀土磁体、航天国防进入VC核心名单，本质是AI基础设施供应链投资。

4. 中国市场更强调政策与退出确定性：政府基金、地方国资、金融机构、险资和AIC活跃出资，叠加IPO/并购退出回暖，硬科技项目融资环境改善。

5. 下一步应追踪：Project Prometheus类physical AI公司的真实客户与成本结构；Agentic AI公司的ARR与续费；聚变/核能监管节点；中国具身智能量产订单；医疗AI合规落地和医院采购周期。

6. 可信度与限制

  • 高可信：Crunchbase Q1统计、AlleyWatch 4月美国VC统计、投资界/清科中国Q1数据，均为结构化市场数据。
  • 中可信：21经济网/投资界对东昇聚变报道，交易方和项目背景较完整，但“数亿元”未披露精确金额。
  • 中可信：量子位地瓜机器人页面抓取403，本轮使用搜索摘要交叉；需后续回溯公司公告或投资方公告。
  • 低/补充：部分AI/Agentic AI榜单类网站适合判断赛道热度，不适合单独作为估值或金额依据。
