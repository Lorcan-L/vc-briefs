VC投资动态调研｜2026-05-29 13:30

1. 本期核心判断

资本正在押注 AI 从模型竞赛进入“可交付系统 + 物理世界 + 关键基础设施”阶段：frontier AI 仍制造超级大额融资，但更可持续的增量机会正在向 Agent 工作流、AI 数据中心、身份风控、医疗 AI、国防航天、能源/半导体、具身智能与中国硬科技扩散。

2. 本期重点交易

  • Project Prometheus｜Late-stage / New round in talks｜100亿美元｜JPMorgan、BlackRock 等传为新投资方｜Physical AI / 世界模型 / 制造业 AI

- AlleyWatch 将其列为 2026年4月美国 VC 市场最大单笔交易，单轮占美国当月融资总额约48.1%。AI Funding Tracker 称其估值约380亿美元，目标垂直包括芯片制造、航天、汽车。

  • Anthropic｜Strategic investment｜50亿美元 April tranche；最高可追加至200亿美元｜Amazon｜Frontier AI / Enterprise Safety

- AI Funding Tracker 信息，属于中可信聚合；结合 Q1 frontier labs 超大额融资背景，说明云厂商仍在用资本锁定模型与算力消耗。

  • Cursor / Anysphere｜New round in talks｜20亿美元｜a16z、Nvidia、Thrive Capital 等传参与｜AI Coding / Dev Tools

- 仍为 in-talks 信息，可信度中；但其反映 AI 编程工具正在从开发者工具进入平台级资产定价。

  • Slate Auto｜Late-stage｜6.5亿美元｜未披露｜EV / 汽车制造

- 说明硬件制造融资窗口并未关闭，但资金更偏好具备供应链与量产路径的大资产。

  • True Anomaly｜Late-stage｜6亿美元｜未披露｜Aerospace / AI / National Security

- 航天安全与 AI 的交叉点继续吸引成长资本。

  • VAST Data｜Late-stage｜5亿美元｜未披露｜AI Infrastructure / 数据操作系统

- AI 基础设施从 GPU 采购外溢到数据、存储、调度与操作系统层。

  • Vulcan Elements｜Late-stage｜4.31亿美元｜未披露｜关键材料 / 国防制造

- 稀土磁材、关键材料与供应链安全资产被重新定价。

  • SiFive｜Late-stage｜4亿美元｜未披露｜RISC-V / 半导体 IP

- 半导体 IP 是 AI 算力生态的二阶受益层。

  • Valar Atomics｜Series B｜3.4亿美元｜未披露｜核能 / 工业电力

  • Sygaldry Technologies｜Series A｜1.05亿美元｜未披露｜Quantum-classical AI infrastructure

3. 赛道热度观察

AI：超级大额融资仍抬高总量，但底层活跃度不弱

  • Crunchbase 称 2026Q1 全球 VC 投资约 3000亿美元 / 6000家 startups，其中 AI 公司获得 2420亿美元，占全球 VC 的 80%。
  • Q1 资金高度集中于 OpenAI、Anthropic、xAI、Waymo 等巨型交易，四家公司合计约 1880亿美元，占全球季度 VC 约 65%。
  • AlleyWatch 显示 2026年4月美国 VC 为 208亿美元 / 442笔，AI 公司吸收 151.8亿美元 / 189笔，占资本 73.0%；剔除 Project Prometheus 后，美国市场仍有 108亿美元 / 441笔，说明并非只有一笔交易撑场面。

Agentic AI：从通用 Agent 叙事转向“垂直工作流替代”

  • New Market Pitch 对 2025年6月-2026年5月公开股权融资的统计显示，Agentic AI 样本含 40笔交易、39家公司、14.13亿美元。
  • 其中 Vertical AI Agents 占 65.0% 交易数量、74.5% 披露资本，说明资金不是投“泛 Agent 概念”，而是投能拥有任务、整合系统并交付结果的垂直工作流。
  • 种子轮与 Series A 合计占披露资本 51.6%，该市场仍处早中期扩张阶段。

AI基础设施：从 GPU 外溢到数据、互连、身份、文档与部署

  • VAST Data、nEye.ai、Mintlify、Didit、Unframe、Orkes、Parallel Web Systems 等案例显示，资本正在补齐 AI 生产化链条。
  • 投资逻辑从“谁有最大模型”转向“谁控制 AI 进入生产系统的关键节点”：数据操作系统、AI data center interconnect、机器可读文档、身份风控、workflow orchestration、enterprise delivery。

硬科技：国防航天、能源、半导体、关键材料走强

  • True Anomaly、Vulcan Elements、Hermeus、SiFive、Valar Atomics 共同指向一个趋势：AI 之外，国家安全、工业供应链、电力瓶颈成为大额资金入口。
  • Valar Atomics 的 3.4亿美元 Series B 尤其值得关注，AI 数据中心电力需求正在把核能、储能、工业电力资产重新推回 VC 视野。

中国一级市场：政策确定性 + 技术落地 + 退出通道

  • 新浪财经转载深圳市投资基金同业公会观点称，2026年一季度中国股权投资市场新募基金数量和金额分别同比增长 114.0% / 79.8%，但资金高度集中于先进制造、人工智能、半导体。
  • 中国证券报观察显示，红杉中国、高瓴创投一季度投资从“追概念”转向“量产与订单”：具身智能核心零部件、AI 硬件、垂直行业应用、低空经济、商业航天、聚变能源是重点。

4. 值得关注的早期项目

  • Didit：AI-native 身份与欺诈基础设施，累计种子轮 750万美元。Agent/AI 应用越多，身份验证、风控、合规越会成为横向刚需。
  • Itera：1200万美元种子轮，流体电路板原型。关注点是电子/计算硬件新范式，后续需验证制造可行性与性能指标。
  • Trent AI：1300万美元 Seed，Agentic cybersecurity / securing autonomous agents。Agent 进入企业系统后，安全治理将成为刚需预算。
  • Pomo：450万美元 Seed，agentic marketing intelligence。小额但方向明确：把 Agent 放进高频、决策密集型营销函数。
  • Trace：300万美元 Seed，enterprise agent adoption/control layer。企业 Agent 部署中的 approval、oversight、audit 是早期但重要的基础层。
  • Floatboat：红杉中国、微光创投参与天使轮，工作上下文型 Agent。值得追踪 Agent 是否会以“工作空间/操作系统”形态进入办公流。
  • 大览科技：天使及 Pre-A 连续融资，红杉中国参与领投。代表 AI + 消费硬件把专业能力压缩成日常能力的路径。

5. 分析师点评

1. VC市场是结构性复苏，不是全面复苏：大额资金集中在 AI、硬科技与政策确定赛道，传统软件、消费与缺乏技术迭代的公司仍然融资困难。

2. AI投资进入基础设施化阶段：frontier lab 吞噬资本，但更适合持续跟踪的是数据、互连、身份、文档、workflow orchestration、企业交付等“生产化管线”。

3. Agentic AI 的投资标准正在变硬：市场不再为“能对话的 Agent”买单，而是看是否能接管工作流、接入系统、验证结果、形成预算替代。

4. Physical AI 是新周期叙事核心：Project Prometheus、Glydways、具身智能、机器人、AI硬件共同说明资本正在押注 AI 与物理世界交互的下一阶段。

5. 能源/半导体/关键材料是 AI 的二阶受益者：算力扩张会继续传导到电力、散热、互连、芯片 IP、关键材料和供应链安全。

6. 中国市场看“量产、订单、政策退出”：红杉/高瓴等头部机构的投资语言明显转向产线验证、万台级交付、真实订单和清晰退出通道。
