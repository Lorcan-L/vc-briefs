二、赛道热度观察

1. AI仍是压倒性主线，但高度集中

  • Crunchbase Q1 2026数据显示，全球VC投资约 $300B / 6,000家公司，环比和同比均增长超过150%。
  • AI获得 $242B，约占全球VC总额 80%。
  • OpenAI、Anthropic、xAI、Waymo 四家公司合计 $188B，约占全球季度投资 65%。这说明市场并非均匀复苏，而是被少数AI mega-round显著拉高。

2. Physical AI / 机器人 / 具身智能进入资本抢筹阶段

  • 中国Q1一级市场披露融资事件 2,865起，披露金额 2,560亿元人民币；其中具身智能 210起、超300亿元人民币。
  • A轮即出现20亿-25亿元人民币级别融资：银河通用、灵初智能、千寻智能是代表。
  • 红杉、高瓴等头部机构的逻辑正在从“概念/技术Demo”转向“量产能力、订单、产业落地”。这意味着具身智能正在从叙事融资进入交付验证期。

3. AI基础设施、半导体、数据系统持续吸金

  • VAST Data $500M、SiFive $400M、Aria Networks $125M、Omni $120M等交易说明，资本继续押注AI运行所需的数据、网络、计算和芯片栈。
  • 这类项目的共同特点是：客户预算明确、切入基础设施层、与AI算力/推理增长强相关。

4. 国防科技、商业航天、能源/核能成为“硬科技资金池”

  • Project Prometheus、Hermeus、Flock Safety、Scout AI、Starfish Space等美国案例显示，国防、自治系统、航空航天仍能获得大额支票。
  • 中国星际荣耀50.37亿元D++轮、汇天飞行汽车近2亿美元融资、宇石空间获高瓴等支持，说明商业航天与低空经济在国内也保持活跃。
  • Valar Atomics $340M说明，AI电力需求和能源安全叙事正在推高核能、能源基础设施的融资热度。

5. Biotech回暖集中在AI制药、平台型药物研发和临床资产

  • Isomorphic Labs $2.1B Series B 是AI制药标志性交易。
  • CellCentric $220M、Windward $165M、Sidewinder $137M、Coultreon $125M A轮、CREATE $122M B轮、Oricell >$110M 等显示，资本偏好具备明确机制、临床节点或平台扩展性的biotech项目。
  • AI/机器学习被嵌入靶点发现、分子设计、试验分析，但投资判断仍需回到湿实验验证和临床推进。

四、分析师点评

共识判断

当前VC市场的主叙事是 “AI与硬科技驱动的结构性繁荣”，不是全面复苏。资金总量看起来非常强，但核心由少数AI mega-round与硬科技大额融资拉动。早期市场并未无差别回暖，反而呈现“项目更少、单轮更大、门槛更高”的趋势。

需要避免的误读

1. 不要把Q1 $300B理解为全市场全面复苏：OpenAI、Anthropic、xAI、Waymo四笔交易已贡献约65%的全球季度金额。

2. 不要把AI占80%外推到所有地区与所有阶段：这是Crunchbase Q1全球口径，且受头部交易影响极大。

3. 中国具身智能热度高，但商业化验证仍是分水岭：量产订单、交付成本、可靠性、场景ROI会决定下一轮分化。

4. Biotech融资回暖不是无差别回暖：更受欢迎的是有明确临床节点、平台可扩展性或AI提升研发效率证据的项目。

值得持续追踪的方向

  • AI基础设施：推理成本优化、数据系统、模型评测与安全、企业级AI运维。
  • Physical AI/机器人：制造、仓储、巡检、能源、医疗护理等有明确ROI的场景。
  • AI硬件与端侧入口：健康硬件、AI相机/眼镜/可穿戴、工作流入口型硬件。
  • 国防科技/商业航天/低空经济：关注政策采购、规模化制造与双用途场景。
  • 能源与核能：AI算力需求带动电力供给、储能、电网软件、核能供应链。
  • AI制药与biotech平台：关注是否能用湿实验和临床数据证明研发效率提升。
  • Agentic AI垂直应用：优先金融、法律、医疗、销售、研发等高频高价值工作流。

投资跟踪框架

早期项目建议重点问六个问题：

1. 是否解决明确预算项，而非只提升体验？

2. 是否有数据、分发、工作流嵌入壁垒？

3. AI能力是否带来10倍效率提升或显著成本下降？

4. 毛利、部署周期、交付复杂度是否可控？

5. 创始团队是否同时懂技术、行业和商业化？

6. 12-18个月内是否有可验证里程碑（订单、临床读数、量产、单位经济模型）？
