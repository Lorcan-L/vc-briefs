VC投资动态调研报告｜2026-05-25 18:40

一、结论速览

本轮检索显示，2026年VC市场继续呈现“总量恢复、结构极端集中”的特征：AI仍是全球资金主轴，且资金从纯软件模型继续外溢到物理世界——具身智能、机器人、AI基础设施、半导体、能源、国防航天和生物医药。Q1全球VC投资额在不同数据源口径下达到约3000亿—3309亿美元，其中AI相关融资占比约80%；美国吸收全球80%以上资金，中国则在具身智能、机器人、半导体、低空经济、商业航天和创新药方向明显升温。

本期值得关注的三个关键词：

1. AI资本集中度继续上升：OpenAI、Anthropic、xAI、Waymo等少数巨头吸收巨量资金，全球VC市场被少数mega-round显著拉高。

2. Physical AI/具身智能成为中美共同焦点：Project Prometheus、Recursive、Ineffable Intelligence，以及中国TARS、Galaxea AI、X Square、Robot Era等项目显示，资金正在押注“AI进入物理世界”。

3. 硬科技早期融资异常活跃：东昇聚变数亿元天使轮、TARS 35亿元种子轮、Tribal 1000万美元种子轮等表明，早期轮次也出现大额化，投资人更愿意在政策确定性、技术壁垒和巨大TAM交汇处前置下注。

三、赛道热度观察

1. AI仍是资金主轴，但形态明显变化

Crunchbase显示，2026年Q1全球约3000亿美元VC投资中，约2420亿美元流向AI公司，占比约80%。KPMG口径下，Q1全球VC投资额达3309亿美元，十笔20亿美元以上交易贡献超过2060亿美元，其中多数集中在AI。

但“AI”内部已经分化：

  • 基础模型/Frontier Labs：继续吞噬超大额资本，典型如OpenAI、Anthropic、xAI。
  • AI基础设施：算力、数据层、半导体、AI OS仍是高确定性方向，VAST Data、SiFive是本轮代表。
  • Physical AI/具身智能：成为新增量最明确的方向，机器人、自动驾驶、制造、空间智能、国防航天都在获得资金。
  • 垂直AI应用：企业AI交付、法律、公共安全、金融等方向仍有融资，但相对mega-round显得更“小而确定”。

2. 具身智能/机器人在中国市场进入资本密集期

中国市场的显著信号是：具身智能不再只是概念赛道，而进入“头部项目大额融资+产业化验证”的阶段。新浪财经引用深圳市投资基金同业公会观点称，2026年4月仅8家具身智能企业融资就超过200亿元人民币。AlleyWatch榜单中，TARS、Galaxea AI、X Square、Robot Era均进入全球4月大额融资榜。

投资逻辑从“机器人概念”转为三类验证：

  • 是否有真实工厂/家庭/服务场景落地；
  • 是否具备量产和供应链能力；
  • 是否掌握高壁垒模型、控制、执行器、数据闭环能力。

3. 能源和核技术出现“AI算力外溢”逻辑

Valar Atomics、Blue Energy、东昇聚变、Boston Metal等交易说明，资本开始系统性押注能源底座。背后逻辑并不只是气候科技，而是AI算力、电气化、工业重构共同推高长期能源需求。

核能/核聚变项目虽然周期长、技术风险高，但其TAM巨大，且具备政策支持和战略属性。本轮东昇聚变天使轮由红杉、IDG、高瓴等头部机构共同参与，是中国硬科技早期融资中非常强的信号。

4. 国防科技、航天和低空经济持续升温

True Anomaly 6亿美元Series D、Hermeus 2亿美元Series C、Volant Aerotech 3亿美元Series C显示，航天、国防、eVTOL等资本密集赛道正在进入更高融资层级。KPMG也明确指出，地缘政治不确定性将推动defense tech、spacetech、cybersecurity获得更多VC关注。

5. 生物医药处于估值修复与差异化创新阶段

生物医药不是本期最大额交易的主角，但从中英文检索结果看，biotech funding tracker类信息密集，市场从前两年低谷修复。中国报道也显示，资本关注从“仿制/同质化”转向“差异化创新、AI制药、出海授权、明确临床价值”。该方向的资金热度低于AI/机器人，但退出环境改善后有望持续回暖。

四、值得关注的早期项目

1. TARS｜35亿元人民币种子轮｜具身智能

TARS成立于2025年，种子轮即达到35亿元人民币，投资方包括北京机器人产业基金、蓝驰、CICC、Hillhouse、HSG、线性资本、美团等。种子轮规模异常大，说明头部资本愿意在具身智能核心卡位上极早期下重注。

关注理由：如果其能形成“模型+本体+场景数据+供应链”的闭环，将不只是机器人公司，而可能成为Physical AI平台型入口。

2. Ineffable Intelligence｜11亿美元种子轮｜交互式学习AI

由David Silver创立，种子轮融资11亿美元，投资方包括NVIDIA、Sequoia、Index、Lightspeed等。项目方向是通过交互和经验学习的机器学习系统。

关注理由：强化学习/交互式学习正在成为后LLM时代的重要路线，若能突破真实世界反馈闭环，可能成为下一代基础模型范式。

3. 东昇聚变｜数亿元人民币天使轮｜可控核聚变

成立仅半年，完成数亿元天使轮，红杉中国、IDG、高瓴、中科创星、鼎晖百孚等入局。路线为氘-氦3聚变，并计划分阶段推进净能量增益目标。

关注理由：这是典型“超长周期+超大TAM+政策背书”的硬科技押注。短期商业化不确定性高，但一旦突破，回报天花板极高。

4. Tribal｜1000万美元种子轮｜企业AI平台

由Salesforce、Wix、Spot.io背景团队创立，定位enterprise-native AI platform。

关注理由：与mega-round相比金额不大，但企业AI落地仍处早期。真正理解复杂企业系统、能完成交付闭环的团队，有机会在AI应用层跑出稳定收入。

5. Rasa Legal｜late seed追加100万美元｜法律科技

Rasa Legal获得NextLadder Ventures追加100万美元，属于此前500万美元late seed的一部分。

关注理由：法律科技是AI垂直应用中付费意愿明确、流程标准化程度较高的方向，适合关注小额但可商业化验证的早期项目。

五、分析师点评

1. 2026年VC不是“全面复苏”，而是“结构性繁荣”

总量数据非常亮眼，但不能误读为所有创业公司融资变容易。资金高度集中在AI、AI基础设施、具身智能、半导体、能源、国防航天、生物医药等高确定性或高战略价值赛道。传统消费、普通SaaS、同质化应用仍然融资困难。

判断：VC市场正在从“增长叙事”转向“战略资产叙事”。能拿到钱的公司通常具备至少一个条件：

  • 接近AI/算力/能源/机器人等底层生产力重构；
  • 具备政策确定性或国家战略属性；
  • 有高技术壁垒，且能展示量产/订单/真实场景；
  • 能支撑超大TAM与潜在退出故事。

2. AI投资进入“少数巨头吸金+应用层淘汰”的双轨阶段

巨额资本流向OpenAI、Anthropic、xAI、Project Prometheus等头部玩家，意味着基础模型竞争门槛继续上升。普通AI创业公司如果只是“套模型做应用”，融资会越来越难。

值得追踪的AI方向：

  • AI基础设施：数据层、存储、推理优化、半导体；
  • Physical AI：机器人、自动驾驶、空间智能、工业控制；
  • 垂直AI：有强工作流、强数据闭环、强付费场景的行业应用；
  • Agentic AI：不是demo型Agent，而是能完成企业交付、可审计、可控、可规模化部署的系统。

3. 中国市场最强主线是“政策确定性+硬科技产业化”

中国一级市场正在从互联网/消费模式切换到硬科技和先进制造模式。具身智能、半导体、低空经济、商业航天、创新药、核聚变、新材料等方向受益于政策、产业链和退出机制改善。

值得注意的是，政策支持并不等于所有项目都能融资。资本更看重：

  • 技术是否可量化验证；
  • 是否进入真实场景；
  • 是否有订单或商业化路径；
  • 是否可能通过并购、IPO、产业资本接盘退出。

4. 早期轮次大额化会抬高创业门槛

TARS 35亿元种子轮、Ineffable 11亿美元种子轮、东昇聚变数亿元天使轮说明，在关键基础技术方向，资本会提前拥挤下注。这对创业者有双重影响：

  • 正面：硬科技公司更早获得长周期研发资金；
  • 负面：赛道估值被快速抬高，后来者若没有核心技术或产业资源，进入难度显著增加。

5. 下一阶段建议重点跟踪方向

1. 具身智能产业链：不只看整机公司，还要看执行器、灵巧手、传感器、控制系统、仿真数据、训练平台。

2. AI能源底座：核能、储能、电网调度、绿色金属、数据中心能源优化。

3. AI基础设施二阶机会：推理成本下降、企业数据层、agent orchestration、可观测性、安全合规。

4. 国防航天与低空经济：地缘不确定性会让资本接受更长周期和更高硬件门槛。

5. AI+生物医药：关注从平台故事转向临床/授权/管线验证的公司。
