2. 赛道热度观察

2.1 AI继续吸收绝大部分增量资本

  • Crunchbase 数据显示，2026年Q1全球VC投资约 $300B，约6,000家创业公司获投，同比/环比均增长超过150%。其中AI相关公司获得约 $242B，占全球VC融资约80%。
  • KPMG Venture Pulse 口径更高：Q1全球VC投资从2025年Q4的 $128.6B 跃升至 $330.9B，交易数为8,464起；10笔超过 $2B 的巨型融资合计贡献超过 $206B。
  • 资金明显集中在美国AI生态：KPMG称美洲Q1融资 $270.1B，美国占 $267.2B；Crunchbase称美国公司拿到约 $250B，占全球83%。

判断：AI不再只是应用软件周期，而是“模型 + 算力 + 数据基础设施 + 机器人/自动驾驶/防务”等复合资本开支周期。巨额融资背后，是AI基础设施与physical-world AI对长期资本的强依赖。

2.2 具身智能/机器人成为中国一级市场最强主线

  • 中文公开报道显示，2026年Q1中国一级市场融资事件约2,865起，披露金额约2,560亿元，同比分别增长52%、48%。
  • 具身智能领域约210起融资、超过300亿元；10亿元级融资超过16笔，A轮即获得10亿+融资正在变成头部项目常态。
  • AlleyWatch 4月全球大额融资榜中，Robot Era、Galaxea AI、X Square、TARS 等中国具身智能/机器人项目密集上榜，且高瓴/HSG、IDG、红杉、启明、CICC等头部机构反复出现。

判断：中国市场本轮不是单纯追逐大模型，而是在机器人本体、传感器、灵巧手、动力系统、仿真数据、边缘算力芯片等全链条下注。

2.3 Defense / Space / Nuclear 等硬科技风险偏好回升

  • True Anomaly Series D 融资 $600M，Hermeus Series C 融资 $200M，显示美国防务科技、航天安全、超高速飞行方向持续升温。
  • Valar Atomics Series B $340M、Blue Energy $380M，说明核能/稳定清洁能源在AI算力用电需求拉动下重新获得大额风险资本。
  • KPMG也明确提到，Q2值得关注的方向包括 defense tech、spacetech、cybersecurity。

判断：AI算力需求 + 地缘政治不确定性共同推高了“硬科技/主权科技”的融资确定性。

2.4 Biotech 与 AI制药出现大额样本

  • 华深智药战略融资 $787M（约56亿元），公开报道称刷新2026年全球生物技术公司融资纪录之一。
  • 赛道逻辑从“AI作为科研工具”转向“AI作为药物研发生产引擎”，更看重pipeline转化、商业订单和临床推进速度。

判断：AI制药会继续受关注，但估值分化会比机器人更快，关键取决于真实研发效率、授权交易与临床里程碑。

4. 分析师点评

4.1 当前市场不是全面复苏，而是“结构性过热”

从总量看，Q1 2026全球VC刷新纪录；但资金并没有平均流向所有创业公司，而是高度集中于AI头部公司和硬科技龙头。Crunchbase数据显示，OpenAI、Anthropic、xAI、Waymo四家公司合计融资 $188B，占Q1全球融资约65%。这意味着市场表面火热，但中腰部项目仍可能面临融资难。

4.2 巨型轮次提前化，早期估值体系被重写

过去Series B/C才出现的金额，现在正在前移到Seed/A轮。Ineffable Intelligence 的 $1.1B Seed、TARS 的 ¥3.5B Seed、Recursive 的 $650M Series A，说明顶级人才、顶级叙事、顶级机构背书正在共同压缩传统融资阶段。

风险是：早期项目尚未充分验证产品收入，却已背负独角兽估值，后续融资和退出都需要极强增长兑现。

4.3 “Physical AI”是本轮AI投资的主线升级

Project Prometheus、Waymo、机器人公司、自动驾驶、空间安全、工业制造AI共同指向一个趋势：资本正在从纯软件AI，转向能够改造物理世界的AI系统。这类公司通常资本开支更重、研发周期更长，但一旦落地，护城河也更深。

4.4 中国VC的关键词：具身智能、国资、硬科技、产业协同

中国一级市场中，国资基金、产业资本、地方平台与头部财务VC共同参与具身智能项目。机器人产业链具有政策支持、制造供应链和场景落地优势，可能成为中国2026年最核心的VC主线之一。

4.5 值得持续追踪的方向

1. AI基础设施：数据存储、推理优化、AI OS、数据中心供电与冷却。

2. 具身智能全链条：机器人本体、灵巧手、传感器、运动控制、仿真数据、边缘芯片。

3. Defense / Space / Cybersecurity：地缘不确定性推动预算和资本双重倾斜。

4. Nuclear / Clean Firm Power：AI算力需求使稳定低碳电力成为战略资产。

5. AI制药：重点看授权交易、临床推进和真实研发周期压缩，而非单纯模型能力。

5. 需要警惕的风险

  • 估值泡沫：巨额融资可能透支未来数年增长预期。
  • 退出压力：KPMG和Crunchbase均提到IPO活动仍偏弱，M&A强于IPO；若公开市场窗口不开，后续流动性承压。
  • 资本集中：头部项目融资火热不代表中小创业公司融资环境普遍改善。
  • 地缘与宏观：油价、通胀和政策不确定性可能影响Q2风险偏好。
  • 技术兑现：physical AI、核能、商业航天、AI制药均为长周期赛道，工程化和商业化验证比融资新闻更重要。

6. 本期结论

2026年VC市场进入“AI巨型资本周期 + 硬科技结构性重估”阶段。短期看，资金继续追逐AI基础模型、AI基础设施、具身智能和防务/能源等硬科技；中期看，关键变量是AI商业化收入、机器人规模交付、IPO窗口重启以及巨额估值能否被增长兑现。

投资跟踪建议：优先建立“Physical AI + AI Infrastructure + Firm Power + Defense/Space + AI Biotech”五条主线的项目池，重点观察融资后6-12个月内的客户订单、交付能力、毛利结构和下一轮融资质量。
