VC投资动态调研｜2026-05-26 12:01

1. 本期核心判断

资本正在继续押注“AI + 物理世界”的超大规模平台机会，同时向防务航天、能源基础设施、半导体与临床节点明确的Biotech扩散；中国市场则更强调政策确定性、硬科技壁垒与商业化落地。

  • VAST Data｜Late-stage｜$500M｜未披露｜AI infrastructure / data OS

- 来源：新浪财经/深圳市投资基金同业公会。中国一级市场资金高度集中于具身智能、AI、半导体与硬科技。

3. 赛道热度观察

AI仍是VC资金的主引擎，但结构更“重资产化”

  • Crunchbase数据显示，2026年Q1全球VC投资约$300B / 6,000家创业公司，其中AI相关融资约$242B，占80%。
  • Q1资金高度集中：OpenAI、Anthropic、xAI、Waymo四家公司合计融资$188B，约占全球VC投资65%。
  • AlleyWatch数据显示，美国4月VC为$20.80B / 442笔，同比资本增长63.9%、交易数量增长21.1%；AI公司获得$15.18B / 189笔，占4月美国VC资本73.0%。
  • 但AI不再只是软件应用，资金正明显流向：基础模型、AI基础设施、自动驾驶、机器人、半导体、数据中心、AI+航天防务。

非AI硬科技正在受益于政策、安全与能源瓶颈

  • 防务/航天：True Anomaly、Vulcan Elements、Hermeus等大额交易说明国家安全相关硬科技具备更强资本确定性。
  • 能源：Valar Atomics的$340M Series B显示核能与工业电力供给成为AI算力、制造业回流背景下的重要VC主题。
  • 工业制造：Boston Metal、Slate Auto等案例说明硬件/制造类公司只要有供应链安全或减碳价值，仍能拿到大额资金。

Biotech进入“选择性修复”，资本偏好临床节点和平台复用

  • Xtalks追踪显示，2026年Biotech融资集中在神经精神、免疫、呼吸系统、肿瘤与AI/ML药物发现。
  • 大额资金流向两类公司：一是临床读出明确的资产型公司，如Windward Bio、CellCentric、Coultreon；二是AI/物理驱动的药物设计平台，如Fathom、Isomorphic Labs。

中国一级市场：政策确定性 + 硬科技壁垒 + 商业化闭环

  • 新浪财经/深圳市投资基金同业公会文章显示，2026年Q1中国股权投资市场新募基金数量和金额分别同比增长114.0%和79.8%。
  • 资金高度集中于人工智能、具身智能、半导体、商业航天/低空经济、生物医药、新能源链与高端装备。
  • 4月仅8家具身智能企业合计融资超200亿元人民币，说明中国VC正在把“机器人/具身智能”视作AI应用落地的核心载体。

4. 值得关注的早期项目

  • Tribal｜Seed $10M｜企业原生AI平台

企业复杂系统里的agentic workflow是AI落地最现实场景之一，关注其是否能从“平台能力”转化为标准化交付。

  • Fathom Therapeutics｜Series A $47M｜物理 + AI小分子设计

相比纯AI药物发现叙事，物理模拟、lab-in-the-loop与候选药物优化更接近可验证里程碑。

  • Signadori Bio｜Seed extension｜现货型实体瘤免疫疗法

金额不大，但实体瘤免疫治疗和体内细胞重编程方向值得持续观察。

  • Serif Biomedicines｜Flagship孵化，$50M启动资金｜Modified DNA genetic medicines

Flagship平台孵化的新型遗传药物技术，早期风险高，但若安全性/表达效率成立，平台天花板较高。

  • 中国消费级/工业级具身智能公司｜Pre-A/A轮密集融资｜机器人本体、关节、控制、大脑模型

重点看是否有真实工厂/家庭场景订单，而不是只看demo和发布会。

5. 分析师点评

1. VC市场表面繁荣，但真实主题高度集中。 2026年VC资金增速被AI巨额轮次显著抬高，尤其是Q1全球$300B与4月美国Project Prometheus $10B这类单点交易。剔除超大轮次后，市场仍健康，但不是全面复苏，而是“强主题吸金 + 普通项目继续融资困难”。

2. 资本正在从“AI软件”转向“AI基础设施与物理接口”。 最值得跟踪的不是泛AI应用，而是能承载大额资金和长期壁垒的方向：算力/数据基础设施、机器人、自动驾驶、航天防务、半导体和能源。

3. 中国VC的核心逻辑比美国更政策驱动。 资金青睐“国家战略 + 退出通道改善 + 产业落地”的交集，具身智能、半导体、低空经济、创新药、新能源链是当前最确定的主线。

4. 早期项目融资仍有窗口，但标准更高。 种子/A轮公司需要证明至少一项：真实订单、临床/技术里程碑、可复用平台能力、供应链稀缺性或政策背书。纯概念融资难度继续上升。

5. 后续追踪方向： Agentic AI企业工作流、AI infra、机器人量产订单、能源/核能与算力耦合、防务AI、AI制药临床读出、中国具身智能头部企业估值泡沫风险。
