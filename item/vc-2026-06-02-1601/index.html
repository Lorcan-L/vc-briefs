3. 赛道热度观察

  • 金额端仍高度集中：KPMG称Q1全球VC达$330.9B，十个$2B+ megadeals贡献超$206B；AlleyWatch称4月美国VC $20.8B中Project Prometheus单笔占48.1%。
  • AI从“模型竞赛”向“控制层/交付层”外溢：OpenRouter、Catena、Canyon Code、agentic workflow公司说明资本更愿意买真实流量、治理和集成能力。
  • 硬科技重新获得机构型资本：国防航天、核能电力、半导体IP、稀土磁材、机器人量产链条同时进入大额融资榜。
  • 中国一级市场主题是“技术落地”：头部机构对具身智能的判断从Demo转向量产、订单、产线与核心零部件。
  • 退出端改善给VC风险偏好托底：清科称Q1 VC/PE支持中企IPO 42家，涉及437家机构，账面价值与回报倍数同步回升。

4. 值得关注的早期项目

  • Catena Labs：用银行牌照、身份、审计和治理去解决“AI agent 碰钱”的合规瓶颈，若 agentic commerce/finance 成立，是高杠杆基础设施位。
  • P2 Science / D-CRBN：工业气候科技不再靠“碳叙事”，而是绑定原料韧性、性能材料和战略客户，说明 climate tech 的融资窗口更偏产业订单型。
  • Floatboat：红杉/微光参与天使轮，试图把 Agent 做成工作上下文与流程入口；仍早期，但符合“AI进入真实工作流”的方向。
  • 大览科技：天使+Pre-A连续融资，红杉参与领投，软硬件算法协同压缩专业长焦能力，属于AI消费硬件的“具体场景优先”路线。

5. 分析师点评

本期不宜把headline解读成全市场泡沫式恢复：美国与全球金额端被OpenAI/Anthropic/xAI/Project Prometheus等巨轮次显著放大。更值得跟踪的是“剔除异常值后仍有广度”的中早期交易：AI inference routing、agent control plane、agentic finance/identity、医疗/采购/财务等垂直工作流、机器人量产链条、能源与半导体基础设施。投资上应优先看三类指标：真实使用量或订单、合规/系统集成壁垒、是否卡住产业链瓶颈。
