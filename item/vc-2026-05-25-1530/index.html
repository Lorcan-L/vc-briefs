二、赛道热度观察

1. AI 仍是绝对主线，但资金更集中于“基础模型 + 物理世界AI + AI基础设施”

Crunchbase 数据显示，2026年一季度全球风险投资约 3000亿美元，环比和同比均增长超过150%；其中 AI 公司获得约 2420亿美元，占全球VC融资约80%。这不是普通意义上的软件周期，而是由模型、算力、机器人、自动驾驶、半导体、数据中心共同推动的资本密集周期。

本期最具代表性的信号是 Project Prometheus 100亿美元融资，以及 Ineffable Intelligence 11亿美元 Seed、Recursive 6.5亿美元 Series A。这说明顶级资本仍愿意在“潜在平台级AI公司”上进行超大额前置下注。

2. 具身智能/机器人在中国市场呈现集群式融资

中国市场的资金集中度同样很高。新浪财经/深圳市投资基金同业公会指出，2026年4月仅8家具身智能企业就获得超过200亿元人民币融资。AlleyWatch 的全球榜单中，TARS、Galaxea AI、Robot Era、X Square、GigaAI、Shengshu Technology 等中国AI/机器人/视频生成公司连续出现。

这类项目共同特征：成立时间短、融资额大、股东阵容强、普遍强调量产/交付/订单，而非只讲技术概念。

3. 国防、航天、低空经济与能源成为“AI之外”的高确定性硬科技

True Anomaly 6亿美元 Series D、Hermeus 2亿美元 Series C、Volant Aerotech 3亿美元 Series C，以及中国证券报提到的汇天飞行汽车约2亿美元融资，均指向一个趋势：高壁垒硬科技正在重新获得大额资本。

能源侧，Valar Atomics 3.4亿美元 Series B、Blue Energy 3.8亿美元融资，显示核能/清洁电力成为 AI 算力扩张之后的“基础设施约束”投资方向。

4. 生物医药和工业脱碳回暖，但资金更挑项目

生物医药从估值低谷修复，资本更偏好差异化创新、AI制药、出海授权路径。工业脱碳方面，Boston Metal 7500万美元融资说明绿色金属、关键材料国产化/本地化供应仍有资金支持，但整体不如AI/机器人赛道激进。

三、值得关注的早期项目

1. Ineffable Intelligence：11亿美元 Seed，创始人为 David Silver，定位“通过交互与经验学习的机器学习系统”。金额异常大，说明基础AI领域仍存在“名人创始人 + 平台级叙事”的估值溢价。

2. TARS：35亿元人民币 Seed，2025年成立，具身智能方向，投资方包括启明、高瓴、HSG、线性资本、美团等。Seed 阶段即进入全球大额融资榜，是中国机器人热度的典型样本。

3. Tribal：1000万美元 Seed，企业原生AI平台，由 Salesforce、Wix、Spot.io 背景团队创立。相比基础模型，企业工作流AI金额较小但商业化路径更短。

4. Floatboat：天使轮获红杉中国、微光创投投资，切入 Agent 工作空间，让 Agent 接管工作上下文、流程与方法。早期但方向贴近“AI应用元年”。

5. OdyssLife / 大览科技：消费级AI硬件早期项目，分别切入健康记录与长焦影像。红杉、高瓴对“AI成为人的延伸”的消费硬件方向开始系统试探。

四、分析师点评

1. 当前VC市场不是全面复苏，而是结构性狂热

融资总量刷新纪录，但并不意味着所有创业公司融资环境变好。资金高度集中于少数赛道、少数头部公司、少数叙事：AI基础模型、算力与数据基础设施、具身智能、国防航天、能源基础设施。传统消费、一般SaaS、缺少技术壁垒的项目仍会感到融资寒意。

2. 2026年的核心投资标准：技术壁垒 + 商业化验证 + 政策/基础设施确定性

中国市场的投资逻辑尤其清晰：先进制造、AI、半导体、具身智能、商业航天、低空经济等方向有政策确定性；但资本不再只为PPT买单，量产能力、订单、真实场景落地成为重要分水岭。

3. AI投资正在从“模型竞赛”外溢到物理世界

Project Prometheus、Waymo、机器人集群融资、核能/清洁能源融资共同说明，AI周期的下一阶段不是纯软件，而是算力、电力、机器人、制造、国防和交通等物理世界系统的重构。资本开支会显著高于上一代互联网周期。

4. 早期机会仍存在，但最好避开“共识最拥挤”的正面战场

基础模型与通用机器人已进入巨额资本军备赛，普通早期团队很难正面竞争。更值得追踪的是：

  • 垂直行业AI Agent，尤其是能直接嵌入业务流程、有明确ROI的方向；
  • 机器人核心零部件、仿真数据、操作系统、测试平台；
  • AI基础设施中的数据、存储、推理优化、合规与安全；
  • 能源、电力、散热、材料等 AI 算力扩张的瓶颈环节；
  • 低空经济、商业航天、国防科技中的供应链和软件系统。
