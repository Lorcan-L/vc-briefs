VC投资动态调研｜2026-06-05 16:30

1. 本期核心判断

资本正在押注 AI 进入真实经济后的“稀缺控制层”：AI 数据中心网络、能源/电力、3D 世界模型、机器人训练数据、垂直 Agent 工作流、国防航天、半导体与 AI-native biotech。

2. 本期重点交易

  • Maxwell Power｜Strategic Capital / Project Finance｜$750M｜Fairtide Partners｜能源基础设施、储能与太阳能部署

可信度：高。TechStartups 报道，属于项目融资/战略资本承诺，不是传统股权轮次；但代表 AI 用电压力下能源融资平台的吸金能力。

  • DriveNets｜Series D｜$410M｜Bessemer Venture Partners、Atreides Management；AMD 等参投｜AI 数据中心网络/以太网 fabric

可信度：高。公开报道显示累计融资达 $1B，已拥有 $1B+ secured business，现金流为正；AI 集群网络层从后台能力变成核心瓶颈。

  • Tripo AI｜Series A+ / A++｜近 $200M｜Ince Capital、长三角科技基金等｜AI 3D / world model

可信度：中高。TechStartups 与二级报道交叉验证；资金用途指向持久化世界模型、3D 生成、数据与基础设施。

  • Mecka AI｜Series A + follow-on｜$60M｜Framework Ventures；Menlo Ventures、SV Angel、Kindred Ventures 等｜机器人训练数据

可信度：中高。披露为两笔融资合并：$25M Series A + $35M follow-on；核心资产是人类运动/行为数据采集。

  • Project Prometheus｜Late Stage｜$10B｜未完全披露｜Physical-world foundational AI

可信度：中高。AlleyWatch 将其列为 2026年4月美国最大 VC 交易，占当月美国融资 48.1%；说明资本仍愿为“物理世界 AI 基座”支付巨额溢价。

  • Valar Atomics｜Series B｜$340M｜未列明｜核能 / 工业电力 / Power Grid

可信度：高。AlleyWatch 统计中为 4月美国 Series B 最大交易之一；AI 电力约束继续把核能推入 VC 主流视野。

  • 中国具身智能与机器人批量融资｜多轮次｜4月8家公司合计超200亿元人民币｜含今日资本、高瓴创投、BV百度风投等｜具身智能、机器人

可信度：中高。新浪财经转载深圳市投资基金同业公会，细节偏宏观但与国内政策硬科技资金流一致。

3. 赛道热度观察

AI 基础设施：从“模型”转向“瓶颈层”

AlleyWatch 统计显示，美国 2026年4月 VC 活动达到 $20.80B / 442 deals，同比资本 +63.9%、交易数 +21.1%；AI 公司拿走 73.0% 资本。单一大额 Project Prometheus 放大了数据，但剔除后仍有 $10.80B / 441 deals，说明底层市场并不弱。

本轮最强信号不是“AI 应用又融资了”，而是资本流向网络、数据中心、机器人数据、能源、电力和物理世界 AI。这类公司控制 AI 落地的关键输入，具备更强议价能力。

Agentic AI：垂直工作流替代 > 泛用 Agent 概念

New Market Pitch 汇总的 2025-2026 agentic AI 数据显示，40 笔披露交易合计 $1.413B；其中垂直 AI Agents 占 65.0% 的交易量与 74.5% 的资本。Seed + Series A 合计占 51.6% 资本，说明市场仍给早中期项目机会，但更偏好“能接管具体工作流”的公司，如 SOC、会计、采购、医疗后台、金融犯罪合规等。

硬科技：中国资金继续围绕政策确定性聚集

中文线索显示，先进制造、AI、半导体、具身智能、商业航天、低空经济、生物医药与新能源链是资金密集区。2026年一季度中国股权市场新募基金数量和金额分别同比 +114.0% / +79.8%，但资金并非普惠，而是集中给“政策背书 + 技术壁垒 + 商业化闭环”的项目。

Biotech：AI-native 与临床工作流更受关注

公开 biotech tracker 页面本轮抓取有效信息有限，仅确认其通过新闻、PR 与数据库持续追踪 2026 biotech funding。结合 TechStartups 的 Waypoint Bio、Veritas Aortic 等案例，资金更偏好能把 AI/实验系统推进到治疗设计、临床流程或医疗器械替代路径的项目。

4. 值得关注的早期项目

  • Mecka AI：不是做机器人本体，而是做机器人训练数据基础设施。若“数据不可从网页抓取”成为具身智能约束，训练数据平台会成为上游收费节点。
  • Agent execution / control layer 公司：如 agent cloud、工具集成、权限控制、人类审批与可观测性平台。Agentic AI 数据显示，基础设施虽不是最多交易，但会随企业部署增长而补涨。
  • AI-native 专业服务 Agent：采购、会计、SOC、医疗后台、金融犯罪合规等高频、高人力成本、强系统集成场景，比泛用助手更容易拿到 Series A/B。
  • 能源/电力开发融资平台：Maxwell Power 这类公司证明，AI 电力需求正在把“项目融资能力”变成科技投资主题。

5. 分析师点评

1. VC 市场不冷，但分化极强。 大额资金涌向少数“基础设施级”资产，普通 AI wrapper 的融资窗口明显收窄。

2. AI 投资从应用叙事转向资源约束叙事。 网络、电力、算力效率、数据、机器人训练环境、世界模型、临床实验系统，都是下游必须购买或绕不过去的环节。

3. 早期项目仍有窗口，但要证明“工作流接管”。 只做聊天界面不够，必须接系统、拿权限、交付可量化结果，并形成数据/流程复利。

4. 中国市场看“政策确定性 + 退出确定性”。 先进制造、半导体、具身智能、商业航天/低空经济、生物医药和新能源链会继续吸引人民币基金，但商业化证据比概念更重要。

5. 后续追踪方向：AI 数据中心 networking、核能/储能融资平台、机器人数据采集、企业 Agent 控制层、AI-native healthcare/biotech、中国具身智能量产订单。
