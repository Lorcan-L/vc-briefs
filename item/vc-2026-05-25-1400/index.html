二、赛道热度观察

1. AI：从“软件模型”升级为“资本密集型基础设施周期”

  • Crunchbase数据显示，2026年Q1全球VC约3000亿美元，AI公司获得约2420亿美元，占80%。
  • 资金高度集中：OpenAI、Anthropic、xAI、Waymo合计1880亿美元，占Q1全球VC约65%。
  • 4月美国市场中，AI公司获得151.8亿美元，占美国VC总额73%；Project Prometheus单笔100亿美元几乎决定了月度总量。
  • 结论：AI融资不是传统软件周期，而更像“算力/能源/制造/物理世界AI”的资本开支周期。融资额会继续被少数巨头和基础设施型公司拉高。

2. 具身智能/机器人：中国一级市场最拥挤的明星赛道

  • 中文来源显示，2026年4月仅8家具身智能企业融资超过200亿元人民币。
  • 海外榜单中，Robot Era、Galaxea AI、X Square 等中国机器人公司进入全球大额融资榜。
  • 顶级机构（HSG/红杉、高瓴、IDG、CICC、小米、阿里等）与地方/产业基金共同入局，推动估值快速上行。
  • 风险：估值可能已经透支短期商业化，下一阶段关键指标会从“Demo能力”转向量产成本、可靠性、订单与交付。

3. 国防、航天、先进制造：非AI资金的主线

  • True Anomaly、Vulcan Elements、Hermeus、SiFive 等交易说明美国市场非AI资金集中在国家安全、供应链安全、半导体和硬件制造。
  • 这类公司通常资本开支重、周期长，但受政策采购和战略安全驱动，融资确定性高于一般消费/软件项目。

4. 能源成为AI时代的“二阶投资主题”

  • Valar Atomics、Blue Energy、东昇聚变等交易显示，核能/聚变/清洁电力开始从远期概念走向VC重点配置。
  • 背后逻辑：AI数据中心、工业电气化和先进制造都需要更稳定、更低成本、更大规模的电力供给。
  • 可控核聚变仍是高风险长周期方向，但顶级机构正在用小比例仓位换取超高上行空间。

5. 生物医药：估值修复，但更挑项目

  • EpiBiologics 1.07亿美元B轮说明，生物医药并未被AI完全挤出资金池。
  • 资金偏好从“泛平台故事”转向差异化靶点、明确临床路径、可授权出海的创新药资产。
  • 中国来源也提到创新药、AI制药与差异化出海是生物医药融资回暖的重点。

三、值得关注的早期项目

1. 东昇聚变（天使轮，数亿元人民币）

- 亮点：红杉中国、IDG、高瓴、中科创星、鼎晖等共同进入天使轮；来自复旦大学磁约束聚变团队，路线为氘-氦3聚变。

- 关注点：商业化极远、技术难度极高，但若上海聚变产业集群形成，可能成为未来能源赛道核心标的。

2. Tribal（Seed，1000万美元）

- 亮点：Salesforce、Wix、Spot.io 背景团队，切入企业原生AI平台。

- 关注点：企业AI应用仍拥挤，需验证是否能绕开大厂平台化挤压，形成系统级集成壁垒。

3. Rasa Legal（Late Seed追加100万美元，累计轮次约500万美元）

- 亮点：法律科技在垂直AI应用中具备高ROI和明确付费场景。

- 关注点：适合观察AI Agent在高频、标准化、强监管业务中的落地节奏。

4. GigaAI / Shengshu Technology（Series B）

- 虽非种子轮，但成立时间短、成长极快。

- 亮点：从视频生成走向空间智能/世界模型，可能成为下一代内容生产和机器人仿真的基础层。

四、分析师点评

市场判断

VC市场表面上全面复苏，但结构上极度分化。2026年的主线不是“风险偏好普遍回升”，而是资本在少数高确定性或超大叙事方向上的集中下注：AI基础设施、具身智能、国防航天、半导体、能源与先进制造。传统互联网、普通SaaS、消费项目仍很难享受同等估值修复。

资金规模趋势

  • 总额被巨型交易严重扭曲：Q1全球3000亿美元、4月美国208亿美元，都被少数AI巨额融资拉高。
  • 健康信号在Series A：AlleyWatch数据显示，美国4月Series A共98笔、平均2000万美元，分布更均匀，是底层风险偏好修复的更好指标。
  • 晚期资本回来了，但只给头部：Late-stage占美国4月资金76.4%，但很大程度受Project Prometheus影响。剔除异常值后市场仍强，但并非所有晚期公司都能轻松融资。

值得追踪方向

1. Physical AI / 具身智能：机器人、自动驾驶、空间智能、世界模型。重点看量产和真实订单。

2. AI基础设施：数据、存储、芯片、算力调度、企业AI交付平台。重点看客户粘性和毛利结构。

3. 能源基础设施：核能、聚变、储能、电网。AI数据中心扩张会持续强化该主题。

4. 国防科技与供应链安全：航天、稀土磁体、先进制造、半导体IP。政策和采购可提供更稳定退出预期。

5. 差异化创新药/平台型生物技术：关注临床推进、BD授权和出海潜力。

风险提示

  • AI与机器人赛道估值存在明显拥挤交易特征，后续融资需依赖商业化验证。
  • 巨额融资数据容易掩盖中小创业公司的真实融资难度。
  • 硬科技项目资本开支重、回报周期长，对基金期限和退出路径要求更高。
  • 中文媒体部分数据为行业协会或媒体整理，需以公司公告、工商/监管披露进一步校验。
