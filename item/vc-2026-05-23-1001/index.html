二、赛道热度观察

1. AI不是单一赛道，而是“算力—模型—数据—物理世界”的资金黑洞

4月全球大额融资榜显示，最大交易集中在 Physical AI、基础模型、AI基础设施和生成式视频。Project Prometheus 单笔 100 亿美元极大拉高了表观金额，但即使剔除异常值，Recursive、Ineffable、VAST Data、Shengshu、GigaAI 等交易也说明：资本仍在为“可形成平台层/入口层/数据层”的AI公司支付高估值。

2. 具身智能/机器人在中国进入“融资月”

新浪/深圳市投资基金同业公会报道指出，2026年4月仅8家具身智能企业融资合计超过200亿元人民币。AlleyWatch 的全球榜单也出现 TARS、自变量、Galaxea AI、Robot Era 等中国机器人公司。资金逻辑正在从“机器人概念”转向“可量产、可交付、可进入工厂/家庭/服务场景”的产品化能力。

3. 硬科技复苏由政策资金和退出改善共同驱动

清科/投资界数据显示，2026年一季度中国股权投资市场新募基金 1,913 只、募资 5,590.69 亿元，同比分别增长 114.0% 和 79.8%；投资案例 2,568 起、披露金额 2,344.25 亿元，同比分别增长 4.9% 和 15.4%。退出侧 IPO 762 笔、同比增长 220.2%，并购退出 213 笔、同比增长 115.2%。这说明市场不是单纯“钱多”，而是 LP、国资、政策工具和退出预期共同改善。

4. 国防航天、低空经济、核能正在成为AI之外的大额交易池

True Anomaly、Hermeus、Volant Aerotech、Blue Energy、Valar Atomics 反映出两个共同点：第一，地缘安全与能源安全正在吸引长期资本；第二，项目需要重资产和长期研发，天然形成高融资额，但也意味着商业化周期和监管风险更高。

四、分析师点评

共识判断

当前VC市场呈现“结构性繁荣”，不是广谱复苏。资金集中流向四类资产：

1. AI平台层与基础设施：能够成为未来应用生态的底座；

2. Physical AI / 具身智能：AI从数字世界进入物理世界，叙事强、票额大；

3. 政策确定性硬科技：半导体、机器人、低空经济、商业航天、创新药；

4. 安全资产：国防航天、核能、能源基础设施。

资金规模趋势

大额交易正在显著扭曲整体市场观感。Project Prometheus 100亿美元、Ineffable 11亿美元等交易会让“融资总额”看起来非常火热，但清科数据中案例数同比仅增长 4.9%，说明底层市场并非全面宽松。真正改善的是头部项目、政策友好赛道和退出预期清晰资产的融资能力。

后续值得追踪的方向

  • 机器人量产交付数据：订单、BOM下降、实际场景复购，比发布会和demo更关键。
  • AI基础设施利润率：VAST Data、SiFive 等需要观察收入增长能否覆盖资本开支和客户集中风险。
  • AI应用从视频/内容走向工业场景：空间智能、世界模型、仿真数据可能成为下一批融资关键词。
  • 核能与国防航天的政策窗口：高融资额背后是长周期审批与政府采购周期，需持续验证非稀释资金和订单质量。
  • 中国人民币基金复苏的可持续性：政府基金、国资平台、AIC和保险资金活跃，但市场化退出能否持续，是判断2026年下半年热度的核心。
