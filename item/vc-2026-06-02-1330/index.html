3. 赛道热度观察

  • AI 仍是最大吸盘，但结构正在分层。 Crunchbase 披露 Q1 全球 VC 约 $300B，其中 AI 约 $242B，占 80%；KPMG 口径 Q1 全球 VC 为 $330.9B，10 笔 $2B+ 交易贡献超过 $206B。资金高度集中于少数头部模型与基础设施公司。
  • 从“模型竞赛”外溢到“可交付系统”。 KPMG 明确提到投资兴趣延伸到 semiconductors、datacenters、AI platforms、agentic AI、physical AI 与垂直解决方案；这比单纯押注 LLM 更接近产业落地。
  • 早中期交易集中在真实工作流。 Waypoint Bio、Triomics、Didit、Reactor、Itera 等近期交易说明：医疗工作流、身份风控、实时视频、硬件研发工具链更容易形成明确客户与预算闭环。
  • 硬科技/国防/能源重新获得溢价。 Defense tech、spacetech、cybersecurity、semiconductor、energy management 和 dual-use technologies 被多来源反复提及，反映地缘与 AI 算力需求共同推高基础设施资产价值。
  • 中国市场偏向政策确定性和量产能力。 先进制造、AI、半导体、具身智能、商业航天、低空经济、创新药、新能源链成为融资主线；资金更看重技术壁垒、研发强度、场景落地和退出路径。

4. 值得关注的早期项目

  • Didit：身份与欺诈 AI infrastructure，seed 总融资 $7.5M；适合追踪其客户数、KYC/KYB API 调用规模、金融/平台型客户留存。
  • Itera：fluid circuit board 概念切入硬件研发效率，$12M seed；如果能缩短电子硬件原型周期，可能成为 robotics / IoT / defense hardware 的共性工具。
  • Waypoint Bio：$20M Series A，AI-native biotech + spatial biology + CAR T；关注其候选管线、动物实验/临床前数据和 pharma partnership。
  • Triomics：$22M Series B，oncology clinical workflow；比“泛医疗 AI”更靠近医院运营和临床试验匹配的可付费场景。
  • Reactor：$59M 出隐身，实时生成视频/世界模型开发者平台；关注其是否从 demo 走向可调用 API、开发者生态和成本曲线。
  • 中国具身智能/机器人供应链早期公司：尤其是“大脑/控制器/关节/灵巧手/场景落地”环节；高估值背后必须看订单、交付、毛利和可靠性。

5. 分析师点评

当前 VC 市场不是简单的“全面回暖”，而是“总量繁荣、结构分化、异常值强干扰”。OpenAI、Anthropic、xAI、Waymo 这类超级轮次会让总量看起来极其乐观，但对早期投资的参考价值有限。真正可操作的投资主线是：

1. 企业 Agent / 工作流 AI：能切进预算、节省人力、提高转化或降低风险的系统；

2. AI infrastructure：数据、存储、半导体、算力调度、开发者平台、模型运维；

3. 身份风控与合规 AI：金融、平台经济、跨境业务都需要；

4. 医疗运营 AI 与 AI-native biotech：从“科研叙事”转向临床、试验、支付与数据闭环；

5. Physical AI / robotics / defense / energy：AI 进入物理世界后，供应链、可靠性、能源和国防需求会创造长周期壁垒。

下一步建议追踪：剔除 $10B+ 级异常交易后，按“客户预算明确 + 技术壁垒可验证 + 数据/交付闭环 + 退出路径可讲清”筛选早期项目；尤其关注 identity/fraud AI、医疗运营 AI、实时视频/世界模型开发平台、中国具身智能量产链条，以及能源/半导体/国防交叉资产。
