VC投资动态调研｜2026-05-29 15:06

1. 本期核心判断

资本正在继续押注“AI 可交付基础设施”：最大票面集中在 physical/frontier AI 与 AI 数据中心，二级扩散到企业 Agent、身份风控、医疗 AI、深科技硬件、能源/半导体与中国具身智能。

2. 本期重点交易

  • Project Prometheus｜Late-stage / 洽谈口径｜$10B，估值约 $38B｜JPMorgan、BlackRock 等传闻参与｜Physical AI / 世界模型 / 制造业。AlleyWatch 与 AI Funding Tracker 均把它列为 2026年4月最大AI/VC交易；AlleyWatch称该交易占美国4月VC总额48.1%。
  • Anthropic｜战略投资 / Frontier AI｜$5B April tranche；另有更大承诺口径｜Amazon 等｜Frontier AI / 企业安全AI。AI Funding Tracker 口径显示 Amazon 继续加码 Claude 生态，云算力绑定是核心。
  • Cursor / Anysphere｜新融资洽谈口径｜$2B，估值或超 $50B｜a16z、Nvidia、Thrive Capital 等传闻｜AI Coding / Developer Tools。投资逻辑从“开发者工具”升级为 AI 编程入口和企业研发工作流。
  • Glydways｜Series C｜$170M｜Suzuki、ACS Group、Khosla Ventures 共同领投｜Autonomous Transit / Physical AI。战略资本组合指向制造、基建交付与商业化落地。
  • Omni｜Series C｜$120M，估值约 $1.5B｜ICONIQ Growth 领投｜AI Analytics / Enterprise BI。企业BI正在从静态报表走向AI原生运营工作流。
  • Sygaldry Technologies｜Series A｜$105M｜深科技投资方｜Quantum-classical AI infrastructure。资本关注AI推理瓶颈下的下一代算力加速层。
  • nEye.ai｜Series C｜$80M｜AI数据中心基础设施投资方｜Optical switching / AI data center interconnect。GPU集群规模扩大后，互连层成为新瓶颈。
  • AcuityMD｜Series C｜$80M｜StepStone Group 领投｜AI Medtech / Commercial intelligence。医疗器械商业化数据与市场准入成为垂直AI成熟场景。
  • Artemis｜Growth Round｜$70M｜Cybersecurity-focused growth investors｜Agentic Cybersecurity。安全场景从告警面板转向可闭环处置的AI Agent。
  • Legora｜Round extension｜$50M｜NVentures｜Legal AI / Enterprise。Nvidia 战略投资说明法律AI正在成为高算力、生产级企业垂直赛道。
  • Mintlify｜Series B｜$45M｜Developer-focused growth investors｜AI documentation infrastructure。AI coding agent 普及后，“机器可读文档”成为上游基础设施。
  • Triomics｜Series B｜$22M｜医疗AI投资方｜Oncology AI / Clinical operations。VCNewsDaily显示其帮助癌症中心运营复杂临床信息。
  • Signos｜未披露轮次｜$20M｜医疗健康投资方｜FDA-cleared OTC glucose monitoring / weight management。
  • Itera｜Seed｜$12M｜深科技投资方｜Fluid circuit board / Advanced manufacturing。
  • Didit｜Seed｜$7.5M total seed；新增 $6M｜早期投资方｜AI-native identity & fraud infrastructure。
  • 维他动力｜Pre-A｜近 5亿元人民币｜今日资本、高瓴创投、BV百度风投等｜消费级机器人 / 机器狗。新浪财经援引行业材料称中国4月具身智能融资明显集中。
  • 地瓜机器人｜B轮累计｜$270M；近期新增 $150M｜未完全披露｜机器人/具身智能基础设施。来自中文检索结果，作为中国具身智能高热度佐证，可信度中等。

3. 赛道热度观察

  • AI资本集中度仍高，但结构更“基础设施化”：AlleyWatch 统计美国2026年4月VC为 $20.80B / 442笔，同比资本 +63.9%、交易数 +21.1%；AI公司吸收 73.0% 资本，其中 Project Prometheus 单笔占48.1%。
  • Late-stage被巨额交易扭曲，Series A更能反映真实广度：AlleyWatch显示美国4月 Late-stage 资本占76.4%，但剔除Prometheus后市场仍有 $10.80B/441笔，说明不是单点泡沫，底层交易活跃度仍强。
  • 企业 Agent / AI 工作流进入预算化阶段：Unframe、Omni、AcuityMD、Artemis、Mintlify、Didit 等显示，资本更偏好能嵌入企业流程、合规、安全、研发和商业化环节的AI基础设施，而不是泛化聊天入口。
  • Physical AI 与硬科技成为AI外溢主线：Project Prometheus、Glydways、中国具身智能融资、Slate Auto、True Anomaly、Vulcan Elements、Valar Atomics、SiFive等共同指向制造、国防航天、半导体、能源与机器人。
  • 中国市场由政策确定性和退出预期驱动：新浪财经/投资机构材料显示，先进制造、AI、半导体、具身智能、商业航天、生物医药、新能源链是当前股权融资优先级最高的方向；融资分化明显，传统弱技术迭代行业仍冷。

4. 值得关注的早期项目

  • Didit：Seed $7.5M，AI-native 身份与反欺诈基础设施。Agent经济和自动化交易越普及，KYC、风控、身份验证越会成为底层刚需。
  • Itera：Seed $12M，fluid circuit board 原型出 stealth。若可规模化，可能切入先进制造、散热、半导体封装或新型电子系统。
  • Mintlify：Series B $45M，AI-readable documentation。AI Coding Agent 可靠性依赖文档可读性，属于“Agent工具链上游卖铲子”。
  • Triomics：Series B $22M，肿瘤临床信息运营AI。医疗AI从影像/诊断扩展到运营和临床流程，商业化阻力相对更可控。
  • 维他动力：Pre-A 近5亿元人民币，消费级机器人。若真实量产和渠道数据持续改善，中国具身智能可能从工业示范进入消费硬件窗口。

5. 分析师点评

本期的核心变化不是“AI还热”，而是资本正在重新定义什么样的AI值得高估值：能够进入真实预算、绑定算力/产业资源、改善交付效率、并服务物理世界或关键基础设施的公司，获得明显溢价。单纯模型Demo、轻量应用和无商业闭环的AI创业会继续被挤压。未来30-60天建议重点追踪三条线：

  • AI基础设施瓶颈：数据中心互连、推理加速、AI可观测性、文档/数据治理、安全与身份。
  • Physical AI商业化：机器人、自动驾驶/交通、工业制造、航空航天和真实场景数据闭环。
  • 政策/退出驱动硬科技：中国半导体、具身智能、商业航天、新能源与AI制药，重点看订单、量产、研发强度和上市/并购退出路径。
