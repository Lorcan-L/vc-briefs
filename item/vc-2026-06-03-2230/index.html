VC投资动态调研｜2026-06-03 22:30

1. 本期核心判断

资本正在押注 AI 从模型能力转向生产资料与真实世界基础设施：超级AI/AI开发入口继续吞下巨额资金，但更可操作的增量信号集中在垂直Agent、AI基础设施、医疗AI/AI-native biotech、身份风控，以及国防航天、核能/电力、半导体、先进制造与中国具身智能量产链。

2. 本期重点交易

  • Project Prometheus｜Late-stage｜$10.0B｜未披露｜Foundational AI / physical-world AI。AlleyWatch称该笔占美国4月VC总额 48.1%，是本期金额端最大异常值。
  • OpenAI｜Q1巨额融资｜$122B｜未披露/多方｜Frontier AI。Crunchbase Q1统计将其列为四大历史级融资之一。
  • Anthropic｜Q1巨额融资｜$30B｜未披露/多方｜Frontier AI。继续强化基础模型公司的资本黑洞效应。
  • xAI｜Q1巨额融资｜$20B｜未披露/多方｜Frontier AI / AI compute。说明算力与模型训练能力仍被高溢价定价。
  • Waymo｜Q1巨额融资｜$16B｜未披露/多方｜自动驾驶 / physical AI。自动驾驶重新成为AI资本外溢到物理世界的关键方向。
  • Slate Auto｜Late-stage｜$650M｜未披露｜EV / 汽车制造。硬件制造类后期资产仍可获得大额融资。
  • True Anomaly｜Late-stage｜$600M｜未披露｜AI + aerospace + national security。国防航天继续承接AI外溢资金。
  • VAST Data｜Late-stage｜$500M｜未披露｜AI Infrastructure / data OS。AI数据基础设施仍是确定性资金池。
  • Vulcan Elements｜Late-stage｜$431M｜未披露｜稀土磁体 / 国防工业制造。供应链安全与先进制造逻辑突出。
  • SiFive｜Late-stage｜$400M｜未披露｜RISC-V / 半导体IP。AI算力链条之外，半导体自主化仍有估值溢价。
  • Valar Atomics｜Series B｜$340M｜未披露｜核能 / 电网能源。AI算力电力需求外溢到新型能源基础设施。
  • Waypoint Bio｜Series A｜$20M｜未披露｜AI-native biotech / CAR-T。用空间生物学和AI设计下一代实体瘤CAR-T。
  • Reactor｜Stealth/seed+｜$59M｜未披露｜实时生成视频开发者平台。生成式视频从模型演示走向可开发平台。
  • Triomics｜Series B｜$22M｜未披露｜肿瘤临床AI。医疗AI进入临床信息运营和试验匹配流程。
  • Itera｜Seed｜$12M｜未披露｜fluid circuit board / 深科技硬件。新计算硬件形态仍能拿到早期资金。
  • Didit｜Seed｜$7.5M｜YC、Pioneer Fund等｜AI-native身份与反欺诈。Agent时代的身份、授权、风控底座需求明确。
  • 维他动力｜Pre-A｜近5亿元人民币｜今日资本、高瓴创投、BV百度风投等｜消费级机器人 / 机器狗。中国具身智能从概念进入量产和消费场景验证。

3. 赛道热度观察

  • AI金额端极度集中，但底层市场并非完全失真：Crunchbase显示2026Q1全球VC达到 $300B / 6,000家，AI拿到 $242B，占80%；OpenAI、Anthropic、xAI、Waymo四笔合计 $188B，占全球季度融资65%。AlleyWatch显示4月美国VC $20.80B / 442笔，AI拿到 $15.18B / 189笔，Project Prometheus单笔占48.1%；剔除后仍有 $10.80B / 441笔，说明非异常值市场仍活跃。
  • 垂直Agent进入“可验收工作流”阶段：New Market Pitch统计agentic AI头部100家公司累计融资约 $9.8B，Top 10拿走约60%。最强场景集中在法律、客服、软件开发、招聘、金融分析、医疗接入等能替代明确流程的业务。
  • AI基础设施从GPU扩展到电力、数据、半导体、身份与安全：VAST Data、SiFive、Valar Atomics、Didit分别对应数据底座、芯片IP、电力约束、身份风控。资金正在寻找AI部署后的稀缺资源，而不只是模型参数。
  • Physical AI / hardtech 占据大额榜单：自动驾驶、EV、国防航天、稀土磁体、核能、先进制造进入Top deals；这类资产的共同点是资本开支高、政策相关性强、供应链壁垒深。
  • 中国市场政策与量产共同驱动：新浪/深圳市投资基金同业公会信息显示，中国2026Q1新募基金数量和金额分别同比增长 114.0% / 79.8%，资金集中到AI、具身智能、半导体、商业航天/低空经济。4月仅8家具身智能企业合计融资超 200亿元人民币，但下一阶段会看订单、量产能力和营收质量。
  • Biotech/医疗AI修复中但更挑剔：Waypoint Bio、Triomics、Signos等交易说明医疗AI与AI-native biotech仍有机会，但资金偏向“研发资产 + 临床流程效率 + 可监管数据闭环”，不是泛AI制药标签。

4. 值得关注的早期项目

  • Didit：$7.5M seed，AI-native身份与反欺诈。Agent规模化执行后，身份校验、授权链路、欺诈识别会从合规成本变成基础设施收入。
  • Itera：$12M seed，fluid circuit board。硬件路线风险高，但若验证性能和制造可行性，可能承接半导体/先进制造资金。
  • Waypoint Bio：$20M Series A，空间生物学 + AI-native CAR-T。具备技术新颖性和大病种空间，后续关键是临床推进与资本效率。
  • Reactor：$59M stealth，实时生成视频开发平台。如果能从“视频生成模型”抽象成开发者工作流，可能成为生成式视频生态的工具层。
  • 中国消费级/工业具身智能链条：维他动力、自变量、智元机器人等案例显示机构不只看demo，而是看真实工厂/家庭场景、量产节奏、供应链控制和售后成本。

5. 分析师点评

本期VC市场的主线不是“AI还热”，而是 AI生产资料化。第一层是基础模型和AI开发入口，它们继续吸走超大金额，形成资本集中度；第二层是垂直Agent和医疗/金融/法律/客服等可验收流程，它们负责把AI转成收入；第三层是电力、数据、半导体、国防航天、机器人和身份风控，这些是AI进入真实经济后的约束条件。

我会重点追踪三类信号：

1. 垂直Agent的续费与毛利：是否真正替代人力/流程，而不是只完成demo。

2. AI基础设施的非GPU瓶颈：电力、数据存储、推理运行环境、身份权限、审计合规会持续冒出新公司。

3. 中国具身智能的订单兑现：融资已热，下一步分化取决于真实交付、单位经济模型、安全责任和售后网络。
