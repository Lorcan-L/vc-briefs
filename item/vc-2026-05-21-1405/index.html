2. 赛道热度观察

AI仍是绝对主线，但高度集中

  • Crunchbase 数据显示，Q1 2026 全球 VC 投资约 $300B，AI 获得 $242B，占 80%。
  • OpenAI、Anthropic、xAI、Waymo 四笔融资合计 $188B，约占全球 VC 的 65%。
  • 这说明 AI 的“总量繁荣”主要由少数头部公司拉动，不能外推为所有 AI 初创公司融资环境全面改善。

AI+实体产业成为下一阶段主线

4月美国大额融资集中在 Project Prometheus、VAST Data、Reliable Robotics、Factory、Glydways 等项目；中国侧红杉/高瓴布局具身智能、低空经济、AI硬件、商业航天。资金正在从“模型层叙事”转向：

  • AI基础设施/数据中心/网络；
  • 机器人与具身智能；
  • 自动驾驶、航空航天、国防科技；
  • 智能制造与供应链软件。

国防、能源、关键材料显著升温

True Anomaly、Hermeus、Vulcan Elements、Valar Atomics、SiFive 等交易显示，资本正在押注“国家安全 + 产业基础设施”交叉点。国防航天、核能、半导体、关键材料的单笔融资规模明显变大，背后是地缘政治、供应链安全、能源安全共振。

中国一级市场是结构性修复，不是全面回到互联网周期

清科数据：

  • Q1 新募集基金 1,913 只，规模 5,590.69 亿元，同比 +114.0% / +79.8%。
  • 投资案例 2,568 起，披露金额 2,344.25 亿元，同比 +4.9% / +15.4%。
  • 热门行业为 IT、半导体、生物技术/医药健康、机械制造；细分热度集中在 AI、GPU、创新药、智能制造。

这轮中国回暖更偏人民币基金、国资/产业资本、硬科技逻辑，不能简单等同于美元基金或消费互联网融资回暖。

Biotech延续“AI平台 + 明确临床管线”双主线

Isomorphic Labs $2.1B 是标志性交易，但其他多数 biotech 融资集中在 $40M-$220M 区间。资金偏好：

  • AI drug design；
  • 免疫、肿瘤、神经、心代谢；
  • 有明确临床节点的资产型公司；
  • 平台能力与管线验证兼备的团队。

4. Team交叉研判

共识

  • VC市场不是全面复苏，而是被少数AI超大额融资显著抬高。
  • 资金偏好从纯软件/纯模型，转向 AI基础设施、实体产业落地、国防能源、半导体和Biotech。
  • 中国市场回暖更偏硬科技和政策/产业资本共振，不是传统互联网融资周期回归。

分歧与校正

  • Analyst 更强调 AI 总量优势；Critic 提醒必须剔除 mega-round 扭曲。
  • Strategist 认为当前更适合“主线集中 + 节点验证 + 估值克制”，不能追逐被大额融资拉高的估值锚。
  • 对早期项目的判断不能只看轮次金额，更要看是否有订单、量产、临床节点或基础设施刚需。

5. 分析师点评

当前 VC 市场处于“资本重新集中定价核心资产”的阶段。AI 并没有让所有创业公司更容易融资，反而提高了早期项目的门槛：只有站在算力、数据中心、机器人、工业场景、国防、能源、创新药等大叙事上，并能证明商业化或技术节点，才更可能拿到大额资金。

后续最值得追踪的方向：

1. AI Infra：推理成本、算力调度、数据中心网络、国产GPU、边缘推理。

2. 具身智能：能否从演示走向量产、订单和稳定交付。

3. Defense Tech / Space / Low-altitude Economy：政策订单、资质牌照、核心部件国产化。

4. 能源与关键材料：核能、聚变、稀土磁体、半导体材料设备。

5. AI+Biotech：是否能证明AI提高命中率、缩短周期，并转化为临床资产。

6. 中国智能制造：工厂、质检、供应链、设备维护、工业软件中的AI落地。

需要警惕的风险：AI mega-round 降温、硬件量产失败、政策订单节奏变化、Biotech临床失败、退出环境不畅以及同质化AI应用层竞争。
