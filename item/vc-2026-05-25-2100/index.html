VC投资动态调研报告｜2026-05-25 21:00

一、结论摘要

本期VC市场仍处于“巨额资金向AI与硬科技头部集中”的状态。全球层面，KPMG Venture Pulse显示，2026年Q1全球VC投资达到 3309亿美元，环比从1286亿美元大幅增长；其中10笔超过20亿美元的巨型融资贡献超过2060亿美元。AI仍是绝对主线，但资金不再只投基础大模型，正在扩散到 AI基础设施、具身智能/机器人、半导体、国防航天、核能/清洁能源、垂直AI应用。

中国市场的核心关键词是“政策确定性 + 技术落地 + 可退出”。先进制造、人工智能、半导体、具身智能、商业航天/低空经济、生物医药与新能源链条获得集中关注。新浪财经/深圳市投资基金同业公会文章提到，2026年Q1中国股权投资市场新募基金数量和金额分别同比增长 114.0% / 79.8%，4月仅8家具身智能企业融资即超过 200亿元人民币。

三、赛道热度观察

1. AI仍是资金中心，但结构正在分层

AI投资已经从“基础模型单点爆炸”进入“三层结构”：

  • 底层基础设施：算力、AI数据平台、半导体、AI OS。代表交易包括 VAST Data 5亿美元、SiFive 4亿美元。
  • 基础/通用智能实验室：Project Prometheus、Ineffable Intelligence、Recursive等仍能拿到超大额资金，且早期轮次金额明显非线性放大。
  • 场景型AI与Physical AI：机器人、自动驾驶、工业自动化、视频生成、空间智能成为资金扩散方向。

KPMG披露，Q1 2026全球软件赛道吸金 2252亿美元，几乎接近2025年全年水平，显示AI相关软件/平台仍是VC资金最集中的资产类别。

2. 具身智能/机器人是中国市场最热主线

中国具身智能融资呈现两个特征：

  • 金额快速抬升：TARS种子轮即达到35亿元人民币；X Square、Galaxea AI、Shengshu、GigaAI均在15-20亿元人民币级别。
  • 机构密度高：HSG/红杉、高瓴、IDG、启明、线性资本、美团、小米、CICC等集中出现。

这说明市场已经不再按传统早期项目估值逻辑定价，而是按“下一代平台型硬科技入口”定价。头部项目融资规模已经接近成熟期公司。

3. 国防、航天、低空经济继续升温

True Anomaly 6亿美元、Hermeus 2亿美元、Volant Aerotech 3亿美元，说明地缘政治与安全需求正在改变VC资产配置。KPMG也明确提到，Q2需要关注国防科技、航天科技、网络安全等方向。

该类项目共性是：

  • 技术门槛高；
  • 政府/军工/公共部门采购可形成稳定需求；
  • 退出路径更依赖并购与战略买方，而非短期IPO。

4. 能源与工业脱碳重新进入主流视野

Blue Energy、Valar Atomics、Boston Metal等项目显示，资本开始重新评估“硬能源资产”。与2021年前后的气候科技热潮不同，本轮更偏向：

  • 核能/先进能源系统；
  • 工业级电力供给；
  • 金属冶炼等高排放环节脱碳；
  • 与AI算力扩张绑定的能源需求。

AI数据中心对电力的长期需求，可能正在给核能、储能、工业能源技术带来新一轮估值逻辑。

5. 生物医药处于修复期，AI制药与差异化创新更受欢迎

Biotech整体没有AI/机器人那么爆炸，但资金开始从低谷修复。值得注意的是：EpiBiologics完成1.07亿美元B轮，用于双抗蛋白降解剂方向；国内文章也提到生物医药与创新药估值从低谷回升，资本更关注差异化创新与出海授权。

五、分析师点评

1. VC市场不是全面复苏，而是“结构性过热”

表面上看，Q1 2026全球VC投资创纪录，但交易数量并未同步爆炸。KPMG数据显示，全球VC从Q4 2025的10097笔下降至Q1 2026的8464笔，但金额从1286亿美元升至3309亿美元。核心结论是：钱更多了，但更集中。

这意味着普通创业公司融资环境并没有同步改善；能拿到钱的是具备以下特征的公司：

  • AI或硬科技主线明确；
  • 有技术壁垒和工程交付能力；
  • 有政策/产业/战略买方背书；
  • 能讲清楚退出路径或并购价值；
  • 能证明商业化闭环，而不是停留在概念阶段。

2. AI估值正在从“模型能力”迁移到“控制物理世界的能力”

Project Prometheus、TARS、X Square、Galaxea AI、Robot Era等交易共同指向一个判断：资本正把“Physical AI / Embodied AI”视为下一阶段AI商业化核心入口。

这条线值得持续追踪，因为它同时连接：

  • 模型能力；
  • 机器人硬件；
  • 工业制造；
  • 数据闭环；
  • 供应链；
  • 政策与国家竞争。

相比纯软件应用，具身智能的资本开支更重，但一旦形成规模化量产和数据闭环，壁垒也更高。

3. 早期轮次金额异常放大，隐含泡沫风险

Ineffable Intelligence Seed轮11亿美元、TARS Seed轮35亿元人民币，说明头部团队/头部赛道出现了“轮次失真”。传统意义上的Seed/A轮风险定价已经被稀缺人才、战略卡位和主权资本/产业资本推高。

风险点：

  • 估值领先商业化太多；
  • 技术路线尚未完全验证；
  • 后续融资需要更大资金池承接；
  • 一旦IPO或并购窗口收紧，估值回调会很剧烈。

4. 值得追踪的方向

未来1-2个季度建议重点追踪：

1. 具身智能核心部件：灵巧手、机器人关节、执行器、视觉/触觉传感器、低成本量产平台。

2. AI基础设施二阶机会：数据层、存储层、推理优化、企业AI部署与治理。

3. AI + 能源：核能、小型模块化反应堆、储能、电网调度、数据中心能源管理。

4. 国防/航天/安全技术：双用途无人系统、空间安全、网络安全、边缘AI。

5. AI制药与创新药出海：尤其是能形成BD授权或临床里程碑的差异化资产。

6. 垂直AI应用：法律、金融、制造、医疗、供应链等明确ROI场景，而不是泛Agent概念。
