VC投资动态调研｜2026-05-31 23:02

1. 本期核心判断

资本继续押注“AI可交付系统 + 物理世界 + 关键基础设施”；巨额资金抬高 headline，早中期增量更集中在企业 Agent、医疗运营、身份风控、实时视频/世界模型、能源/半导体与中国具身智能供应链。

2. 本期重点交易

  • ReactorNet / Reactor：种子轮/早期｜约数百万至千万美元级（来源口径不一）｜领投/主要投资方：深科技/能源类投资人｜赛道：核能、电力基础设施、AI能源
  • Triomics：Series B｜约数千万美元级｜领投/主要投资方：医疗AI/生命科学投资人｜赛道：医疗运营AI、临床试验工作流
  • Itera：早期轮｜未披露或小额早期｜领投/主要投资方：早期VC｜赛道：企业AI/Agentic workflow
  • Didit：Series A/B 延展｜数千万美元级｜领投/主要投资方：金融科技与身份安全投资人｜赛道：身份风控、KYC/反欺诈
  • Unframe：Seed/Series A｜早期轮｜领投/主要投资方：企业软件投资人｜赛道：企业AI交付、垂直Agent
  • 中国具身智能/半导体/商业航天项目群：A轮/B轮为主｜人民币数千万至数亿元｜领投/主要投资方：产业资本、地方基金、头部人民币基金｜赛道：具身智能、硬科技供应链

3. 赛道热度观察

  • AI 从模型转向系统交付：资金不再只买“AI概念”，更偏向能嵌入企业流程、缩短交付周期、替代高价值岗位工作流的公司。
  • 物理世界与关键基础设施升温：能源、电力、半导体、国防航天、机器人/具身智能成为VC继续加仓的硬资产方向。
  • 医疗AI更偏运营端：临床试验、医生/医院后台流程、患者运营等更容易形成付费闭环，比纯研发叙事更受关注。
  • 身份风控与金融基础设施保持韧性：AI放大自动化攻击后，KYC、反欺诈、身份验证需求继续增强。
  • 中国人民币融资集中在硬科技确定性：具身智能、半导体、商业航天、低空经济等受政策与产业链协同支撑。

4. 值得关注的早期项目

  • 垂直 Agent / 企业AI交付层：关注能把模型能力包装为稳定流程、带实施方法论和ROI证明的团队。
  • 医疗运营AI：如果能进入医院/药企现有系统并减少人工审核、招募、文档处理成本，商业化确定性强。
  • 能源与AI基础设施交叉项目：AI算力约束正在从GPU扩散到电力、散热、互连和选址，早期基础设施机会增多。
  • 具身智能数据与供应链中间层：相比整机品牌，数据采集、仿真、传感器、执行器和量产工具链可能更早形成收入。

5. 分析师点评

当前VC市场并非全面回暖，而是“结构性拥挤”：AI大额轮次吸走注意力，但真正可复制的早中期机会在AI落地的中间层——企业工作流、医疗运营、身份风控、硬件研发工具、能源/半导体基础设施。建议持续跟踪三类信号：

1. 客户愿意为节省人力或提升产能直接付费的AI Agent；

2. 与AI算力增长绑定的物理基础设施；

3. 中国硬科技项目中具备量产订单和产业资本背书的公司。
