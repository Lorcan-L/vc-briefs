VC投资动态调研｜2026-06-01 19:02

1. 本期核心判断

资本继续押注“AI可交付系统 + 物理世界 + 关键基础设施”：4月美国VC由Project Prometheus的$10B physical AI巨轮次显著拉高，但剔除异常值后，资金仍活跃流向AI基础设施、企业Agent交付、国防航天、能源电力、半导体、医疗运营、中国具身智能与硬科技供应链。

2. 本期重点交易

  • Project Prometheus｜Late-stage｜$10.0B｜未披露｜Foundational AI / physical-world applications

- AlleyWatch：占美国4月VC总额48.1%，是本期最大单笔扰动。

  • Slate Auto｜Late-stage｜$650M｜未披露｜EV / manufacturing
  • True Anomaly｜Late-stage｜$600M｜未披露｜AI + aerospace / national security
  • VAST Data｜Late-stage｜$500M｜未披露｜AI infrastructure / data platform
  • Vulcan Elements｜Late-stage｜$431M｜未披露｜Defense / rare-earth magnets / advanced manufacturing
  • SiFive｜Late-stage｜$400M｜未披露｜Semiconductor / RISC-V IP
  • Valar Atomics｜Series B｜$340M｜未披露｜Nuclear energy / industrial power
  • Reactor｜Stealth/未披露｜$59M｜未披露｜实时生成视频开发平台
  • Unframe｜Additional funding｜$50M｜Highland Europe｜企业AI交付平台
  • Triomics｜Series B｜$22M｜未披露｜肿瘤AI / 医疗运营
  • Itera｜Seed｜$12M｜未披露｜Fluid circuit board / deep tech hardware
  • Didit｜Seed extension｜$7.5M total seed｜未披露｜Identity & fraud AI infrastructure
  • 东昇聚变｜天使轮｜数亿元人民币｜红杉中国、IDG资本、高瓴、中科创星等｜核聚变 / 未来能源

3. 赛道热度观察

AI：集中度仍高，但“可交付系统”比纯模型更有解释力

  • AlleyWatch统计：2026年4月美国VC为 $20.80B / 442笔，同比资金 +63.9%、交易数 +21.1%；AI公司获得 $15.18B / 189笔，占资金 73.0%。
  • Project Prometheus单笔 $10B 占美国4月资金 48.1%；剔除后仍有 $10.80B / 441笔，说明底层交易并不冷。
  • 更干净的信号来自Series A：98笔、平均 $20.0M，没有被单笔交易严重扭曲，显示中早期资金仍在分散布局。

Agentic AI：头部高度集中，垂直场景更易拿钱

  • NewMarketPitch榜单显示，前100家agentic AI公司累计融资约 $9.8B，Top 10占约 60%；Harvey、Sierra、Poolside、Cognition、Parloa等成为资本集中标的。
  • 资本偏好正在从“通用Agent概念”转向能嵌入法律、客服、软件工程、医疗、金融分析等明确工作流的垂直Agent。

硬科技：国防航天、能源、电力、半导体进入核心配置

  • True Anomaly、Vulcan Elements、Hermeus、SiFive、Valar Atomics共同指向一个趋势：AI时代的瓶颈不只在模型，也在安全、芯片、电力、材料和制造能力。
  • Valar Atomics $340M Series B 是能源基础设施进入VC主流视野的强信号；Vulcan Elements与SiFive分别代表关键材料与半导体IP。

中国一级市场：硬科技和具身智能是资金密集区

  • 新浪财经转引深圳市投资基金同业公会：2026年Q1中国股权投资市场新募基金数量和金额分别同比 +114.0% / +79.8%，先进制造、AI、半导体最受关注。
  • 中国4月仅8家具身智能企业融资超 200亿元人民币；投资逻辑从“概念验证”转向“量产、订单和核心零部件”。
  • 中国证券报梳理：红杉中国、高瓴创投等一季度布局具身智能核心零部件、AI硬件、垂直行业应用、低空经济、商业航天与聚变能源。

Biotech / 医疗AI：从泛医疗叙事转向运营闭环

  • VCNewsDaily显示，Triomics $22M Series B 代表肿瘤中心复杂临床信息运营化的医疗AI方向。
  • Signos $20M 指向消费医疗/代谢健康硬件；医疗AI更值得关注的是能进入医院或用户日常流程、并形成数据闭环的产品。

4. 值得关注的早期项目

  • Didit｜Seed｜身份与欺诈AI基础设施：小额但方向关键，适合跟踪KYC、反欺诈、AI agent身份层。
  • Itera｜Seed｜流体电路板：deep tech硬件，若技术成立，可能进入芯片散热、柔性电子或机器人硬件链条。
  • Reactor｜Stealth｜实时生成视频开发平台：生成式视频从内容工具进入实时交互/开发平台，应用空间包括游戏、广告、仿真和创意工作流。
  • Triomics｜Series B｜肿瘤AI：医疗运营AI比泛问答更接近付费和流程改造。
  • Floatboat｜天使轮｜AI工作空间/Agent形态：红杉中国参与，关注Agent如何接管用户上下文与工作流。
  • 东昇聚变｜天使轮｜聚变能源：周期极长，但顶级机构阵容显示其战略价值与期权属性。

5. 分析师点评

1. 不要被巨轮次误判为全面泡沫。 Project Prometheus显著扭曲4月美国资金规模，但剔除后仍有441笔交易，说明市场在“高集中 headline + 底层结构性修复”中运行。

2. AI投资从模型竞赛转为产业交付。 企业Agent、AI infra、实时视频、医疗运营、身份风控和physical AI比“又一个通用模型”更能解释新增资金流向。

3. 硬科技正在成为AI资本的二阶交易。 能源、电力、半导体、材料、国防安全和制造能力，是AI规模化后的约束条件，也正在被VC提前定价。

4. 中国市场看重政策、订单和退出路径。 具身智能、半导体、低空经济、商业航天、聚变能源都需要技术突破与产业政策双轮驱动；头部机构更偏好“技术已突破但市场尚未统一定价”的窗口。

5. 后续追踪优先级：企业Agent真实ARR/续费；physical AI客户与成本结构；具身智能量产订单与毛利率；能源/聚变监管节点；医疗AI合规和医院采购周期。

6. 可信度与限制

  • 高可信：AlleyWatch 4月美国VC统计、VCNewsDaily单笔融资列表、中国证券报对红杉/高瓴投资方向梳理。
  • 中可信：新浪财经转载的行业协会数据，适合判断中国赛道热度，但单笔交易仍需公司公告交叉验证。
  • 中可信：NewMarketPitch agentic AI榜单，适合判断头部格局和资本集中度，不宜单独作为估值依据。
