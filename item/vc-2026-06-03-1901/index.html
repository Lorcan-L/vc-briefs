3. 赛道热度观察

AI：从“模型崇拜”转向“可交付系统”

  • AlleyWatch 统计显示，美国 2026 年 4 月 VC 融资 $20.80B / 442 笔，同比资本额 +63.9%、交易数 +21.1%。AI 公司获得 $15.18B，占总资本 73.0%。
  • 但 Project Prometheus 单笔 $10B 占全美当月 48.1%；剔除后，市场仍有 $10.80B / 441 笔，说明不是只有单一 megadeal，而是底层热度仍有宽度。
  • Crunchbase tracker 显示 Anthropic、Cognition 等 AI 超大轮继续刷新估值边界，资本仍愿意为“基础模型/开发者工具/AI生产力入口”支付高溢价。

Agentic AI：垂直工作流替代成为早中期主线

  • New Market Pitch 数据库覆盖 2025.06-2026.05 的 40 笔 pure-play agentic AI 公开股权融资，共 $1.413B。
  • 垂直 AI Agent 占 26/40 笔、$1.052B，占披露资本 74.5%；说明资本更偏好能接管具体业务结果的 Agent，而非泛用“agent platform”概念。
  • Seed + Series A 合计占披露资本 51.6%，表明赛道仍处于早中期密集试错期，但 round size 中位数已达 $24M，优秀项目融资门槛明显抬高。

Hard Tech：国防、航天、核能、半导体正在成为AI外溢资金承接层

  • 美国 Top 10 交易中，True Anomaly、Vulcan Elements、Hermeus、Valar Atomics、SiFive 等均属于物理世界基础设施与战略科技。
  • 资金逻辑从“软件高毛利”扩展到“AI部署所需的算力、能源、安全与制造能力”。这类项目资本强度高，但政策/供应链确定性更强。

中国：具身智能、半导体、商业航天和低空经济集中吸金

  • 新浪/深圳市投资基金同业公会称，2026年一季度中国股权投资市场新募基金数量和金额分别同比 +114.0% 和 +79.8%，先进制造、AI、半导体高度集中。
  • 2026年4月仅 8 家具身智能企业获超 200 亿元融资，市场已从“概念验证”转向“量产、订单、工厂/家庭场景落地”。
  • 中国证券报对红杉中国、高瓴创投一季度投资梳理显示，头部机构更看重具身智能核心零部件、AI硬件、垂直应用、低空经济、商业航天与聚变等长周期资产。

Biotech / Climate Tech：不是主流金额中心，但AI-native和能源基础设施仍值得追踪

  • Labiotech tracker 对 2026 biotech funding 做持续追踪，但本次抓取页交易细节有限；结合搜索结果看，AI制药/差异化创新药仍处于估值修复阶段。
  • 气候科技中，核能、电网、储能与工业脱碳更容易与“AI算力需求”和“政策确定性”绑定，Valar Atomics 的 $340M Series B 是本期代表信号。

4. 值得关注的早期项目

  • Veris AI：Seed $8.5M，做企业 AI Agent 训练/测试模拟环境。关注点是 Agent 安全、评测和部署前验证，可能成为企业 Agent 普及的基础设施。
  • Blaxel：Seed $7.3M，做 autonomous AI agents 的云基础设施，First Round + YC。关注点是 Agent runtime / execution infra。
  • VoiceCare AI：Seed $4.54M，医疗行政语音 Agent，切入 RCM 和 back-office conversation，适合观察医疗工作流自动化。
  • AppFactor：Seed $4M，企业软件维护的 agentic orchestration，若能在 legacy enterprise software 里创造可量化 ROI，商业化路径会很清晰。
  • Trace：Seed $3M，解决企业 Agent 采纳与控制问题，属于“human approval / governance layer”，可能受益于企业从试点走向规模部署。
  • 大览科技：天使 + Pre-A 连续融资，红杉中国参与，AI+长焦消费硬件；代表“AI硬件不是拼参数，而是重构场景体验”的方向。
  • Floatboat：天使轮，红杉中国与微光创投参与，定位人机协同工作空间，回答 Agent 以何种产品形态进入日常工作。

5. 分析师点评

1. 市场不是简单回暖，而是结构性集中。 资本增长强劲，但资金并未均匀流入所有创业方向；AI、硬科技、政策支持的新质生产力赛道明显虹吸。

2. AI融资进入第二阶段：从模型到系统。 仅讲模型能力已经不够，资本更关心能否形成可部署工作流、可验证 ROI、数据/权限/集成闭环。

3. Physical AI 与硬科技溢价上升。 Project Prometheus、具身智能、航天/国防/核能/半导体等交易说明，投资人正在把 AI 视为驱动物理世界生产率提升的底层变量。

4. Agentic AI 的最佳切口是垂直结果，不是通用助手。 公开数据里，垂直 Agent 占据大多数资金；财务、采购、医疗、ITSM、安全、法务等“任务可定义、结果可衡量、权限可控制”的场景更容易融资。

5. 中国头部机构押注更务实：量产、订单、产业链位置。 红杉/高瓴等对具身智能和 AI 硬件的布局，核心不是追热点，而是寻找“技术突破已发生、但市场尚未完全定价”的非共识窗口。
