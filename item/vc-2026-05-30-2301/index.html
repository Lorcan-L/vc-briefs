3. 赛道热度观察

AI仍是资金黑洞，但“物理世界化”比纯软件更强

  • AlleyWatch 统计：2026年4月美国VC融资 $20.80B / 442笔，同比资本+63.9%、交易数+21.1%。AI公司拿到 $15.18B，占全部资本 73.0%。
  • Project Prometheus 单轮 $10B 占全美4月资本 48.1%。剔除它后，美国市场仍有 $10.80B / 441笔，说明不是只有单一巨轮次在支撑市场。
  • AI资金的关键词从“模型性能”转向 physical AI、AI基础设施、AI数据中心互连、agent search、AI coding、企业BI与医疗/法务/金融等受监管垂直场景。

DeepTech / 国防 / 能源成为AI之外最硬的第二主线

  • 4月美国Top 10中，True Anomaly、Vulcan Elements、Hermeus、Valar Atomics、SiFive、Slate Auto、Glydways集中覆盖航天防务、稀土磁体、核能、半导体IP、EV/自动交通。
  • 这说明VC正在重新接受资本开支更重、周期更长但战略价值更高的硬科技项目，尤其是能和国家安全、算力瓶颈、能源瓶颈绑定的公司。

Mid-stage相对健康，Series A是更干净的市场温度计

  • AlleyWatch显示4月美国Series A为 98笔 / $1.96B，平均约 $20.0M，不像Late-stage被Prometheus极端扭曲。
  • Cooley Q1数据显示，Q1 2026其经手 165笔 / $39.9B VC融资，交易数量降至2016 Q3以来低点，但投入资本上升；Up round占比升至 86%，Series D+中位pre-money从 $1B升至 $2.5B。
  • 结论：市场不是全面火热，而是“少数高质量/高战略项目高价融资 + 普通项目交易量收缩”。

中国VC：政策确定性 + 硬科技壁垒 + 商业化验证

  • 新浪财经转引深圳市投资基金同业公会：2026年Q1中国股权投资新募基金数量同比+114.0%、金额同比+79.8%；先进制造、人工智能、半导体集中吸走资金。
  • 2026年4月，仅8家具身智能企业融资超200亿元，估值突破千亿；AI、具身智能、半导体、商业航天/低空经济、生物医药、新能源链、高端装备/新材料是主轴。
  • 中国市场投资门槛从“概念/论文”变成“研发强度、政策背书、产品落地、订单/营收增长和退出路径”。

Biotech与Climate/能源：不再是泛主题，资金追临床/技术读出

  • Labiotech tracker说明其2026 biotech融资跟踪基于新闻、PR和专业数据库，重点筛选有行业影响的融资事件；抓取页未暴露完整项目清单，故本期只作为趋势辅助来源。
  • 生物医药在中国侧已从估值低谷修复，资本偏好差异化创新、AI制药和出海授权；能源侧则更偏核能、储能、氢能、低碳材料等硬资产方向。

4. 值得关注的早期项目

  • Hardline：$2M Pre-seed，Mucker Capital领投；用 voice-first AI 解决工地高噪声、低屏幕可用性的施工协同问题，是“AI进入物理现场”的小票面样本。
  • Rely：$4.5M Seed，2048 Ventures领投；AI-native multifamily real estate diligence，价值在于把租约、rent roll、合同、财务文件审核变成可追溯流程。
  • Sygaldry Technologies：$105M Series A；量子加速AI服务器，虽金额已不小，但从阶段看仍属于早期深科技基础设施押注。
  • 中国具身智能供应链早期公司：维他动力等消费/工业机器人公司开始拿到头部机构资金，关注从demo走向量产订单的能力。

5. 分析师点评

本期最重要的判断：VC没有离开AI，但已经不满足“模型层故事”。资本正在寻找三类更接近终局价值的公司：

1. 把AI接入物理世界的公司：physical AI、机器人、自动驾驶/交通、航天防务、制造业AI。这类公司更重资产，但一旦形成工程闭环，壁垒比纯SaaS更厚。

2. 解决AI规模化部署瓶颈的公司：AI-native search、数据中心互连、AI BI、AI-readable docs、金融与合规基础设施。它们是Agent大规模使用后的“铲子”。

3. 和政策/战略安全绑定的硬科技公司：半导体、稀土磁体、核能、商业航天、低空经济、先进制造。资金愿意为确定性退出路径和国家级需求支付溢价。

风险也很清楚：部分头部AI交易处于“in talks”或战略投资口径，估值和金额可能被新闻周期放大；late-stage总量被极端巨轮次扭曲，不能简单推断全市场回暖。更可靠的观察指标应放在Series A/B的交易数量、客户收入质量、毛利率改善、机器人/硬科技的实际交付能力，以及AI Agent在受监管行业的合规闭环。

后续建议追踪：

  • Project Prometheus、Cursor等超大AI轮次是否正式close；
  • AI基础设施公司（Exa、nEye.ai、Mintlify等）能否绑定真实Agent调用量；
  • 中国具身智能企业从融资新闻到量产订单的转化；
  • 核能/半导体/稀土磁体等硬科技项目后续是否有产业资本接盘；
  • Seed/Pre-seed垂直Agent项目是否在12个月内证明可量化ROI。
