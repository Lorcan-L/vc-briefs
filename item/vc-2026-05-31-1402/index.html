3. 赛道热度观察

  • AI仍是最大资金磁铁，但结构更硬：AlleyWatch数据显示，美国4月VC为 208亿美元 / 442笔，同比资本+63.9%、交易数+21.1%；AI拿到 151.8亿美元 / 189笔，占资本 73%。
  • 单一巨轮次扭曲总量：Project Prometheus 的 100亿美元 占美国4月VC总额 48.1%。剔除它后，美国市场仍有 108亿美元 / 441笔，说明底层市场不弱，但不能用headline金额判断整体估值温度。
  • Late-stage吸走大部分资本，Series A更能代表真实广度：4月美国Late-stage占 76.4% 资本；Series A为 19.6亿美元 / 98笔，平均 2000万美元，分布更均匀，是观察机构风险偏好的更干净样本。
  • 非AI硬科技开始接棒：国防航天、稀土磁体、核能、电力、半导体IP、自动交通进入Top 10，说明“AI约束”正在外溢到能源、算力、供应链与国家安全。
  • Agentic AI进入头部集中阶段：New Market Pitch统计Top 100 agentic AI公司累计融资约 98亿美元，Top 10吸收约 60%；Harvey、Sierra、Poolside、Cognition、Parloa等已经形成垂直场景和工作流入口优势。
  • 中国市场偏政策确定性和可退出：新浪财经/深圳市投资基金同业公会口径显示，一季度中国新募基金数量和金额同比+114.0%/+79.8%，资金集中在AI、具身智能、半导体、商业航天、低空经济、生物医药和新能源链。

4. 值得关注的早期项目

  • Reactor：5900万美元出隐，做实时生成视频开发平台和实时world model。看点是能否成为游戏、机器人训练、广告和工业仿真的基础设施层。
  • Itera：1200万美元种子轮，做“流体电路板”，主张让电子设计像软件一样快速迭代。若验证成立，可能切入硬件研发工具链的高价值节点。
  • Didit：总种子轮750万美元，KYC/KYB、交易监控、钱包筛查一体化API。AI agent commerce、跨境金融和链上钱包都需要更低摩擦的身份/风控基础设施。
  • Triomics：2200万美元Series B，切肿瘤临床信息结构化、试验匹配和pre-visit chart prep。医疗AI里“流程节点型产品”比泛问答更容易付费和合规落地。
  • 维他动力：近5亿元Pre-A，消费级机器狗“大头”体现中国具身智能从工业/科研走向消费场景，但需要继续验证成本、渠道和可持续需求。

5. 分析师点评

当前VC市场的主线不是“所有AI都涨”，而是“AI把资本推向真实世界的约束条件”。模型能力仍是叙事中心，但最值得追踪的是三层资产：

1. 交付层：企业AI delivery、垂直Agent、医疗/法律/客服/工程工作流，重点看TCV、部署周期、净收入留存和客户扩张。

2. 物理世界层：world model、机器人、自动驾驶、仿真数据、国防航天、先进制造，重点看真实场景验证和供应链控制力。

3. 约束基础设施层：电力/核能、AI数据中心、半导体IP、稀土材料、身份合规与安全，重点看政策确定性、资本开支能力和退出路径。

投资判断上，本期建议降低对“单笔融资金额排行榜”的权重，重点跟踪剔除巨额异常值后的中位数、Series A/B分布、客户收入质量、监管资质和政策退出确定性。
