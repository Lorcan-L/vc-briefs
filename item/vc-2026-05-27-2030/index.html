VC投资动态调研｜2026-05-27 20:30

1. 本期核心判断

资本正在从“泛AI叙事”转向“AI + 物理世界 + 基础设施”的超大额集中押注：前沿模型、AI算力/数据基础设施、具身智能/机器人、自动驾驶、半导体与国防/能源硬科技成为本期资金主线。

3. 赛道热度观察

AI：资金集中度继续创历史极值

Crunchbase 数据显示，2026年Q1全球创业投资约 3000亿美元，同比和环比均增长150%以上；其中AI获得约 2420亿美元，占全球VC融资约80%。四家头部公司 OpenAI、Anthropic、xAI、Waymo 合计融资约 1880亿美元，占季度全球VC约65%。这意味着市场并不是全面复苏，而是被少数AI超级平台和基础设施项目强力拉动。

具身智能/机器人：中国市场出现“融资月”

中文资料显示，2026年4月仅8家具身智能企业融资超过200亿元人民币。资金偏好已经从概念验证转向“量产交付、真实订单、核心零部件和控制系统”。天机智能、TARS、Robot Era、Galaxea AI、X Square 等交易共同指向：机器人正在成为中国硬科技VC最拥挤、也最具估值溢价的方向之一。

AI基础设施：算力、数据中心、数据系统、芯片继续吸金

DayOne Data Centers、Nscale、VAST Data、SiFive 等融资显示，VC正在沿AI训练和推理成本链条向上游配置：数据中心、GPU云、AI OS、RISC-V/芯片IP等均属于“卖铲子”资产。

生物科技：AI制药与免疫/遗传平台并行升温

Xtalks tracker 显示，Isomorphic Labs 21亿美元Series B 是本期生物科技里最突出的大额交易；Serif Biomedicines、Fathom Therapeutics、Coultreon、Vivacta Bio、Windward Bio、CREATE Medicines 等项目覆盖 Modified DNA、物理+AI小分子设计、免疫疾病、in vivo CAR 等方向。资金更偏向有平台能力、临床读数窗口和大药企合作可能的公司。

硬科技与政策确定性：中国人民币基金更重“可退出、可落地”

中国证券报与新浪财经资料均指向：红杉、高瓴等头部机构在Q1明显偏好“AI与实体产业融合”，包括具身智能核心零部件、AI硬件、低空经济、商业航天、聚变等长周期高壁垒方向。政策层面的并购基金、创业板新标准与未来产业支持，为人民币基金提供了退出预期。

4. 值得关注的早期项目

  • TARS：2025年成立即获得35亿元人民币种子轮，聚焦具身智能。种子轮金额异常大，说明头部机构正在提前抢占机器人基础模型/平台型入口。
  • Serif Biomedicines：Flagship Pioneering 以5000万美元启动，Modified DNA 平台若能降低免疫激活并提升表达效率，具备平台外溢潜力。
  • Fathom Therapeutics：4700万美元Series A，使用物理、AI和仿真研究蛋白动态，属于AI制药里更偏“机制建模”的路线。
  • Signadori Bio：约1110万欧元种子轮，off-the-shelf实体瘤免疫疗法，金额不大但切入实体瘤免疫重编程这个高难方向。
  • Floatboat：红杉中国与微光创投参与天使轮，方向是让AI Agent接管工作上下文和流程，值得作为“Agent工作空间”形态观察样本。
  • 大览科技 / OdyssLife：AI硬件早期项目，分别切入长焦影像和健康感知，代表消费硬件从“联网设备”向“AI感知终端”演进。

5. 分析师点评

本期VC市场表面是“融资总量大幅回暖”，实质是两极化更强：一边是AI和物理世界硬科技吞噬大部分增量资金，另一边是缺乏技术迭代、缺乏退出路径的传统项目仍然融资困难。

几个判断：

1. AI投资已经进入资本开支周期：赢家不只在模型层，也在算力、数据中心、芯片、数据系统和机器人执行端。未来6-12个月应持续跟踪“AI infra单位经济性”和“推理需求真实增长”。

2. 具身智能的估值锚从Demo转向订单：能证明万台级交付、客户复购、关键零部件自研的公司，会继续享受融资溢价；纯概念公司会被快速挤出。

3. 中国人民币VC的关键词是政策确定性 + 产业落地 + 退出预期：低空经济、商业航天、半导体、机器人、生物医药的融资韧性来自政策和产业双轮驱动。

4. 早期机会更可能出现在“非共识硬问题”：比如AI Agent工作空间、Modified DNA、实体瘤免疫重编程、核能/新型能源、机器人数据闭环，而不是再做同质化模型或套壳应用。

5. 风险点在估值集中和退出压力：OpenAI/Anthropic/xAI等超大额融资正在抬高市场估值锚，但IPO窗口并未同步完全恢复。如果二级市场风险偏好下降，晚期估值回调会向早期传导。
